# Defanging an IP Address

**Problem Statement:**  
Given a valid (IPv4) IP address, return a **defanged** version of that IP address.

A defanged IP address replaces every period `"."` with `"[.]"`.


### Examples

### Example 1
**Input:** `address = "1.1.1.1"`  
**Output:** `"1[.]1[.]1[.]1"`

### Example 2
**Input:** `address = "255.100.50.0"`  
**Output:** `"255[.]100[.]50[.]0"`


### Constraints
- The given `address` is a valid IPv4 address.

---
### Intuition
We just need to replace every dot `'.'` in the IP address with the string `"[.]"`.  
All other characters (digits) stay the same, so we scan the string once and build a new result string.


### Algorithm Steps
1. Initialize an empty string `ans`.
2. Traverse the input string `address` character by character.
3. If the current character is `'.'`, append `"[.]"` to `ans`.
4. Otherwise, append the character itself to `ans`.
5. After the traversal, return `ans`.

``` cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    string defangIPaddr(string address) {
        int index = 0;
        string ans = "";

        while (index < (int)address.size()) {
            if (address[index] == '.') {
                ans += "[.]";               // replace '.' with "[.]"
            } else {
                ans += address[index];      // keep digit as it is
            }
            index++;
        }
        return ans;
    }
};

int main() {
    Solution sol;

    string address1 = "1.1.1.1";
    cout << "Input:  " << address1 << "\n";
    cout << "Output: " << sol.defangIPaddr(address1) << "\n\n";

    string address2 = "255.100.50.0";
    cout << "Input:  " << address2 << "\n";
    cout << "Output: " << sol.defangIPaddr(address2) << "\n";

    return 0;
}
```
### Complexity Analysis

**Time Complexity: `O(n)`**  
We scan the string once, where `n` is the length of the IP address.

**Space Complexity: `O(n)`**  
We build a new string for the defanged address, which can take `O(n)` space.
