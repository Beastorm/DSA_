# String Rotated by 2 Places

**Problem Statement:**  
Given two strings `s1` and `s2`, return `true` if `s2` can be obtained by rotating (in any direction) `s1` by **exactly 2 places**, otherwise return `false`.

> **Note:** Both rotations should be performed in the **same direction** chosen initially.


### Examples

### Example 1
**Input:** `s1 = "amazon"`, `s2 = "azonam"`  
**Output:** `true`  
**Explanation:** `"amazon"` rotated anti-clockwise (left) by 2 becomes `"azonam"`.

### Example 2
**Input:** `s1 = "geeksforgeeks"`, `s2 = "geeksgeeksfor"`  
**Output:** `false`  
**Explanation:** Rotating `"geeksforgeeks"` by 2 in either direction will not produce `"geeksgeeksfor"`.

### Example 3
**Input:** `s1 = "ab"`, `s2 = "ab"`  
**Output:** `true`  
**Explanation:** Rotating `"ab"` by 2 places in any direction gives `"ab"` again.


### Constraints
- `1 ≤ s1.length, s2.length ≤ 10^5`
---
### Intuition
Rotating a string by **exactly 2 places** can be done in only two valid ways:
- **Clockwise (right) rotation by 2**
- **Anti-clockwise (left) rotation by 2**

So we generate both rotated versions of `s1` and check if either one equals `s2`.

### Algorithm Steps
1. If `s1.length() != s2.length()`, return `false`.
2. Create a copy `clockwise = s1`:
   - Rotate it right by 1 place two times.
   - If `clockwise == s2`, return `true`.
3. Create a copy `anticlockwise = s1`:
   - Rotate it left by 1 place two times.
   - If `anticlockwise == s2`, return `true`.
4. If neither matches, return `false`.

``` cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    // Rotate string by 1 place to the right (clockwise)
    void rotateClockwise(string &s) {
        int n = (int)s.size();
        if (n <= 1) return;

        char last = s[n - 1];
        int index = n - 2;

        while (index >= 0) {
            s[index + 1] = s[index];
            index--;
        }
        s[0] = last;
    }

    // Rotate string by 1 place to the left (anti-clockwise)
    void rotateAntiClockwise(string &s) {
        int n = (int)s.size();
        if (n <= 1) return;

        char first = s[0];
        int index = 1;

        while (index < n) {
            s[index - 1] = s[index];
            index++;
        }
        s[n - 1] = first;
    }

    // Return true if str2 can be obtained by rotating str1 by exactly 2 places
    // either clockwise or anti-clockwise
    bool isRotated(string str1, string str2) {
        if (str1.size() != str2.size()) return false;

        // Clockwise by 2
        string clockwise = str1;
        rotateClockwise(clockwise);
        rotateClockwise(clockwise);
        if (clockwise == str2) return true;

        // Anti-clockwise by 2
        string anticlockwise = str1;
        rotateAntiClockwise(anticlockwise);
        rotateAntiClockwise(anticlockwise);
        if (anticlockwise == str2) return true;

        return false;
    }
};

int main() {
    Solution sol;

    // Example 1
    string s1 = "amazon", s2 = "azonam";
    cout << (sol.isRotated(s1, s2) ? "true" : "false") << "\n"; // true

    // Example 2
    string s3 = "geeksforgeeks", s4 = "geeksgeeksfor";
    cout << (sol.isRotated(s3, s4) ? "true" : "false") << "\n"; // false

    // Example 3
    string s5 = "ab", s6 = "ab";
    cout << (sol.isRotated(s5, s6) ? "true" : "false") << "\n"; // true

    return 0;
}
```
### Complexity Analysis

Let `n = s1.length()`.

**Time Complexity: `O(n)`**  
Each single rotation takes `O(n)` time. We perform 2 clockwise rotations and 2 anti-clockwise rotations → `4 * O(n)` which is `O(n)`.

**Space Complexity: `O(n)`**  
We create two copies of the string (`clockwise` and `anticlockwise`), each of size `n`.
