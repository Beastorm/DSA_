## Add Strings
**Problem Statement:**  
Given two non-negative integers, `num1` and `num2` represented as strings, return the sum of `num1` and `num2` as a string.

You must solve the problem without using any built-in library for handling large integers (such as BigInteger). You must also not convert the inputs to integers directly.

---

>### Examples:
### Example 1:

**Input:** num1 = "11", num2 = "123"

**Output:** "134"

### Example 2:

**Input:** num1 = "456", num2 = "77"

**Output:** "533"

### Example 3:

**Input:** num1 = "0", num2 = "0"

**Output:** "0"

---

### Constraints:

- `1 <= num1.length, num2.length <= 10^4`
- `num1` and `num2` consist of only digits.
- `num1` and `num2` don't have any leading zeros except for the zero itself.

---

## Approach

### Intuition:

When numbers are very large, they cannot be stored safely in integer data types.
So we treat the numbers as **strings** and add them the same way we do manually.

We start adding digits from the **rightmost side** (least significant digit),
keep track of a **carry**, and move leftwards.
Each digit sum produces:
- a result digit (`sum % 10`)
- a carry (`sum / 10`)

The result is built in reverse order and finally reversed to get the correct answer.


### Algorithm Steps:

1. Initialize two pointers:
   - `index1` at the last character of `num1`
   - `index2` at the last character of `num2`

2. Initialize:
   - `carry = 0`
   - empty string `ans`

3. While both indices are valid:
   - Convert characters to digits
   - Compute `sum = digit1 + digit2 + carry`
   - Update `carry = sum / 10`
   - Append `(sum % 10)` to `ans`
   - Decrement both indices

4. While digits remain in `num1`:
   - Add digit with carry
   - Update carry and append result digit
   - Decrement index

5. While digits remain in `num2`:
   - Add digit with carry
   - Update carry and append result digit
   - Decrement index

6. If `carry` is non-zero, append it to `ans`

7. Reverse `ans` to obtain the final sum

8. Return the resulting string


### C++ Code:
``` cpp
#include <bits/stdc++.h>
using namespace std;

string addStrings(string num1, string num2) {
    int index1 = num1.size() - 1;
    int index2 = num2.size() - 1;

    int carry = 0;
    string ans = "";

    // Add while both strings have digits
    while (index1 >= 0 && index2 >= 0) {
        int sum = (num1[index1] - '0') + (num2[index2] - '0') + carry;
        carry = sum / 10;
        char c = '0' + (sum % 10);
        ans += c;

        index1--;
        index2--;
    }

    // If digits left in num1
    while (index1 >= 0) {
        int sum = (num1[index1] - '0') + carry;
        carry = sum / 10;
        char c = '0' + (sum % 10);
        ans += c;

        index1--;
    }

    // If digits left in num2
    while (index2 >= 0) {
        int sum = (num2[index2] - '0') + carry;
        carry = sum / 10;
        char c = '0' + (sum % 10);
        ans += c;

        index2--;
    }

    // If carry remains
    if (carry) {
         // or  ans += (carry + '0');
        ans += '1'; // max carry can go in case of two numbers, i.e., 19 carry 1.
    
    }

    // Reverse the result
    reverse(ans.begin(), ans.end());
    return ans;
}

int main() {
    string num1, num2;
    cout << "Enter first number: ";
    cin >> num1;
    cout << "Enter second number: ";
    cin >> num2;

    cout << "Sum = " << addStrings(num1, num2) << endl;
    return 0;
}
```
### Complexity Analysis:
- **Time Complexity**
   - *O(max(n, m))*, where `n` and `m` are lengths of the two strings

- **Space Complexity**
  - *O(max(n, m))* for storing the result
