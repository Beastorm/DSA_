# Sort a String

**Problem Statement:**
Given a string `S` consisting of lowercase letters, arrange all its letters in **ascending order**.


### Examples

### Example 1
**Input:** `S = "edcab"`  
**Output:** `"abcde"`  
**Explanation:** Characters are in ascending order in `"abcde"`.

### Example 2
**Input:** `S = "xzy"`  
**Output:** `"xyz"`  
**Explanation:** Characters are in ascending order in `"xyz"`.


### Your Task
You don't need to read input or print anything. Your task is to complete the function `sort()` which takes the string as input and returns the modified string.

### Expected Complexity
- **Time Complexity:** `O(|S| * log |S|)`
- **Auxiliary Space:** `O(1)`

---

### Intuition
Since the string contains only **lowercase English letters** (`'a'` to `'z'`), we can count how many times each letter occurs using a frequency array of size `26`.  
Then we rebuild the answer by printing letters from `'a'` to `'z'` according to their frequencies. This automatically produces a sorted string.


### Algorithm Steps
1. Create an integer array `alpha[26]` initialized to `0`.
2. Traverse the string `s` and for each character `ch`:
   - Increment its frequency: `alpha[ch - 'a']++`
3. Initialize an empty string `ans`.
4. For `i` from `0` to `25`:
   - Let `c = 'a' + i`
   - Append `c` to `ans` exactly `alpha[i]` times.
5. Return `ans`.

``` cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    // Function to sort a lowercase string in ascending order
    string sort(string s) {
        vector<int> alpha(26, 0);

        // Count frequency of each character
        for (int i = 0; i < (int)s.size(); i++) {
            alpha[s[i] - 'a']++;
        }

        // Build sorted string using frequencies
        string ans = "";
        for (int i = 0; i < 26; i++) {
            char c = 'a' + i;
            while (alpha[i] > 0) {
                ans += c;
                alpha[i]--;
            }
        }

        return ans;
    }
};

int main() {
    Solution sol;

    string s1 = "edcab";
    cout << "Input:  " << s1 << "\n";
    cout << "Output: " << sol.sort(s1) << "\n\n"; // abcde

    string s2 = "xzy";
    cout << "Input:  " << s2 << "\n";
    cout << "Output: " << sol.sort(s2) << "\n";   // xyz

    return 0;
}
```
### Complexity Analysis

Let `n = |S|`.

**Time Complexity: `O(n)`**  
We count frequencies in `O(n)` and rebuild the string by iterating over 26 letters (constant) and appending a total of `n` characters.

**Space Complexity: `O(1)`**  
We use a fixed-size frequency array of size `26`, which is constant extra space (excluding the output string).
