## Merge K Sorted Linked Lists

**Problem Statement:**   
Given an array of `K` sorted linked lists, merge all the linked lists into **one sorted linked list** and return it.

---

>### Examples:

**Example 1:**
- **Input:**
```
  List 1: 1 вҶ’ 4 вҶ’ 7
  List 2: 2 вҶ’ 5 вҶ’ 8
  List 3: 3 вҶ’ 6 вҶ’ 9
```
- **Output:** `1 вҶ’ 2 вҶ’ 3 вҶ’ 4 вҶ’ 5 вҶ’ 6 вҶ’ 7 вҶ’ 8 вҶ’ 9`

**Example 2:**
- **Input:**
```
  List 1: 1 вҶ’ 4 вҶ’ 5
  List 2: 1 вҶ’ 3 вҶ’ 4
  List 3: 2 вҶ’ 6
```
- **Output:** `1 вҶ’ 1 вҶ’ 2 вҶ’ 3 вҶ’ 4 вҶ’ 4 вҶ’ 5 вҶ’ 6`

---

### Key Insight:
```
  We have K sorted lists. At each step we need the MINIMUM
  among the current front elements of all K lists.

  вҶ’ MIN HEAP of size K is perfect!

  Push first element of each list.
  Pop min, attach to result, push next element from same list.
  Repeat until heap empty.

  Like Merge K Sorted Arrays but with linked list pointers!
```

---

### Approaches Overview:
```
  Method 1: Collect all + Sort    вҶ’ O(N log N)    simple
  Method 2: Merge two at a time   вҶ’ O(NK)         intuitive
  Method 3: Min Heap              вҶ’ O(N log K)    optimal
  Method 4: Divide and Conquer    вҶ’ O(N log K)    recursive
```

---

## Method 1: Collect All Values + Sort

### Algorithm:
```
  1. Traverse all K lists, collect all values
  2. Sort the values
  3. Build new linked list from sorted values
```

### Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};

ListNode* mergeKLists_sort(vector<ListNode*>& lists) {
    vector<int> vals;

    // Collect all values
    for (ListNode* list : lists) {
        ListNode* curr = list;
        while (curr) {
            vals.push_back(curr->val);
            curr = curr->next;
        }
    }

    // Sort
    sort(vals.begin(), vals.end());

    // Build result linked list
    ListNode* dummy = new ListNode(0);
    ListNode* curr = dummy;
    for (int v : vals) {
        curr->next = new ListNode(v);
        curr = curr->next;
    }

    return dummy->next;
}
```

**Time:** O(N log N) where N = total nodes | **Space:** O(N)

## Method 2: Merge Two Lists at a Time

### Algorithm:
```
  Merge lists[0] and lists[1] вҶ’ result
  Merge result and lists[2]   вҶ’ result
  ...
  Merge result and lists[K-1] вҶ’ final result
```

### Code
```cpp
// Merge two sorted linked lists
ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) {
    ListNode* dummy = new ListNode(0);
    ListNode* curr = dummy;

    while (l1 && l2) {
        if (l1->val <= l2->val) {
            curr->next = l1;
            l1 = l1->next;
        } else {
            curr->next = l2;
            l2 = l2->next;
        }
        curr = curr->next;
    }

    curr->next = l1 ? l1 : l2;
    return dummy->next;
}

ListNode* mergeKLists_pairwise(vector<ListNode*>& lists) {
    if (lists.empty()) return nullptr;

    ListNode* result = lists[0];
    for (int i = 1; i < lists.size(); i++) {
        result = mergeTwoLists(result, lists[i]);
    }

    return result;
}
```
```
  K=3 lists of size N/K each:
  Merge 1+2: touches N/K + N/K = 2N/K nodes
  Merge +3:  touches 2N/K + N/K = 3N/K nodes
  Total: 2N/K + 3N/K + ... вүҲ O(NK) in worst case
```

**Time:** O(NK) | **Space:** O(1)

## Method 3: Min Heap (Optimal)

### Algorithm:
```
  1. Push first node of each list into Min Heap
     Heap entry: {node->val, list_index, node pointer}

  2. While heap not empty:
     a. Pop minimum node
     b. Attach to result list
     c. If popped node has next вҶ’ push next node

  3. Return result list
```
```
  Lists: [1вҶ’4вҶ’7], [2вҶ’5вҶ’8], [3вҶ’6вҶ’9]

  Initial heap: {1,0,*1}, {2,1,*2}, {3,2,*3}
        1
       / \
      2   3

  Pop {1}: attach 1, push {4,0,*4}
  heap: {2,1,*2}, {3,2,*3}, {4,0,*4}
  result: 1

  Pop {2}: attach 2, push {5,1,*5}
  heap: {3,2,*3}, {4,0,*4}, {5,1,*5}
  result: 1вҶ’2

  Pop {3}: attach 3, push {6,2,*6}
  heap: {4,0,*4}, {5,1,*5}, {6,2,*6}
  result: 1вҶ’2вҶ’3

  Pop {4}: attach 4, push {7,0,*7}
  Pop {5}: attach 5, push {8,1,*8}
  Pop {6}: attach 6, push {9,2,*9}
  Pop {7}: attach 7, no next
  Pop {8}: attach 8, no next
  Pop {9}: attach 9, no next

  result: 1вҶ’2вҶ’3вҶ’4вҶ’5вҶ’6вҶ’7вҶ’8вҶ’9 вң“
```

### Code
```cpp
#include <bits/stdc++.h>
using namespace std;

struct ListNode {
    int val;
    ListNode* next;
    ListNode(int x) : val(x), next(nullptr) {}
};

// Custom comparator for Min Heap
struct Compare {
    bool operator()(ListNode* a, ListNode* b) {
        return a->val > b->val; // Min Heap: smaller value = higher priority
    }
};

ListNode* mergeKLists_heap(vector<ListNode*>& lists) {
    // Min Heap of ListNode pointers
    priority_queue<ListNode*, vector<ListNode*>, Compare> minPQ;

    // Push first node of each non-empty list
    for (ListNode* list : lists) {
        if (list) minPQ.push(list);
    }

    // Dummy head for result list
    ListNode* dummy = new ListNode(0);
    ListNode* curr = dummy;

    while (!minPQ.empty()) {
        // Get node with smallest value
        ListNode* node = minPQ.top();
        minPQ.pop();

        // Attach to result
        curr->next = node;
        curr = curr->next;

        // Push next node from same list
        if (node->next) {
            minPQ.push(node->next);
        }
    }

    return dummy->next;
}

// Helper to print linked list
void printList(ListNode* head) {
    while (head) {
        cout << head->val;
        if (head->next) cout << " вҶ’ ";
        head = head->next;
    }
    cout << endl;
}

// Helper to build linked list from vector
ListNode* buildList(vector<int> vals) {
    ListNode* dummy = new ListNode(0);
    ListNode* curr = dummy;
    for (int v : vals) {
        curr->next = new ListNode(v);
        curr = curr->next;
    }
    return dummy->next;
}

int main() {
    // Example 1
    vector<ListNode*> lists1 = {
        buildList({1, 4, 7}),
        buildList({2, 5, 8}),
        buildList({3, 6, 9})
    };

    cout << "Example 1: ";
    printList(mergeKLists_heap(lists1));

    // Example 2
    vector<ListNode*> lists2 = {
        buildList({1, 4, 5}),
        buildList({1, 3, 4}),
        buildList({2, 6})
    };

    cout << "Example 2: ";
    printList(mergeKLists_heap(lists2));

    return 0;
}
```

## Method 4: Divide and Conquer

### Algorithm:
```
  Recursively merge pairs of lists:
  Round 1: merge(L0,L1), merge(L2,L3), ...
  Round 2: merge results pairwise...
  ...until one list remains.

  Like Merge Sort but for linked lists!

  K=8 lists:
  Round 1: [L01, L23, L45, L67]
  Round 2: [L0123, L4567]
  Round 3: [L01234567] вң“
```

### Code:
```cpp
ListNode* mergeKLists_divConquer(vector<ListNode*>& lists) {
    if (lists.empty()) return nullptr;

    int interval = 1;
    int n = lists.size();

    while (interval < n) {
        for (int i = 0; i + interval < n; i += 2 * interval) {
            lists[i] = mergeTwoLists(lists[i], lists[i + interval]);
        }
        interval *= 2;
    }

    return lists[0];
}
```
```
  K=4 lists:
  Initial:  [L0, L1, L2, L3]

  interval=1:
    i=0: merge(L0, L1) вҶ’ lists[0] = L01
    i=2: merge(L2, L3) вҶ’ lists[2] = L23
  lists = [L01, L1, L23, L3]

  interval=2:
    i=0: merge(L01, L23) вҶ’ lists[0] = L0123
  lists = [L0123, L1, L23, L3]

  Return lists[0] = L0123 вң“
```

**Time:** O(N log K) | **Space:** O(log K) recursion stack

### Edge Cases:
```
  Empty input:               All empty lists:
  lists = []                 lists = [null, null]
  return null вң“              push nothing вҶ’ heap empty
                             return dummy->next = null вң“

  Single list:               Lists of different sizes:
  lists = [1вҶ’2вҶ’3]            [1вҶ’2вҶ’3вҶ’4вҶ’5], [6], [7вҶ’8]
  push 1, pop 1вҶ’2вҶ’3          Works correctly, heap adjusts вң“
  return 1вҶ’2вҶ’3 вң“

  Single element each:
  [1], [2], [3]
  Result: 1вҶ’2вҶ’3 вң“
```

## Complexity Comparison:
```
  Method              Time          Space    Notes
  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
  1. Collect+Sort     O(N log N)    O(N)     Simple, extra space
  2. Pairwise Merge   O(NK)         O(1)     Slow for large K
  3. Min Heap         O(N log K)    O(K)     Optimal, recommended
  4. Divide+Conquer   O(N log K)    O(log K) Same time, less space

  N = total nodes across all lists
  K = number of lists

  Why O(N log K) for heap method?
  вҶ’ Each of N nodes is pushed/popped once
  вҶ’ Each push/pop on heap of size K = O(log K)
  вҶ’ Total: O(N log K) вң“

  When K << N:
  log K << log N вҶ’ much faster than O(N log N) sort!
```
