## Priority Queue in C++

### What is a Priority Queue?

A Priority Queue is a container adaptor in C++ STL that provides constant time lookup of the largest (or smallest) element. By default, C++ `priority_queue` is a **Max Heap**.
```
  Normal Queue (FIFO):         Priority Queue (Max Heap by default):
  push 3, 1, 4, 1, 5           push 3, 1, 4, 1, 5
  pop → 3, 1, 4, 1, 5          pop → 5, 4, 3, 1, 1
  (arrival order)               (priority/value order)
```

---

### Syntax:
```cpp
#include <queue>

// Max Heap (default)
priority_queue<int> maxPQ;

// Min Heap
priority_queue<int, vector<int>, greater<int>> minPQ;

// Max Heap of pairs
priority_queue<pair<int,int>> maxPairPQ;

// Min Heap of pairs
priority_queue<pair<int,int>, vector<pair<int,int>>, greater<pair<int,int>>> minPairPQ;
```

---

### Core Operations:
```
  Operation     Function       Time        Description
  ─────────────────────────────────────────────────────────
  Insert        push(val)      O(log N)    Add element
  Remove top    pop()          O(log N)    Remove max/min
  Access top    top()          O(1)        View max/min
  Size          size()         O(1)        Number of elements
  Empty check   empty()        O(1)        Is PQ empty?
```

---

### Max Heap (Default):
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    // Max Heap — largest element at top
    priority_queue<int> maxPQ;

    // Insert elements
    maxPQ.push(4);
    maxPQ.push(1);
    maxPQ.push(7);
    maxPQ.push(3);
    maxPQ.push(9);
    maxPQ.push(2);

    cout << "Max Heap — top to bottom: ";
    while (!maxPQ.empty()) {
        cout << maxPQ.top() << " ";  // Access max
        maxPQ.pop();                 // Remove max
    }
    cout << endl;
    // Output: 9 7 4 3 2 1

    return 0;
}
```
```
  After pushing [4,1,7,3,9,2]:

  Internal Max Heap:
           9
          / \
         4   7
        / \ /
       1  3 2

  Pops in order: 9 → 7 → 4 → 3 → 2 → 1 ✓
```

### Min Heap:
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    // Min Heap — smallest element at top
    priority_queue<int, vector<int>, greater<int>> minPQ;

    minPQ.push(4);
    minPQ.push(1);
    minPQ.push(7);
    minPQ.push(3);
    minPQ.push(9);
    minPQ.push(2);

    cout << "Min Heap — top to bottom: ";
    while (!minPQ.empty()) {
        cout << minPQ.top() << " ";
        minPQ.pop();
    }
    cout << endl;
    // Output: 1 2 3 4 7 9

    return 0;
}
```
```
  After pushing [4,1,7,3,9,2]:

  Internal Min Heap:
           1
          / \
         3   2
        / \ /
       4  9 7

  Pops in order: 1 → 2 → 3 → 4 → 7 → 9 ✓
```

### Priority Queue of Pairs:
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    // Max Heap of pairs — sorted by first element, then second
    priority_queue<pair<int,int>> maxPQ;

    maxPQ.push({3, 10});
    maxPQ.push({1, 20});
    maxPQ.push({5, 5});
    maxPQ.push({3, 15});

    cout << "Max Heap pairs (first, second):" << endl;
    while (!maxPQ.empty()) {
        auto [f, s] = maxPQ.top();
        cout << "(" << f << ", " << s << ")" << endl;
        maxPQ.pop();
    }
    // Output: (5,5) → (3,15) → (3,10) → (1,20)
    // Ties broken by second element (larger second = higher priority)

    cout << endl;

    // Min Heap of pairs
    priority_queue<pair<int,int>,
                   vector<pair<int,int>>,
                   greater<pair<int,int>>> minPQ;

    minPQ.push({3, 10});
    minPQ.push({1, 20});
    minPQ.push({5, 5});
    minPQ.push({3, 15});

    cout << "Min Heap pairs (first, second):" << endl;
    while (!minPQ.empty()) {
        auto [f, s] = minPQ.top();
        cout << "(" << f << ", " << s << ")" << endl;
        minPQ.pop();
    }
    // Output: (1,20) → (3,10) → (3,15) → (5,5)

    return 0;
}
```
```
  Pair comparison:
  (3,15) vs (3,10) → first equal → compare second → 15 > 10
  So (3,15) has higher priority in Max Heap ✓
```

### Custom Comparator:
```cpp
#include <bits/stdc++.h>
using namespace std;

struct Task {
    int priority;
    string name;
};

// Custom comparator: lower priority number = higher priority
struct Compare {
    bool operator()(Task& a, Task& b) {
        return a.priority > b.priority; // Min Heap by priority value
    }
};

int main() {
    priority_queue<Task, vector<Task>, Compare> taskPQ;

    taskPQ.push({3, "Low priority task"});
    taskPQ.push({1, "Critical task"});
    taskPQ.push({2, "Medium task"});

    cout << "Task execution order:" << endl;
    while (!taskPQ.empty()) {
        Task t = taskPQ.top();
        taskPQ.pop();
        cout << "Priority " << t.priority << ": " << t.name << endl;
    }
    // Output:
    // Priority 1: Critical task
    // Priority 2: Medium task
    // Priority 3: Low priority task

    return 0;
}
```

### Initializing from a Vector:
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    vector<int> nums = {4, 1, 7, 3, 9, 2};

    // Build Max Heap from vector — O(N)
    priority_queue<int> maxPQ(nums.begin(), nums.end());

    // Build Min Heap from vector — O(N)
    priority_queue<int, vector<int>, greater<int>> minPQ(nums.begin(), nums.end());

    cout << "Max: " << maxPQ.top() << endl;  // 9
    cout << "Min: " << minPQ.top() << endl;  // 1

    return 0;
}
```

## Common Use Cases

### 1. K Largest Elements
```cpp
#include <bits/stdc++.h>
using namespace std;

vector<int> kLargest(vector<int>& nums, int k) {
    // Use Min Heap of size K
    priority_queue<int, vector<int>, greater<int>> minPQ;

    for (int num : nums) {
        minPQ.push(num);
        if (minPQ.size() > k) {
            minPQ.pop(); // Remove smallest
        }
    }

    // Collect K largest
    vector<int> result;
    while (!minPQ.empty()) {
        result.push_back(minPQ.top());
        minPQ.pop();
    }
    return result;
}

int main() {
    vector<int> nums = {3, 1, 5, 12, 2, 11};
    int k = 3;
    vector<int> res = kLargest(nums, k);
    cout << "K largest: ";
    for (int x : res) cout << x << " ";
    // Output: 5 11 12
    return 0;
}
```

### 2. Sort Nearly Sorted Array
```cpp
#include <bits/stdc++.h>
using namespace std;

// Each element is at most k positions away from its sorted position
void sortNearlySorted(vector<int>& arr, int k) {
    priority_queue<int, vector<int>, greater<int>> minPQ;

    int idx = 0;
    for (int i = 0; i <= k && i < arr.size(); i++) {
        minPQ.push(arr[i]);
    }

    for (int i = k + 1; i < arr.size(); i++) {
        arr[idx++] = minPQ.top();
        minPQ.pop();
        minPQ.push(arr[i]);
    }

    while (!minPQ.empty()) {
        arr[idx++] = minPQ.top();
        minPQ.pop();
    }
}

int main() {
    vector<int> arr = {6, 5, 3, 2, 8, 10, 9};
    int k = 3;
    sortNearlySorted(arr, k);
    for (int x : arr) cout << x << " ";
    // Output: 2 3 5 6 8 9 10
    return 0;
}
```

### 3. Merge K Sorted Arrays
```cpp
#include <bits/stdc++.h>
using namespace std;

vector<int> mergeKSorted(vector<vector<int>>& arrays) {
    // {value, array_index, element_index}
    priority_queue<tuple<int,int,int>,
                   vector<tuple<int,int,int>>,
                   greater<>> minPQ;

    // Push first element of each array
    for (int i = 0; i < arrays.size(); i++) {
        if (!arrays[i].empty()) {
            minPQ.push({arrays[i][0], i, 0});
        }
    }

    vector<int> result;
    while (!minPQ.empty()) {
        auto [val, arrIdx, elemIdx] = minPQ.top();
        minPQ.pop();
        result.push_back(val);

        // Push next element from same array
        if (elemIdx + 1 < arrays[arrIdx].size()) {
            minPQ.push({arrays[arrIdx][elemIdx+1], arrIdx, elemIdx+1});
        }
    }

    return result;
}

int main() {
    vector<vector<int>> arrays = {
        {1, 4, 7},
        {2, 5, 8},
        {3, 6, 9}
    };

    vector<int> merged = mergeKSorted(arrays);
    for (int x : merged) cout << x << " ";
    // Output: 1 2 3 4 5 6 7 8 9
    return 0;
}
```

## Max Heap vs Min Heap Quick Reference
```
  Max Heap:                        Min Heap:
  priority_queue<int>              priority_queue<int,
                                     vector<int>,
                                     greater<int>>

  top() = largest element          top() = smallest element
  Used for: K largest,             Used for: K smallest,
            scheduling by          Dijkstra, Prim's,
            highest priority       median finding
```

### Complexity Summary:
```
  Operation              Time        Space
  ─────────────────────────────────────────
  push(val)              O(log N)    O(1)
  pop()                  O(log N)    O(1)
  top()                  O(1)        O(1)
  size()                 O(1)        O(1)
  empty()                O(1)        O(1)
  Build from N elements  O(N)        O(N)

  Overall space of PQ:   O(N)
```
