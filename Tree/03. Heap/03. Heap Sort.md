## Heap Sort

### What is Heap Sort?

Heap Sort is a comparison-based sorting algorithm that uses a **Binary Heap** data structure. It works in two phases:
1. **Build a Max Heap** from the input array.
2. **Extract elements** one by one from the heap to get a sorted array.
```
  Input:  [4, 1, 2, 6, 7, 3, 8, 5]
  Output: [1, 2, 3, 4, 5, 6, 7, 8]
```

---

### Key Idea:
```
  Why Max Heap for ascending sort?

  Max Heap root = largest element
  Swap root with last element → largest goes to correct position
  Reduce heap size, heapify root → next largest bubbles up
  Repeat → array sorted in ascending order ✓

  Min Heap → descending order
  Max Heap → ascending order  ← used here
```

---
## Appraoch
### Algorithm:

**Phase 1 — Build Max Heap:**
- Start from last internal node `(n/2 - 1)` down to index `0`.
- Call `Heapify()` at each node (bottom-up).

**Phase 2 — Sort:**
- Swap root (max element) with last element.
- Reduce heap size by 1.
- Call `Heapify(0)` to restore max heap.
- Repeat until heap size = 1.
```
  Phase 1: Build Max Heap
  ┌──────────────────────────────────────┐
  │ for i = n/2-1 down to 0:            │
  │     Heapify(arr, n, i)               │
  └──────────────────────────────────────┘

  Phase 2: Extract max repeatedly
  ┌──────────────────────────────────────┐
  │ for i = n-1 down to 1:              │
  │     swap(arr[0], arr[i])             │
  │     Heapify(arr, i, 0)  ← size = i  │
  └──────────────────────────────────────┘
```

### C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

// Heapify subtree rooted at index i
// n = current heap size
void Heapify(int arr[], int n, int i) {
    int largest = i;        // Assume root is largest
    int left    = 2*i + 1;  // Left child
    int right   = 2*i + 2;  // Right child

    // Check if left child is larger than root
    if (left < n && arr[left] > arr[largest])
        largest = left;

    // Check if right child is larger than current largest
    if (right < n && arr[right] > arr[largest])
        largest = right;

    // If largest is not root, swap and continue heapifying
    if (largest != i) {
        swap(arr[i], arr[largest]);
        Heapify(arr, n, largest);
    }
}

// Main Heap Sort function
void HeapSort(int arr[], int n) {
    // Phase 1: Build Max Heap (bottom-up)
    for (int i = n/2 - 1; i >= 0; i--) {
        Heapify(arr, n, i);
    }

    // Phase 2: Extract elements from heap one by one
    for (int i = n - 1; i > 0; i--) {
        // Move current root (max) to end
        swap(arr[0], arr[i]);

        // Heapify the reduced heap
        Heapify(arr, i, 0);
    }
}

void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++)
        cout << arr[i] << " ";
    cout << endl;
}

int main() {
    int arr[] = {4, 1, 2, 6, 7, 3, 8, 5};
    int n = sizeof(arr) / sizeof(arr[0]);

    cout << "Before Heap Sort: ";
    printArray(arr, n);

    HeapSort(arr, n);

    cout << "After  Heap Sort: ";
    printArray(arr, n);

    return 0;
}
```

### Step-by-Step Trace:

### Phase 1 — Build Max Heap
```
  Input: [4, 1, 2, 6, 7, 3, 8, 5]
  n=8, start from index (8/2)-1 = 3

  Initial tree:
           4  (0)
          / \
         1   2  (1,2)
        / \ / \
       6  7 3  8  (3,4,5,6)
      /
     5  (7)

  Heapify(3): node=6, children=5
    max(6,5) = 6 → no swap

  Heapify(2): node=2, children=3,8
    max(2,3,8) = 8 → SWAP(2,8)
           4
          / \
         1   8
        / \ / \
       6  7 3  2

  Heapify(1): node=1, children=6,7
    max(1,6,7) = 7 → SWAP(1,7)
           4
          / \
         7   8
        / \ / \
       6  1 3  2

    Heapify(4): node=1, no children bigger → stop

  Heapify(0): node=4, children=7,8
    max(4,7,8) = 8 → SWAP(4,8)
           8
          / \
         7   4
        / \ / \
       6  1 3  2

    Heapify(2): node=4, children=3,2
    max(4,3,2) = 4 → no swap

  Max Heap built: [8, 7, 4, 6, 1, 3, 2, 5]

  Wait — let's redo carefully:
  After all heapify calls:
  [8, 7, 4, 6, 1, 3, 2, 5]

           8
          / \
         7   4
        / \ / \
       6  1 3  2
      /
     5
```

### Phase 2 — Extract & Sort
```
  Step 1: swap(arr[0], arr[7]) → [5,7,4,6,1,3,2, |8|]
  Heapify(0, size=7):
           7
          / \
         6   4
        / \ /
       5  1 3
  arr: [7,6,4,5,1,3,2, |8|]

  Step 2: swap(arr[0], arr[6]) → [2,6,4,5,1,3, |7,8|]
  Heapify(0, size=6):
           6
          / \
         5   4
        / \
       2   1
  arr: [6,5,4,2,1,3, |7,8|]

  Step 3: swap(arr[0], arr[5]) → [3,5,4,2,1, |6,7,8|]
  Heapify(0, size=5):
           5
          / \
         3   4
        / \
       2   1
  arr: [5,3,4,2,1, |6,7,8|]

  Step 4: swap(arr[0], arr[4]) → [1,3,4,2, |5,6,7,8|]
  Heapify(0, size=4):
           4
          / \
         3   1
        /
       2
  arr: [4,3,1,2, |5,6,7,8|]

  Step 5: swap(arr[0], arr[3]) → [2,3,1, |4,5,6,7,8|]
  Heapify(0, size=3):
           3
          / \
         2   1
  arr: [3,2,1, |4,5,6,7,8|]

  Step 6: swap(arr[0], arr[2]) → [1,2, |3,4,5,6,7,8|]
  Heapify(0, size=2):
           2
          /
         1
  arr: [2,1, |3,4,5,6,7,8|]

  Step 7: swap(arr[0], arr[1]) → [1, |2,3,4,5,6,7,8|]
  Heap size=1 → done

  Final: [1, 2, 3, 4, 5, 6, 7, 8] ✓
```

### Comparison with Other Sorting Algorithms
```
  Algorithm     Best      Average    Worst     Space   Stable?
  ──────────────────────────────────────────────────────────────
  Heap Sort     O(N logN) O(N logN)  O(N logN) O(1)    No
  Merge Sort    O(N logN) O(N logN)  O(N logN) O(N)    Yes
  Quick Sort    O(N logN) O(N logN)  O(N²)     O(logN) No
  Bubble Sort   O(N)      O(N²)      O(N²)     O(1)    Yes

  Heap Sort advantage: O(1) space + guaranteed O(N log N)
  Heap Sort drawback:  Not cache-friendly, not stable
```

---

### Complexity Analysis:

**Time Complexity:**
```
  Phase 1 (Build Max Heap):  O(N)       ← bottom-up heapify
  Phase 2 (N extractions):   O(N log N) ← each extraction = O(log N)

  Total: O(N log N) in all cases (best, average, worst)
```

**Space Complexity:** O(log N) for the recursive Heapify call stack.
- No extra array needed — sorting is done **in place**.
- Recursion depth = height of heap = O(log N).
```
  In-place sorting:
  arr = [4, 1, 2, 6, 7, 3, 8, 5]
         ↑ heap portion | sorted portion ↑

  Heap shrinks as sorted portion grows:
  [8,7,4,6,1,3,2 | 5] → [7,6,4,5,1,3 | 2,5] → ...
```
