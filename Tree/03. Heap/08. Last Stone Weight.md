## Last Stone Weight

**Problem Statement:**   
You are given an array of integers `stones` where `stones[i]` is the weight of the `i`th stone. Each turn, choose the **two heaviest stones** and smash them:
- If `x == y`: both stones are destroyed.
- If `x != y`: stone of weight `x` is destroyed, stone of weight `y` becomes `y - x`.

Return the weight of the **last remaining stone**, or `0` if none remain.

---

>### Examples:

**Example 1:**
- **Input:** `stones = [2, 7, 4, 1, 8, 1]`
- **Output:** `1`
```
  stones = [2, 7, 4, 1, 8, 1]

  Round 1: Smash 8 and 7 → |8-7|=1   → [2, 4, 1, 1, 1]
  Round 2: Smash 4 and 2 → |4-2|=2   → [2, 1, 1, 1]
  Round 3: Smash 2 and 1 → |2-1|=1   → [1, 1, 1]
  Round 4: Smash 1 and 1 → equal=0   → [1]
  Round 5: Only 1 left   → return 1 ✓
```

**Example 2:**
- **Input:** `stones = [1]`
- **Output:** `1`
```
  Only one stone → nothing to smash → return 1 ✓
```

**Example 3:**
- **Input:** `stones = [3, 3]`
- **Output:** `0`
```
  Smash 3 and 3 → equal → both destroyed → return 0 ✓
```

**Example 4:**
- **Input:** `stones = [10, 4, 2, 10]`
- **Output:** `0`
```
  Round 1: Smash 10 and 10 → equal → destroyed → [4, 2]
  Round 2: Smash 4  and 2  → 4-2=2 → [2]
  Return 2

  Wait, actual answer: let's retrace
  stones = [10, 4, 2, 10]
  Round 1: 10 and 10 → both destroyed → [4, 2]
  Round 2: 4 and 2 → 4-2=2 → [2]
  Return 2
```

---

### Key Insight:
```
  At each step → always pick TWO HEAVIEST stones
  → MAX HEAP is perfect for this!

  top() = heaviest stone in O(1)
  pop()  = remove heaviest in O(log N)
  push() = add new stone in O(log N)

  Repeat until the heap has ≤ 1 stone.
```

---
## Approach
### Algorithm:
```
  1. Build Max Heap from stones array
  2. While heap.size() > 1:
       a. y = heap.top(); heap.pop()  ← heaviest
       b. x = heap.top(); heap.pop()  ← second heaviest
       c. If y != x: heap.push(y - x) ← remainder
       d. If y == x: both destroyed, don't push
  3. Return heap.empty() ? 0 : heap.top()
```
```
  Max Heap trace for [2, 7, 4, 1, 8, 1]:

  Initial heap: [8, 7, 4, 2, 1, 1]
        8
       / \
      7   4
     / \ /
    2  1 1

  Step 1: pop 8, pop 7 → 8≠7 → push 1
  heap: [4, 2, 1, 1, 1]

  Step 2: pop 4, pop 2 → 4≠2 → push 2
  heap: [2, 1, 1, 1]

  Step 3: pop 2, pop 1 → 2≠1 → push 1
  heap: [1, 1, 1]

  Step 4: pop 1, pop 1 → 1==1 → don't push
  heap: [1]

  size=1 → return heap.top() = 1 ✓
```

---

### C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

int lastStoneWeight(vector<int>& stones) {
    // Build Max Heap — O(N)
    priority_queue<int> maxPQ(stones.begin(), stones.end());

    // Keep smashing until 0 or 1 stone remains
    while (maxPQ.size() > 1) {
        // Pick two heaviest stones
        int y = maxPQ.top(); maxPQ.pop(); // Heaviest
        int x = maxPQ.top(); maxPQ.pop(); // Second heaviest

        // If not equal, push remainder back
        if (y != x) {
            maxPQ.push(y - x);
        }
        // If equal, both destroyed — don't push
    }

    // Return last stone or 0 if none
    return maxPQ.empty() ? 0 : maxPQ.top();
}

int main() {
    vector<int> s1 = {2, 7, 4, 1, 8, 1};
    cout << lastStoneWeight(s1) << endl; // 1

    vector<int> s2 = {1};
    cout << lastStoneWeight(s2) << endl; // 1

    vector<int> s3 = {3, 3};
    cout << lastStoneWeight(s3) << endl; // 0

    vector<int> s4 = {10, 4, 2, 10};
    cout << lastStoneWeight(s4) << endl; // 2

    vector<int> s5 = {1, 3};
    cout << lastStoneWeight(s5) << endl; // 2

    return 0;
}
```

### Detailed Step-by-Step Trace:
```
  Input: [2, 7, 4, 1, 8, 1]

  ── Build Max Heap ──
  heap = [8, 7, 4, 2, 1, 1]
        8
       / \
      7   4
     / \ /
    2  1 1

  ── Round 1 ──
  y = 8, x = 7
  8 ≠ 7 → push (8-7) = 1
  heap = [4, 2, 1, 1, 1]
        4
       / \
      2   1
     / \
    1   1

  ── Round 2 ──
  y = 4, x = 2
  4 ≠ 2 → push (4-2) = 2
  heap = [2, 1, 1, 1]
        2
       / \
      1   1
     /
    1

  ── Round 3 ──
  y = 2, x = 1
  2 ≠ 1 → push (2-1) = 1
  heap = [1, 1, 1]
        1
       / \
      1   1

  ── Round 4 ──
  y = 1, x = 1
  1 == 1 → both destroyed, don't push
  heap = [1]

  ── Done ──
  heap.size() = 1 → return 1 ✓
```

---

### Edge Cases:
```
  Single stone:              Two equal stones:
  [5]                        [4, 4]
  size=1 → return 5 ✓        pop 4, pop 4 → equal
                             don't push → heap empty
                             return 0 ✓

  All equal stones:          Descending order:
  [2, 2, 2, 2]               [10, 8, 6, 4, 2]
  Round 1: 2==2 → []         Round 1: pop 10, 8 → push 2 → [6,4,2,2]
  Round 2: 2==2 → []         Round 2: pop 6,  4 → push 2 → [2,2,2]
  return 0 ✓                 Round 3: pop 2,  2 → equal  → [2]
                             Round 4: pop 2,  2 → equal  → []
                             return 0

  Two unequal stones:
  [7, 3]
  pop 7, pop 3 → 7-3=4 → push 4
  heap=[4] → return 4 ✓
```

### Complexity Analysis:

**Time Complexity:** O(N log N)
- Building Max Heap: O(N)
- At most N-1 rounds (each round removes at least one stone):
  - Each round: 2 pops + at most 1 push = O(log N)
- Total: O(N) + O(N × log N) = **O(N log N)**

**Space Complexity:** O(N) for the heap.
```
  N stones → at most N-1 smashing rounds
  Each round: O(log N) heap operations
  Total: O(N log N) ✓

  In-place? No — uses extra O(N) heap space
  But the original array is not modified ✓
```
