## Minimum Time Taken to BURN the Binary Tree from a Node

**Problem Statement:**   
Given a target node data and the root of a binary tree. If the target is set on fire, determine the shortest amount of time needed to burn the entire binary tree. It is known that in 1 second, all nodes connected to a given node get burned — that is, its left child, right child, and parent.

---

>### Examples:

**Example 1:**
- **Input:** `root = [1, 2, 3, 4, null, 5, 6, null, 7]`, `target = 1`
- **Output:** `3`
- **Explanation:** Node 1 is set on fire. In 1st second, it burns 2 and 3. In 2nd second it burns 4, 5, 6. In 3rd second it burns 7.
```
           1  ← fire starts here
          / \
         2   3         t=1: 2, 3 burn
        /   / \
       4   5   6       t=2: 4, 5, 6 burn
        \
         7             t=3: 7 burns

  Total time = 3 ✓
```

**Example 2:**
- **Input:** `root = [1, 2, 3, null, 5, null, 4]`, `target = 4`
- **Output:** `4`
- **Explanation:** Node 4 is set on fire. In 1st second, it burns 3. In 2nd second it burns 1. In 3rd second it burns 2. In 4th second it burns 5.
```
       1
      / \
     2   3              Fire starts at 4 (right child of 3)
      \    \
       5    4  ← fire

  t=1: 3 burns  (parent of 4)
  t=2: 1 burns  (parent of 3)
  t=3: 2 burns  (child of 1)
  t=4: 5 burns  (child of 2)

  Total time = 4 ✓
```

---

### Key Insight:

In a binary tree, fire can spread in **three directions**: left child, right child, and parent. Since binary trees don't store parent links, we convert the tree into an **undirected graph** (adjacency list) and run BFS from the target node, counting levels until all nodes are burned.
```
  Binary Tree:              Undirected Graph:

       1                    1 — 2 — 4
      / \                   |       |
     2   3         →        3   5   7
      \   \                 |
       5   4                4 (target)

  Fire spreads 1 hop per second in any direction ✓
```

---

### Algorithm:

**Step 1 — Build Undirected Graph:**
Use DFS to traverse the tree and create bidirectional edges between each node and its children.
```
  buildGraph(1, null):
    graph[1] ↔ graph[2]
    graph[1] ↔ graph[3]
    graph[2] ↔ graph[4]
    graph[3] ↔ graph[5]
    graph[3] ↔ graph[6]
    graph[4] ↔ graph[7]
```

**Step 2 — BFS from Target:**
Start BFS from the target node, spreading fire level by level. Increment time only when new nodes are burned.
```
  BFS from target=1, k=?:

  t=0: queue=[1],       visited={1}
  t=1: burn 2, 3        queue=[2,3],   burned=true → time=1
  t=2: burn 4, 5, 6     queue=[4,5,6], burned=true → time=2
  t=3: burn 7           queue=[7],     burned=true → time=3
  t=4: queue empty      burned=false → stop

  Return time = 3 ✓
```

**Step 3 — Return time** once the queue becomes empty.

---

### C++ Code:
```cpp
# Minimum Time to Burn Binary Tree — Full Code
#include <bits/stdc++.h>
using namespace std;

// Definition of Binary Tree Node
struct BinaryTreeNode {
    int data;
    BinaryTreeNode* left;
    BinaryTreeNode* right;
    BinaryTreeNode(int x) : data(x), left(nullptr), right(nullptr) {}
};

// ─────────────────────────────────────────────
// Step 1: BFS to build parent map + find target
// ─────────────────────────────────────────────
BinaryTreeNode* bfsToMapParents(BinaryTreeNode* root,map<BinaryTreeNode*, BinaryTreeNode*>& mpp,int start){
    queue<BinaryTreeNode*> q;
    q.push(root);
    BinaryTreeNode* res = nullptr;

    while (!q.empty()) {
        BinaryTreeNode* node = q.front();
        q.pop();

        // Save pointer to the target node
        if (node->data == start) res = node;

        // Map left child → parent
        if (node->left) {
            mpp[node->left] = node;
            q.push(node->left);
        }

        // Map right child → parent
        if (node->right) {
            mpp[node->right] = node;
            q.push(node->right);
        }
    }

    return res;
}

// ─────────────────────────────────────────────
// Step 2: BFS from target to find burn time
// ─────────────────────────────────────────────
int findMaxDistance(map<BinaryTreeNode*, BinaryTreeNode*>& mpp, BinaryTreeNode* target){
    queue<BinaryTreeNode*> q;
    q.push(target);

    map<BinaryTreeNode*, int> vis;
    vis[target] = 1;

    int maxi = 0;

    while (!q.empty()) {
        int sz = q.size();
        int fl = 0;  // flag: did fire spread this round?

        for (int i = 0; i < sz; i++) {
            auto node = q.front();
            q.pop();

            // Spread fire LEFT
            if (node->left && !vis[node->left]) {
                fl = 1;
                vis[node->left] = 1;
                q.push(node->left);
            }

            // Spread fire RIGHT
            if (node->right && !vis[node->right]) {
                fl = 1;
                vis[node->right] = 1;
                q.push(node->right);
            }

            // Spread fire UP (to parent)
            if (mpp[node] && !vis[mpp[node]]) {
                fl = 1;
                vis[mpp[node]] = 1;
                q.push(mpp[node]);
            }
        }

        // Only increment time if the fire actually spreads
        if (fl) maxi++;
    }

    return maxi;
}

// ─────────────────────────────────────────────
// Step 3: Main function
// ─────────────────────────────────────────────
int timeToBurnTree(BinaryTreeNode* root, int start) {
    // Build parent map and find the target node
    map<BinaryTreeNode*, BinaryTreeNode*> mpp;
    BinaryTreeNode* target = bfsToMapParents(root, mpp, start);

    // BFS from target to find minimum burn time
    int maxi = findMaxDistance(mpp, target);

    return maxi;
}

// ─────────────────────────────────────────────
// Driver Code
// ─────────────────────────────────────────────
int main() {
    // Construct Binary Tree:
    //        1
    //       / \
    //      2   3
    //     /   / \
    //    4   5   6
    //     \
    //      7

    BinaryTreeNode* root = new BinaryTreeNode(1);
    root->left           = new BinaryTreeNode(2);
    root->right          = new BinaryTreeNode(3);
    root->left->left     = new BinaryTreeNode(4);
    root->right->left    = new BinaryTreeNode(5);
    root->right->right   = new BinaryTreeNode(6);
    root->left->left->right = new BinaryTreeNode(7);

    int start = 1;

    cout << "Minimum time to burn the tree from node "
         << start << ": "
         << timeToBurnTree(root, start)
         << " seconds" << endl;

    // Test 2: target = 4
    start = 4;
    cout << "Minimum time to burn the tree from node "
         << start << ": "
         << timeToBurnTree(root, start)
         << " seconds" << endl;

    return 0;
}
```
---

### Edge Cases:
```
  Single node:            Target is a leaf:
       1                         1
                                / \
  target=1                     2   3
  No neighbors → time=0 ✓
                          target=2:
                          t=1: 1 burns
                          t=2: 3 burns
                          time=2 ✓

  Linear (skewed) tree:
  1 → 2 → 3 → 4 → 5

  target=3 (middle):
  t=1: 2, 4 burn
  t=2: 1, 5 burn
  time=2 ✓
```

---

### Complexity Analysis:
- **Time Complexity:** O(N). We perform two full traversals of the tree — a DFS to build the graph (O(N)) and a BFS to simulate fire spreading (O(N)). Total = O(N).
- **Space Complexity:** O(N). The adjacency list stores O(N) edges, the visited set holds up to O(N) nodes, and the BFS queue can hold up to O(N) nodes in the worst case.
