## Flatten Binary Tree to Linked List

**Problem Statement:**  
Given a Binary Tree, convert it to a Linked List where the linked list nodes follow the same order as the pre-order traversal of the binary tree. Use the right pointer as the 'next' pointer and set the left pointer to null. Do this in place without creating extra nodes.

---

**Example 1:**
- **Input:** Binary Tree: 
```               
       1                 
      / \                  
     2   5                
    / \   \               
   3   4   6                 
            \                 
             7
```
- **Output:** `1 в†’ 2 в†’ 3 в†’ 4 в†’ 5 в†’ 6 в†’ 7`
```

```

## Brute Force Approach

### Algorithm:

Uses **reverse preorder (Right в†’ Left в†’ Root)** with a `prev` pointer. Processing in reverse ensures we can link nodes correctly.

- Initialize `prev = null`.
- Recursively flatten right subtree, then left subtree.
- Set `root->right = prev`, `root->left = null`.
- Update `prev = root`.
```
  Reverse preorder of [1,2,3,4,5,6,7]:
  Process: 7в†’6в†’5в†’4в†’3в†’2в†’1

  prev=null
  Visit 7: right=null, left=null, prev=7
  Visit 6: right=7,    left=null, prev=6
  Visit 5: right=6,    left=null, prev=5
  ...builds chain from right to left вњ“
```

### C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

struct TreeNode {
    int val;
    TreeNode *left, *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    TreeNode* prev = nullptr;

    void flatten(TreeNode* root) {
        if (root == nullptr) return;
        // Process RIGHT first, then LEFT (reverse preorder)
        flatten(root->right);
        flatten(root->left);
        // Link current to previously processed node
        root->right = prev;
        root->left  = nullptr;
        prev = root;
    }
};

void printFlattenTree(TreeNode* root) {
    if (!root) return;
    cout << root->val << " ";
    printFlattenTree(root->right);
}

int main() {
    TreeNode* root = new TreeNode(1);
    root->left = new TreeNode(2);
    root->right = new TreeNode(3);
    root->left->left = new TreeNode(4);
    root->left->right = new TreeNode(5);
    root->left->right->right = new TreeNode(6);
    root->right->right = new TreeNode(7);
    root->right->left = new TreeNode(8);

    Solution sol;
    sol.flatten(root);

    cout << "After Flatten: ";
    printFlattenTree(root);
    cout << endl;

    return 0;
}
```

### Complexity Analysis:

- **Time Complexity:** O(N), each node is visited once.
- **Space Complexity:** O(H), where H is the height of the tree due to the recursive call stack. In the ideal case H = logв‚‚N, worst case H = N (skewed tree).

## Better Approach (Iterative Stack)

### Algorithm:

Uses a stack to simulate preorder traversal. At each node, link its right pointer to the next node (top of stack).

- Push root onto stack.
- While the stack is not empty:
  - Pop current node.
  - Push the right child first, then the left child.
  - If stack not empty, `cur->right = stack.top()`.
  - Set `cur->left = null`.
```
  Tree: 1(left=2, right=3), 2(left=4, right=5)

  stack=[1]
  Pop 1: push right(3), push left(2)  в†’ stack=[3,2]
         cur->right = top=2, left=null
  Pop 2: push right(5), push left(4)  в†’ stack=[3,5,4]
         cur->right = top=4, left=null
  Pop 4: no children                  в†’ stack=[3,5]
         cur->right = top=5, left=null
  Pop 5: no children                  в†’ stack=[3]
         cur->right = top=3, left=null
  Pop 3: no children                  в†’ stack=[]
         left=null
  Result: 1в†’2в†’4в†’5в†’3 вњ“
```

### C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

struct TreeNode {
    int val;
    TreeNode *left, *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    void flatten(TreeNode* root) {
        if (root == nullptr) return;

        stack<TreeNode*> st;
        st.push(root);

        while (!st.empty()) {
            TreeNode* cur = st.top();
            st.pop();

            // Push right first (processed later), then left (processed next)
            if (cur->right != nullptr) st.push(cur->right);
            if (cur->left  != nullptr) st.push(cur->left);

            // Link right to the next node in preorder
            if (!st.empty()) cur->right = st.top();

            cur->left = nullptr;
        }
    }
};

void printFlattenTree(TreeNode* root) {
    if (!root) return;
    cout << root->val << " ";
    printFlattenTree(root->right);
}

int main() {
    TreeNode* root = new TreeNode(1);
    root->left = new TreeNode(2);
    root->right = new TreeNode(3);
    root->left->left = new TreeNode(4);
    root->left->right = new TreeNode(5);

    Solution sol;
    sol.flatten(root);

    cout << "After Flatten: ";
    printFlattenTree(root);
    cout << endl;

    return 0;
}
```

### Complexity Analysis:

- **Time Complexity:** O(N), each node is visited once.
- **Space Complexity:** O(H), stack holds at most H nodes at any time.

## Optimal Approach (O(1) Space)


### Key Insight:
```
  Target: Preorder traversal (Root в†’ Left в†’ Right)
          stored using only right pointers, left = null

  The key observation (for optimal approach):
  For any node with a left child:
    1. Find the RIGHTMOST node of the left subtree
    2. Connect it to the current's right child
    3. Move the entire left subtree to the right
    4. Null the left pointer

       1                    1
      / \                    \
     2   3      в†’             2
    / \                      / \
   4   5                    4   5
                                 \
                                  3   в†ђ right subtree attached here
```
### Algorithm:

Uses the same idea as Morris Traversal вЂ” no stack or recursion needed.

- While `curr` is not null:
  - If `curr` has a left child:
    - Find **rightmost node** in left subtree.
    - Connect it to `curr->right`.
    - Move left subtree to the right.
    - Set left to null.
  - Move to `curr->right`.
```
  Step-by-step on tree:
       1
      / \
     2   3
    / \
   4   5

  curr=1: left=2 exists
    rightmost of left(2) = 5
    5->right = 1->right(3)     в†’ attach right subtree
    1->right = 1->left(2)      в†’ move left subtree right
    1->left  = null
    Tree:  1в†’2, 2 has left=4, right=5, 5->right=3

  curr=2: left=4 exists
    rightmost of left(4) = 4
    4->right = 2->right(5)
    2->right = 4
    2->left  = null
    Tree:  1в†’2в†’4в†’5в†’3

  curr=4: left=null в†’ move right
  curr=5: left=null в†’ move right
  curr=3: left=null в†’ move right
  curr=null в†’ STOP

  Result: 1в†’2в†’4в†’5в†’3 вњ“
```

### C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

struct TreeNode {
    int val;
    TreeNode *left, *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    void flatten(TreeNode* root) {
        TreeNode* curr = root;

        while (curr) {
            if (curr->left) {
                // Find rightmost node in left subtree
                TreeNode* pre = curr->left;
                while (pre->right) {
                    pre = pre->right;
                }

                // Step 1: Connect rightmost в†’ curr's right subtree
                pre->right = curr->right;

                // Step 2: Move left subtree to right
                curr->right = curr->left;

                // Step 3: Null the left pointer
                curr->left = NULL;
            }

            // Move to next node
            curr = curr->right;
        }
    }
};

void printPreorder(TreeNode* root) {
    if (!root) return;
    cout << root->val << " ";
    printPreorder(root->left);
    printPreorder(root->right);
}

void printFlattenTree(TreeNode* root) {
    if (!root) return;
    cout << root->val << " ";
    printFlattenTree(root->right);
}

int main() {
    TreeNode* root = new TreeNode(1);
    root->left = new TreeNode(2);
    root->right = new TreeNode(3);
    root->left->left = new TreeNode(4);
    root->left->right = new TreeNode(5);
    root->left->right->right = new TreeNode(6);
    root->right->right = new TreeNode(7);
    root->right->left = new TreeNode(8);

    Solution sol;

    cout << "Preorder Before: ";
    printPreorder(root);
    cout << endl;

    sol.flatten(root);

    cout << "After Flatten:   ";
    printFlattenTree(root);
    cout << endl;

    return 0;
}
```

### Complexity Analysis:

- **Time Complexity:** O(2N) в‰€ O(N). Each node is visited at most twice вЂ” once when traversed and once when found as the rightmost node.
- **Space Complexity:** O(1). Only a constant number of pointers are used. No stack, no recursion.
