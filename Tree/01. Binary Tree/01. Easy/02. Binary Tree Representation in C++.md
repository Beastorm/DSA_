
## Binary Tree Representation in C++

A Binary Tree consists of nodes. Each node contains a data element and pointers to its children.

### 1. The Basic Structure
In C++, we use a `struct` or `class` to define the node. Pointers are used because the children are separate objects in memory.

### C++ Code:
```cpp
#include <iostream>

struct Node {
    int data;           // The value stored in the node
    Node* left;         // Pointer to the left child
    Node* right;        // Pointer to the right child

    // Constructor to easily initialize a new node
    Node(int val) {
        data = val;
        left = nullptr;
        right = nullptr;
    }
};
```

### 2. Memory Visualization
When you create a tree, each node is allocated on the heap. The pointers hold the memory addresses of the children.

```
        [ Node ]
        | Data |
        |------|
       | L* | R* |
        /      \
    [Node]    [Node]
    |Data|    |Data|
    |----|    |----|
    |0| 0|    |0| 0|  <- Nullptrs (Leaf Nodes)
```

### 3. Creating a Simple Tree
Here is how you manually link nodes to form a tree in your main function:
``` CPP
int main() {
    // 1. Create the root node
    Node* root = new Node(1);

    // 2. Add children
    root->left = new Node(2);
    root->right = new Node(3);

    // 3. Add a grandchild to the left child
    root->left->left = new Node(4);

    /*
          1
         / \
        2   3
       /
      4
    */

    return 0;
}
```
