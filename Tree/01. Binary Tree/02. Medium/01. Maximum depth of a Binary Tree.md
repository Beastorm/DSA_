## Maximum depth of a Binary Tree

**Problem Statement:**  
Given the root of a Binary Tree, return the height of the tree. The height of the tree is equal to the number of nodes on the longest path from the root to a leaf.

---

>### Examples:

### Example 1:
**Input:** Binary Tree: 1 2 5 -1 -1 4 6 5   
**Output:** 4   
**Explanation:**   
In the example above, the height of the binary tree is the length of the longest path from the root node: 1 -> 5 -> 4 -> 5.

## Approach - Recursive

### The Intuition:
The height of a tree is simply 1 (the current node) + the height of its tallest child. It’s a bottom-up calculation where each leaf returns 0 to its parent, and the parent picks the maximum.

### The Algorithm:
- Base Case: If the node is NULL, return 0.
- Recurse: Call maxDepth on the Left child and the right child.
- Return: Take the max of the two results and add 1.

``` cpp

#include <bits/stdc++.h>
using namespace std;

// Node structure for the binary tree
struct Node {
    int data;
    Node* left;
    Node* right;
    // Constructor to initialize
    // the node with a value
    Node(int val) : data(val), left(nullptr), right(nullptr) {}
};

class Solution{
public:
    // Function to find the
    // maximum depth of a binary tree
    int maxDepth(Node* root){
        // If the root is NULL
        // (empty tree), depth is 0
        if(root == NULL){
            return 0;
        }
        
        // Recursive call to find the
        // maximum depth of the left subtree
        int lh = maxDepth(root->left);
        
        // Recursive call to find the
        // maximum depth of the right subtree
        int rh = maxDepth(root->right);
        
        // Return the maximum depth of the
        // tree, adding 1 for the current node
        return 1 + max(lh, rh);
    }
};

// Main function
int main() {
    // Creating a sample binary tree
    Node* root = new Node(1);
    root->left = new Node(2);
    root->right = new Node(3);
    root->left->left = new Node(4);
    root->left->right = new Node(5);
    root->left->right->right = new Node(6);
    root->left->right->right->right = new Node(7);
    

    Solution solution;
    int depth = solution.maxDepth(root);

    cout << "Maximum depth of the binary tree: " << depth << endl;

    return 0;
}
```
### Complexity Analysis:
- **Time:** $O(n)$ because we visit every node exactly once.
- **Space:** $O(h)$ for the function call stack, where $h$ is the height. In a skewed tree, this could be $O(n)$.

## Approach - Iterative
### The Intuition:
Think of this like peeling an onion. You count how many layers (levels) the tree has. You process the entire first level, then the entire second, and so on. The number of times you "peel" a level is your maximum depth.

### The Algorithm:

- Initialize: Push the root into a Queue and set depth = 0.

- Level Loop: While the queue isn't empty:
  - Increment depth.
  - Get the current Queue Size (this is the number of nodes at this specific level).
  
  - Pop that many nodes, and for each, push its children into the queue.

- Result: Return depth once the queue is empty.

``` cpp
        // maximum depth of the right subtree

``` 
### Complexity Analysis:
- **Time Complexity:** O(N), each node is processed once in Level Order Traversal.
- **Space Complexity:** O(N), in the worst case, a maximum of N/2 nodes can be present in the queue.
