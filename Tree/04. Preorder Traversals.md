## Preorder Traversals

**Problem Statement:**  
Given the root of a Binary Tree, return the preorder.

---

>### Examples:
### Example 1:
**Input:** Binary Tree: 4 2 5 3 -1 7 6 -1 9 -1 -1 8 -1 1
```
            4 (Root)
           / \
          2   5
         /   / \
        3   7   6
         \     /
          9   8
           \
            1
```
**Output:** Preorder: [4, 2, 3, 9, 1, 5, 7, 6, 8], Inorder: [3, 1, 9, 2, 4, 7, 5, 8, 6], Postorder: [1, 9, 3, 2, 7, 8, 6, 5, 4] 


---

### Example 2:
**Input:** Binary Tree: 1 2 3 4 5 6 7 -1 -1 8 -1 -1 -1 9 10
```
              1 (Root)
            /   \
           2     3
          / \   / \
         4   5 6   7
            /     / \
           8     9   10
```
**Output:** Preorder: [1, 2, 4, 5, 8, 3, 6, 7, 9, 10], Inorder: [4, 2, 8, 5, 1, 6, 3, 9, 7, 10], Postorder: [4, 8, 5, 2, 6, 9, 10, 7, 3, 1] 

---

### C++ Code:
``` cpp
#include <iostream>
#include <vector>

using namespace std;

// Standard Binary Tree Node
struct Node {
    int data;
    Node* left;
    Node* right;

    Node(int val) {
        data = val;
        left = nullptr;
        right = nullptr;
    }
};

class Solution {
public:
    // Helper function to handle the recursion
    void solve(Node* root, vector<int>& ans) {
        // Base Case: If we hit a null node, just go back
        if (root == nullptr) {
            return;
        }

        // 1. Root: Visit the current node first
        ans.push_back(root->data);

        // 2. Left: Then dive into the left child
        solve(root->left, ans);

        // 3. Right: Finally, dive into the right child
        solve(root->right, ans);
    }

    vector<int> preorder(Node* root) {
        vector<int> ans;
        solve(root, ans);
        return ans;
    }
};

int main() {
    // Creating a sample tree
    Node* root = new Node(1);
    root->left = new Node(2);
    root->right = new Node(3);
    root->left->left = new Node(4);
    root->left->right = new Node(5);

    Solution obj;
    vector<int> result = obj.preorder(root);

    // Printing the result
    cout << "Preorder Traversal: ";
    for (int i : result) {
        cout << i << " ";
    }
    cout << endl;

    return 0;
}
```

### Complexity Analysis:
- **Time Complexity:** O(N)
  - **Reason:** The algorithm visits every node exactly once.
  - **Operation:** In each call, we perform constant-time tasks—checking for nullptr and storing the node's value.
  - **Total:** Since there are N nodes, the complexity is linear.
    
- **Space Complexity:** O(H)
  - This refers to the Auxiliary Space used by the system's recursion stack.
  - **Balanced Tree (Best/Average Case):** O(logN). The stack depth matches the tree height.
  - **Skewed Tree (Worst Case):** O(N). If the tree is a straight line, the stack holds all N nodes simultaneously.
