## Binary Search Tree Iterator

**Problem Statement:**   
Implement the `BSTIterator` class that represents an iterator over the inorder traversal of a BST. The iterator should support:
- `next()` — Returns the next smallest number.
- `hasNext()` — Returns `true` if there exists a number in the traversal to the right of the current pointer, otherwise `false`.

Both functions should run in **O(H)** space, where H is the height of the tree, and **amortized O(1)** time.

---

>### Examples:

**Example 1:**
```
  BST:
         7
        / \
       3   15
          /  \
         9   20

  BSTIterator iterator = new BSTIterator(root);
  iterator.next();    // return 3
  iterator.next();    // return 7
  iterator.hasNext(); // return true
  iterator.next();    // return 9
  iterator.hasNext(); // return true
  iterator.next();    // return 15
  iterator.hasNext(); // return true
  iterator.next();    // return 20
  iterator.hasNext(); // return false
```

**Example 2:**
```
  BST:
       5
      / \
     3   7
    / \
   2   4

  Inorder: [2, 3, 4, 5, 7]

  next()    → 2
  next()    → 3
  hasNext() → true
  next()    → 4
  next()    → 5
  hasNext() → true
  next()    → 7
  hasNext() → false
```

---

### Key Insight:

A naive approach stores the full inorder traversal in an array (O(N) space). To achieve O(H) space, we simulate the inorder traversal using a **stack**, only pushing nodes along the **leftmost path** at any time.
```
  BST:                Stack evolution:
       7
      / \
     3   15           Init: push leftmost path → [7, 3]
        /  \
       9   20         next() → pop 3, push 3's right subtree's leftmost
                              stack = [7]

                      next() → pop 7, push 7's right subtree's leftmost
                              stack = [15, 9]

                      next() → pop 9
                              stack = [15]

                      next() → pop 15, push 15's right subtree's leftmost
                              stack = [20]

                      next() → pop 20
                              stack = [] → hasNext() = false
```

---

### Algorithm:

**Constructor:**
- Push all nodes along the leftmost path from the root onto the stack.

**`next()`:**
- Pop the top node from the stack — this is the next smallest.
- If the popped node has a right child, push all nodes along the leftmost path of the right subtree onto the stack.
- Return the popped node's value.

**`hasNext()`:**
- Return `true` if the stack is not empty, `false` otherwise.
```
  Pushing leftmost path:

       15
      /  \
     9   20        pushLeft(15):
                     push 15
                     push 9 (15's left)
                     9 has no left → stop
                     stack = [15, 9]

  After next() pops 9:
    9 has no right child → nothing to push
    stack = [15]

  After next() pops 15:
    15 has right child (20) → pushLeft(20)
    push 20 (no left child)
    stack = [20]
```

## C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

// Definition of TreeNode
struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class BSTIterator {
private:
    stack<TreeNode*> st;

    // Push all nodes along the leftmost path
    void pushLeft(TreeNode* node) {
        while (node != nullptr) {
            st.push(node);
            node = node->left;
        }
    }

public:
    // Constructor: initialize with leftmost path from root
    BSTIterator(TreeNode* root) {
        pushLeft(root);
    }

    // Returns the next smallest number
    int next() {
        // Top of stack is the next smallest
        TreeNode* top = st.top();
        st.pop();

        // If right subtree exists, push its leftmost path
        if (top->right != nullptr) {
            pushLeft(top->right);
        }

        return top->val;
    }

    // Returns true if there is a next element
    bool hasNext() {
        return !st.empty();
    }
};

// Driver code
int main() {
    // Construct BST
    TreeNode* root = new TreeNode(7);
    root->left = new TreeNode(3);
    root->right = new TreeNode(15);
    root->right->left = new TreeNode(9);
    root->right->right = new TreeNode(20);

    BSTIterator iterator(root);

    cout << "BST Iterator Output: ";
    while (iterator.hasNext()) {
        cout << iterator.next() << " ";
    }
    cout << endl;

    return 0;
}
```

---

### Step-by-Step Trace
```
  BST:
         7
        / \
       3   15
          /  \
         9   20

  Init → pushLeft(7): stack = [7, 3]  (bottom to top)

  next()    → pop 3, 3 has no right → stack = [7]       return 3
  next()    → pop 7, pushLeft(15)   → stack = [15, 9]   return 7
  hasNext() → stack not empty        → true
  next()    → pop 9, 9 has no right → stack = [15]      return 9
  hasNext() → stack not empty        → true
  next()    → pop 15, pushLeft(20)  → stack = [20]      return 15
  hasNext() → stack not empty        → true
  next()    → pop 20, no right      → stack = []        return 20
  hasNext() → stack empty            → false
```

---

### Why O(H) Space?
```
  At any point, the stack only holds nodes
  along a single root-to-leaf path.

  Balanced BST:    Stack size ~ log N  →  O(log N)
  Skewed BST:      Stack size ~ N      →  O(N)
  General:         Stack size ~ H      →  O(H)

  Contrast with naive approach:
  Store full inorder array → O(N) always
```

---

### Complexity Analysis:

**Time Complexity:**
- `next()`: Amortized O(1). Each node is pushed and popped exactly once over all calls.
- `hasNext()`: O(1). Simply checks if the stack is empty.

**Space Complexity:** O(H), where H is the height of the BST. At any time, the stack holds at most H nodes (one path from root to a node).
```
  Balanced BST:              Skewed BST:
       7                         1
      / \                         \
     3   15                        3
        /  \                        \
       9   20                        7
                                      \
  Stack max size = 3 (H)              15
  = O(log N)                        Stack max size = N
                                    = O(N)
```
