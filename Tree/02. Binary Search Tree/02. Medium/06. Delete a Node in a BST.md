# Delete a Node in a BST

**Problem Statement:**   
Deleting a node in a BST is slightly more complex than insertion because after removal, the BST property must still hold. There are three cases to handle depending on the node being deleted.
```
  Delete 3 from this BST:

       5                    5
      / \                  / \
     3   7      →         4   7
    / \                  /
   2   4                2
```

---

## Three Cases of Deletion

### Case 1: Node has NO children (Leaf Node)
Simply remove the node.
```
  Delete 4:
       5                  5
      / \      →         / \
     3   7              3   7
    / \
   2   4            4 is removed ✓
```

### Case 2: Node has ONE child
Replace the node with its only child.
```
  Delete 3:
       5                  5
      / \      →         / \
     3   7              2   7
    /
   2               3 is replaced by 2 ✓
```

### Case 3: Node has TWO children
Replace the node's value with its **inorder successor** (smallest node in right subtree) or **inorder predecessor** (largest node in left subtree), then delete that successor/predecessor.
```
  Delete 5 (root):
       5                  7
      / \      →         / \
     3   7              3   9
    / \   \            / \
   2   4   9          2   4

  Inorder successor of 5 = 7
  Replace 5 with 7, delete 7 from right subtree ✓
```

  Before:                After:

  Inorder successor of 3 = 4
  Replace 3 with 4, delete 4 from right subtree ✓

## Approach
### 💡 Intuition:
 
Deletion in a BST has three cases based on the node's children:
 
1. **No left child** → return right child directly
2. **No right child** → return left child directly
3. **Both children exist** → attach the right subtree to the **rightmost node of the left subtree**, then return the left child as the new root of this subtree
Why the rightmost of the left subtree? Because it's the **largest in the left subtree**, so all nodes in the right subtree are still greater — BST property is preserved.
 
---
 
### 🧭 Algorithm:
 
**Step 1 — Find the node:**
- If `root->val == key`, call `helper(root)` directly.
- Otherwise, traverse the BST:
  - Go **left** if `key < root->val`
  - Go **right** if `key > root->val`
  - Stop one level **above** the target (so we can rewire the parent's pointer)
**Step 2 — Delete via `helper(node)`:**
 
```
if no left child  → return right child
if no right child → return left child
else:
    rightChild        = node->right
    lastRight         = rightmost node of node->left
    lastRight->right  = rightChild      // attach right subtree
    return node->left                   // left child becomes new local root
```
 
**Step 3 — Return:**
- If the root itself was deleted, return the result of `helper` directly.
- Otherwise, return the original `dummy` root (tree structure above the deleted node is unchanged).

---

### C++ Code:
```cpp
#include <iostream>
using namespace std;

// Node structure
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    // Main function to delete a node
    TreeNode* deleteNode(TreeNode* root, int key) {
        if (root == nullptr) return nullptr;

        // If the root itself is the node to be deleted
        if (root->val == key) {
            return helper(root);
        }

        TreeNode *dummy = root;
        while (root != nullptr) {
            if (root->val > key) {
                // Look left
                if (root->left != nullptr && root->left->val == key) {
                    root->left = helper(root->left);
                    break;
                } else {
                    root = root->left;
                }
            } else {
                // Look right
                if (root->right != nullptr && root->right->val == key) {
                    root->right = helper(root->right);
                    break;
                } else {
                    root = root->right;
                }
            }
        }
        return dummy;
    }

    // Connects the remaining subtrees after a node is removed
    TreeNode* helper(TreeNode* root) { // root is key node
        if (root->left == nullptr) return root->right;
        else if (root->right == nullptr) return root->left;

        TreeNode* rightChild = root->right;
        TreeNode* lastRight = findLastRight(root->left);
        lastRight->right = rightChild;
        return root->left;
    }

    // Finds the rightmost node in the left subtree
    TreeNode* findLastRight(TreeNode* root) {
        if (root->right == nullptr) return root;
        return findLastRight(root->right);
    }

    // Simple In-order traversal to print the tree
    void printTree(TreeNode* root) {
        if (!root) return;
        printTree(root->left);
        cout << root->val << " ";
        printTree(root->right);
    }
};

int main() {
    // Creating a VALID BST
    //        15
    //       /  \
    //      4    20
    //     / \
    //    0   6
    TreeNode* root = new TreeNode(15);
    root->left = new TreeNode(4);
    root->right = new TreeNode(20);
    root->left->left = new TreeNode(0);
    root->left->right = new TreeNode(6);

    Solution sol;
    
    cout << "Original Tree (In-order): ";
    sol.printTree(root);
    cout << endl;

    int keyToDelete = 4;
    cout << "Deleting node: " << keyToDelete << "..." << endl;
    root = sol.deleteNode(root, keyToDelete);

    cout << "New Tree (In-order): ";
    sol.printTree(root);
    cout << endl;

    return 0;
}
```

### Complexity Analysis:

**Time Complexity:** O(H), where H is the height of the BST.
- Balanced BST: O(log N)
- Skewed BST: O(N)
```
  Balanced BST             Skewed BST
       5                       1
      / \                       \
     3   7                       3
    / \   \                       \
   2   4   8                       5
                                    \
  Delete ~ log(N) steps              7
                                  Delete ~ N steps
```

**Space Complexity:** O(H) due to the recursive call stack.
- Balanced BST: O(log N)
- Skewed BST: O(N)
