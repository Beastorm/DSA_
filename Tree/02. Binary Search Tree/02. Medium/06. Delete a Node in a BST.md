## Delete a Node in a BST

**Problem Statement:**   
Deleting a node in a BST is slightly more complex than insertion because after removal, the BST property must still hold. There are three cases to handle depending on the node being deleted.
```
  Delete 3 from this BST:

       5                    5
      / \                  / \
     3   7      →         4   7
    / \                  /
   2   4                2
```

---

## Three Cases of Deletion

### Case 1: Node has NO children (Leaf Node)
Simply remove the node.
```
  Delete 4:
       5                  5
      / \      →         / \
     3   7              3   7
    / \
   2   4            4 is removed ✓
```

### Case 2: Node has ONE child
Replace the node with its only child.
```
  Delete 3:
       5                  5
      / \      →         / \
     3   7              2   7
    /
   2               3 is replaced by 2 ✓
```

### Case 3: Node has TWO children
Replace the node's value with its **inorder successor** (smallest node in right subtree) or **inorder predecessor** (largest node in left subtree), then delete that successor/predecessor.
```
  Delete 5 (root):
       5                  7
      / \      →         / \
     3   7              3   9
    / \   \            / \
   2   4   9          2   4

  Inorder successor of 5 = 7
  Replace 5 with 7, delete 7 from right subtree ✓
```

## Examples

**Example 1:** Delete `3` (node with two children)
```
  Before:                After:
       5                    5
      / \                  / \
     3   6                4   6
    / \                  /
   2   4                2

  Inorder successor of 3 = 4
  Replace 3 with 4, delete 4 from right subtree ✓
```

**Example 2:** Delete `6` (leaf node)
```
  Before:                After:
       5                    5
      / \                  /
     3   6      →         3
    / \                  / \
   2   4                2   4

  6 has no children → simply remove ✓
```

**Example 3:** Delete `5` (root with two children)
```
  Before:                After:
       5                    6
      / \                  / \
     3   8      →         3   8
        / \                  \
       6   9                  9

  Inorder successor of 5 = 6
  Replace 5 with 6, delete 6 from right subtree ✓
```

### Edge Cases:
```
  Delete from single node tree:     Delete non-existent key:
         5                                 5
                                          / \
  Delete 5:                              3   7
    return null ✓
                                    Delete 9 → tree unchanged ✓


  Delete root with two children:
         5
        / \
       3   8
          / \
         6   9

  Delete 5 → inorder successor = 6
         6
        / \
       3   8
            \
             9
```

## Approach
### Algorithm:

- If the root is `null`, return `null` (node not found).
- If the key is **less than** the root's value, recurse on the left subtree.
- If the key is **greater than** the root's value, recurse on the right subtree.
- If the key **equals** the root's value (node found):
  - If node has **no left child**, return the right child.
  - If node has **no right child**, return the left child.
  - If node has **two children**, find the inorder successor (leftmost node in right subtree), replace the current node's value with it, and delete the inorder successor from the right subtree.
```
       Find node to delete
              |
       node found?
       /           \
     YES             NO
      |               |
  0 children?     return null
  /         \
YES           NO
 |             |
remove      1 child?
            /      \
          YES        NO (2 children)
           |              |
      replace with    find inorder
      only child      successor
                           |
                    replace value +
                    delete successor
```

## Code
```cpp
#include <iostream>
using namespace std;

// Node structure
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    // Main function to delete a node
    TreeNode* deleteNode(TreeNode* root, int key) {
        if (root == nullptr) return nullptr;

        // If the root itself is the node to be deleted
        if (root->val == key) {
            return helper(root);
        }

        TreeNode *dummy = root;
        while (root != nullptr) {
            if (root->val > key) {
                // Look left
                if (root->left != nullptr && root->left->val == key) {
                    root->left = helper(root->left);
                    break;
                } else {
                    root = root->left;
                }
            } else {
                // Look right
                if (root->right != nullptr && root->right->val == key) {
                    root->right = helper(root->right);
                    break;
                } else {
                    root = root->right;
                }
            }
        }
        return dummy;
    }

    // Connects the remaining subtrees after a node is removed
    TreeNode* helper(TreeNode* root) {
        if (root->left == nullptr) return root->right;
        else if (root->right == nullptr) return root->left;

        TreeNode* rightChild = root->right;
        TreeNode* lastRight = findLastRight(root->left);
        lastRight->right = rightChild;
        return root->left;
    }

    // Finds the rightmost node in the left subtree
    TreeNode* findLastRight(TreeNode* root) {
        if (root->right == nullptr) return root;
        return findLastRight(root->right);
    }

    // Simple In-order traversal to print the tree
    void printTree(TreeNode* root) {
        if (!root) return;
        printTree(root->left);
        cout << root->val << " ";
        printTree(root->right);
    }
};

int main() {
    // Creating a VALID BST
    //        15
    //       /  \
    //      4    20
    //     / \
    //    0   6
    TreeNode* root = new TreeNode(15);
    root->left = new TreeNode(4);
    root->right = new TreeNode(20);
    root->left->left = new TreeNode(0);
    root->left->right = new TreeNode(6);

    Solution sol;
    
    cout << "Original Tree (In-order): ";
    sol.printTree(root);
    cout << endl;

    int keyToDelete = 4;
    cout << "Deleting node: " << keyToDelete << "..." << endl;
    root = sol.deleteNode(root, keyToDelete);

    cout << "New Tree (In-order): ";
    sol.printTree(root);
    cout << endl;

    return 0;
}
```

### Complexity Analysis:

**Time Complexity:** O(H), where H is the height of the BST.
- Balanced BST: O(log N)
- Skewed BST: O(N)
```
  Balanced BST             Skewed BST
       5                       1
      / \                       \
     3   7                       3
    / \   \                       \
   2   4   8                       5
                                    \
  Delete ~ log(N) steps              7
                                  Delete ~ N steps
```

**Space Complexity:** O(H) due to the recursive call stack.
- Balanced BST: O(log N)
- Skewed BST: O(N)
