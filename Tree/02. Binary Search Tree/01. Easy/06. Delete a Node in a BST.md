## Delete a Node in a BST

**Problem Statement:**   
Deleting a node in a BST is slightly more complex than insertion because after removal, the BST property must still hold. There are three cases to handle depending on the node being deleted.
```
  Delete 3 from this BST:

       5                    5
      / \                  / \
     3   7      →         4   7
    / \                  /
   2   4                2
```

---

## Three Cases of Deletion

### Case 1: Node has NO children (Leaf Node)
Simply remove the node.
```
  Delete 4:
       5                  5
      / \      →         / \
     3   7              3   7
    / \
   2   4            4 is removed ✓
```

### Case 2: Node has ONE child
Replace the node with its only child.
```
  Delete 3:
       5                  5
      / \      →         / \
     3   7              2   7
    /
   2               3 is replaced by 2 ✓
```

### Case 3: Node has TWO children
Replace the node's value with its **inorder successor** (smallest node in right subtree) or **inorder predecessor** (largest node in left subtree), then delete that successor/predecessor.
```
  Delete 5 (root):
       5                  7
      / \      →         / \
     3   7              3   9
    / \   \            / \
   2   4   9          2   4

  Inorder successor of 5 = 7
  Replace 5 with 7, delete 7 from right subtree ✓
```

## Examples

**Example 1:** Delete `3` (node with two children)
```
  Before:                After:
       5                    5
      / \                  / \
     3   6                4   6
    / \                  /
   2   4                2

  Inorder successor of 3 = 4
  Replace 3 with 4, delete 4 from right subtree ✓
```

**Example 2:** Delete `6` (leaf node)
```
  Before:                After:
       5                    5
      / \                  /
     3   6      →         3
    / \                  / \
   2   4                2   4

  6 has no children → simply remove ✓
```

**Example 3:** Delete `5` (root with two children)
```
  Before:                After:
       5                    6
      / \                  / \
     3   8      →         3   8
        / \                  \
       6   9                  9

  Inorder successor of 5 = 6
  Replace 5 with 6, delete 6 from right subtree ✓
```

### Edge Cases:
```
  Delete from single node tree:     Delete non-existent key:
         5                                 5
                                          / \
  Delete 5:                              3   7
    return null ✓
                                    Delete 9 → tree unchanged ✓


  Delete root with two children:
         5
        / \
       3   8
          / \
         6   9

  Delete 5 → inorder successor = 6
         6
        / \
       3   8
            \
             9
```

## Approach
### Algorithm:

- If the root is `null`, return `null` (node not found).
- If the key is **less than** the root's value, recurse on the left subtree.
- If the key is **greater than** the root's value, recurse on the right subtree.
- If the key **equals** the root's value (node found):
  - If node has **no left child**, return the right child.
  - If node has **no right child**, return the left child.
  - If node has **two children**, find the inorder successor (leftmost node in right subtree), replace the current node's value with it, and delete the inorder successor from the right subtree.
```
       Find node to delete
              |
       node found?
       /           \
     YES             NO
      |               |
  0 children?     return null
  /         \
YES           NO
 |             |
remove      1 child?
            /      \
          YES        NO (2 children)
           |              |
      replace with    find inorder
      only child      successor
                           |
                    replace value +
                    delete successor
```

## Code
```cpp
#include <bits/stdc++.h>
using namespace std;

// Definition of TreeNode
struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;

    TreeNode(int data) {
        val = data;
        left = right = nullptr;
    }
};

class Solution {
public:
    // Helper to find the leftmost (minimum) node
    TreeNode* findMin(TreeNode* node) {
        while (node->left != nullptr) {
            node = node->left;
        }
        return node;
    }

    // Function to delete a node from BST
    TreeNode* deleteNode(TreeNode* root, int key) {
        // Base case: node not found
        if (root == nullptr) return nullptr;

        // Navigate to the node to delete
        if (key < root->val) {
            // Key is in the left subtree
            root->left = deleteNode(root->left, key);
        }
        else if (key > root->val) {
            // Key is in the right subtree
            root->right = deleteNode(root->right, key);
        }
        else {
            // Node to delete found

            // Case 1 & 2: No left child or no right child
            if (root->left == nullptr) {
                TreeNode* temp = root->right;
                delete root;
                return temp;
            }
            else if (root->right == nullptr) {
                TreeNode* temp = root->left;
                delete root;
                return temp;
            }

            // Case 3: Two children
            // Find inorder successor (min of right subtree)
            TreeNode* successor = findMin(root->right);

            // Replace current node's value with successor's value
            root->val = successor->val;

            // Delete the inorder successor from right subtree
            root->right = deleteNode(root->right, successor->val);
        }

        return root;
    }
};

// Helper to print inorder traversal
void inorder(TreeNode* root) {
    if (!root) return;
    inorder(root->left);
    cout << root->val << " ";
    inorder(root->right);
}

// Driver code
int main() {
    TreeNode* root = new TreeNode(5);
    root->left = new TreeNode(3);
    root->right = new TreeNode(6);
    root->left->left = new TreeNode(2);
    root->left->right = new TreeNode(4);

    cout << "Inorder before deletion: ";
    inorder(root);
    cout << endl;

    Solution obj;
    root = obj.deleteNode(root, 3);

    cout << "Inorder after deleting 3: ";
    inorder(root);
    cout << endl;

    return 0;
}
```

### Complexity Analysis:

**Time Complexity:** O(H), where H is the height of the BST.
- Balanced BST: O(log N)
- Skewed BST: O(N)
```
  Balanced BST             Skewed BST
       5                       1
      / \                       \
     3   7                       3
    / \   \                       \
   2   4   8                       5
                                    \
  Delete ~ log(N) steps              7
                                  Delete ~ N steps
```

**Space Complexity:** O(H) due to the recursive call stack.
- Balanced BST: O(log N)
- Skewed BST: O(N)
