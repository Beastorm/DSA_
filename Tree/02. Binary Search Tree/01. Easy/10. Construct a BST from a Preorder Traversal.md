## Construct a BST from a Preorder Traversal

**Problem Statement:**  
Given an array of integers representing the preorder traversal of a BST, construct the BST and return its root node.

---

>### Examples:

**Example 1:**
- **Input:** `preorder = [8, 5, 1, 7, 10, 12]`
- **Output:** Root of BST
```
  Preorder: [8, 5, 1, 7, 10, 12]

         8
        / \
       5   10
      / \    \
     1   7    12

  Preorder verification:
  Root(8) → Left(5,1,7) → Right(10,12) ✓
```

**Example 2:**
- **Input:** `preorder = [4, 2, 1, 3, 6, 5, 7]`
- **Output:** Root of BST
```
  Preorder: [4, 2, 1, 3, 6, 5, 7]

         4
        / \
       2   6
      / \ / \
     1  3 5  7
```

---

### Key Insight:

In a preorder traversal `[Root, Left, Right]`:
- The **first element** is always the root.
- Elements **smaller than root** form the left subtree.
- Elements **larger than root** form the right subtree.
```
  Preorder: [8, 5, 1, 7, 10, 12]
             ↑
            root = 8

  Split:  [5, 1, 7] < 8  →  LEFT subtree
          [10, 12]  > 8  →  RIGHT subtree

  Recurse on each half:
  Left  [5, 1, 7]:  root=5, left=[1], right=[7]
  Right [10, 12]:   root=10, right=[12]
```

---

## Method 1: Naive / Brute Force

### Algorithm:

- The first element of preorder is the root.
- Iterate through the array to find the first element greater than the root — this is the boundary between left and right subtrees.
- Recursively build the left subtree from elements before the boundary.
- Recursively build the right subtree from elements after the boundary.
```
  preorder = [8, 5, 1, 7, 10, 12]

  root = 8
  Find split: first element > 8 is 10 at index 4

  Left  = [5, 1, 7]   (indices 1 to 3)
  Right = [10, 12]    (indices 4 to 5)

  Recurse:
    Left root=5, split at 7: left=[1], right=[7]
    Right root=10, split at 12: right=[12]
```

### C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    TreeNode* bstFromPreorder(vector<int>& preorder) {
        return build(preorder, 0, preorder.size() - 1);
    }

private:
    TreeNode* build(vector<int>& pre, int start, int end) {
        // Base case
        if (start > end) return nullptr;

        // First element is root
        TreeNode* root = new TreeNode(pre[start]);

        // Find boundary between left and right subtrees
        int split = end + 1;
        for (int i = start + 1; i <= end; i++) {
            if (pre[i] > pre[start]) {
                split = i;
                break;
            }
        }

        // Recursively build left and right subtrees
        root->left = build(pre, start + 1, split - 1);
        root->right = build(pre, split, end);

        return root;
    }
};

void inorder(TreeNode* root) {
    if (!root) return;
    inorder(root->left);
    cout << root->val << " ";
    inorder(root->right);
}

int main() {
    vector<int> preorder = {8, 5, 1, 7, 10, 12};
    Solution obj;
    TreeNode* root = obj.bstFromPreorder(preorder);
    cout << "Inorder of constructed BST: ";
    inorder(root);
    cout << endl;
    return 0;
}
```

### Complexity Analysis:

**Time Complexity:** O(N²) in the worst case (skewed BST), as for each node we scan through the array to find the split point.

**Space Complexity:** O(N) due to the recursion stack.

## Method 2: Using Upper Bound (Optimal Recursive)

### Algorithm:

- Use a global index pointer to track the current element being processed.
- Pass an upper bound to each recursive call — only insert a node if its value is within the allowed range.
- For the left subtree, the upper bound is the current node's value.
- For the right subtree, the upper bound remains unchanged from the parent.
```
  preorder = [8, 5, 1, 7, 10, 12]
  idx = 0, bound = INT_MAX

  root = 8 (8 < INT_MAX ✓), idx=1
    left call:  bound = 8
      root = 5 (5 < 8 ✓),   idx=2
        left call:  bound = 5
          root = 1 (1 < 5 ✓), idx=3
        right call: bound = 5
          7 < 5? NO → return null
      right call: bound = 8
          root = 7 (7 < 8 ✓), idx=4
    right call: bound = INT_MAX
      root = 10 (10 < MAX ✓), idx=5
        right call:
          root = 12 ✓
```

### C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    int idx = 0;

    TreeNode* bstFromPreorder(vector<int>& preorder) {
        return build(preorder, INT_MAX);
    }

private:
    TreeNode* build(vector<int>& pre, int bound) {
        // If all elements processed or current exceeds bound
        if (idx == pre.size() || pre[idx] > bound) return nullptr;

        // Create current root
        TreeNode* root = new TreeNode(pre[idx++]);

        // Left subtree: upper bound is root's value
        root->left = build(pre, root->val);

        // Right subtree: upper bound remains same
        root->right = build(pre, bound);

        return root;
    }
};

void inorder(TreeNode* root) {
    if (!root) return;
    inorder(root->left);
    cout << root->val << " ";
    inorder(root->right);
}

int main() {
    vector<int> preorder = {8, 5, 1, 7, 10, 12};
    Solution obj;
    TreeNode* root = obj.bstFromPreorder(preorder);
    cout << "Inorder of constructed BST: ";
    inorder(root);
    cout << endl;
    return 0;
}
```

### Complexity Analysis:

**Time Complexity:** O(N), each element is processed exactly once.

**Space Complexity:** O(H) due to the recursion stack, where H is the height of the BST.

## Method 3: Iterative Using Stack

### Algorithm:

- Use a stack to simulate the recursive construction.
- The first element of preorder is the root — push it onto the stack.
- For each subsequent element:
  - If the element is **less than** the stack's top, it becomes the **left child** of the top node.
  - Otherwise, **pop** nodes from the stack until the top is greater than the element. The last popped node becomes the **parent**, and the new node is its **right child**.
- Push the new node onto the stack.
```
  preorder = [8, 5, 1, 7, 10, 12]

  Process 8:  stack=[8],        root=8
  Process 5:  5<8 → left of 8, stack=[8,5]
  Process 1:  1<5 → left of 5, stack=[8,5,1]
  Process 7:  7>1 → pop 1
              7>5 → pop 5, last popped=5
              7<8 → stop, 7=right of 5
              stack=[8,7]
  Process 10: 10>7 → pop 7
              10>8 → pop 8, last popped=8
              stack empty → 10=right of 8
              stack=[10]
  Process 12: 12>10 → pop 10, last popped=10
              stack empty → 12=right of 10
              stack=[12]
```

### C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

class Solution {
public:
    TreeNode* bstFromPreorder(vector<int>& preorder) {
        if (preorder.empty()) return nullptr;

        TreeNode* root = new TreeNode(preorder[0]);
        stack<TreeNode*> st;
        st.push(root);

        for (int i = 1; i < preorder.size(); i++) {
            TreeNode* node = new TreeNode(preorder[i]);
            TreeNode* parent = nullptr;

            // Pop nodes smaller than current element
            while (!st.empty() && st.top()->val < preorder[i]) {
                parent = st.top();
                st.pop();
            }

            if (parent != nullptr) {
                // New node is right child of last popped
                parent->right = node;
            } else {
                // New node is left child of current top
                st.top()->left = node;
            }

            st.push(node);
        }

        return root;
    }
};

void inorder(TreeNode* root) {
    if (!root) return;
    inorder(root->left);
    cout << root->val << " ";
    inorder(root->right);
}

int main() {
    vector<int> preorder = {8, 5, 1, 7, 10, 12};
    Solution obj;
    TreeNode* root = obj.bstFromPreorder(preorder);
    cout << "Inorder of constructed BST: ";
    inorder(root);
    cout << endl;
    return 0;
}
```

### Complexity Analysis:

**Time Complexity:** O(N), each node is pushed and popped from the stack at most once.

**Space Complexity:** O(N) for the stack in the worst case (skewed BST).

---

## Comparison of All Three Methods
```
  Method          Time      Space     Approach
  ─────────────────────────────────────────────────
  Brute Force     O(N²)     O(N)      Find split index
  Optimal Recur.  O(N)      O(H)      Upper bound + index
  Iterative Stack O(N)      O(N)      Stack simulation
```
