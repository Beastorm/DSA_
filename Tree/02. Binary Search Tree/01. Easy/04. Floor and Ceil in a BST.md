## Floor and Ceil in a BST

>### Example:

- **Floor** of a key in a BST is the **largest value** in the BST that is **less than or equal to** the key.
- **Ceil** of a key in a BST is the **smallest value** in the BST that is **greater than or equal to** the key.
```
              8
            /   \
           4     12
          / \   /  \
         2   6 10   14
            /
           5

  Key = 7
  Floor = 6  (largest value ≤ 7)
  Ceil  = 8  (smallest value ≥ 7)

  Key = 5
  Floor = 5  (exact match)
  Ceil  = 5  (exact match)
```

---

### Edge Cases
```
  Key smaller than all nodes:        Key larger than all nodes:
         8                                  8
        / \                                / \
       4   12                             4   12

  Key = 1                            Key = 15
  Floor = -1 (none exists)           Floor = 12
  Ceil  = 4                          Ceil  = -1 (none exists)
```

---

## Approach
### Algorithm:

### Find Floor
- If the current node's value equals the key → it is the floor, return it.
- If the key is less than the current node's value → floor must be in the left subtree, move left.
- If the key is greater than the current node's value → current node **could be** the floor, store it and move right to find a closer value.
```
  Find the floor of 7:

       8         → 7 < 8,  go LEFT
      /
     4            → 7 > 4,  floor candidate = 4, go RIGHT
      \
       6          → 7 > 6,  floor candidate = 6, go RIGHT
        \
        null      → stop, return floor = 6 ✓
```

### Find Ceil
- If the current node's value equals the key → it is the ceil, return it.
- If the key is greater than the current node's value → ceil must be in the right subtree, move right.
- If the key is less than the current node's value → current node **could be** the ceil, store it and move left to find a closer value.
```
  Find the ceil of 7:

       8          → 7 < 8,  ceil candidate = 8, go LEFT
      /
     4            → 7 > 4,  go RIGHT
      \
       6          → 7 > 6,  go RIGHT
        \
        null      → stop, return ceil = 8 ✓
```

## C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

// Definition of TreeNode
struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;

    TreeNode(int data) {
        val = data;
        left = right = nullptr;
    }
};

class Solution {
public:
    // Function to find floor of key in BST
    int findFloor(TreeNode* root, int key) {
        int floor = -1;

        while (root != nullptr) {
            // Exact match is the floor
            if (root->val == key) {
                return root->val;
            }
            // Current node is too large, go left
            else if (key < root->val) {
                root = root->left;
            }
            // Current node is a candidate, go right for closer value
            else {
                floor = root->val;
                root = root->right;
            }
        }

        return floor;
    }

    // Function to find ceil of key in BST
    int findCeil(TreeNode* root, int key) {
        int ceil = -1;

        while (root != nullptr) {
            // Exact match is the ceil
            if (root->val == key) {
                return root->val;
            }
            // Current node is too small, go right
            else if (key > root->val) {
                root = root->right;
            }
            // Current node is a candidate, go left for closer value
            else {
                ceil = root->val;
                root = root->left;
            }
        }

        return ceil;
    }
};

// Driver code
int main() {
    TreeNode* root = new TreeNode(8);
    root->left = new TreeNode(4);
    root->right = new TreeNode(12);
    root->left->left = new TreeNode(2);
    root->left->right = new TreeNode(6);
    root->left->right->left = new TreeNode(5);
    root->right->left = new TreeNode(10);
    root->right->right = new TreeNode(14);

    Solution obj;

    int key = 7;
    cout << "Key: " << key << endl;
    cout << "Floor: " << obj.findFloor(root, key) << endl;
    cout << "Ceil:  " << obj.findCeil(root, key)  << endl;

    return 0;
}
```

### Complexity Analysis:

**Time Complexity:** O(H), where H is the height of the BST.
- Balanced BST: O(log N)
- Skewed BST: O(N)

**Space Complexity:** O(1), only a few variables are used for traversal, no extra data structures or recursion involved.
