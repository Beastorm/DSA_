## Find Min/Max in a Binary Search Tree

>### Example:

In a Binary Search Tree, due to the BST property:
- The **minimum** value is always at the **leftmost** node.
- The **maximum** value is always at the **rightmost** node.
```
              8
            /   \
           3     10
          / \      \
         1   6      14
            / \     /
           4   7   13

  MIN = 1 (leftmost node)
  MAX = 14 (rightmost node)
```

---

## Approach
### Algorithm:

### Find Minimum:
Keep traversing to the left child until we reach a node with no left child — that is the minimum.
```
  8 → go LEFT
  3 → go LEFT
  1 → no left child ✓ MIN = 1
```

### Find Maximum:
Keep traversing to the right child until we reach a node with no right child — that is the maximum.
```
  8  → go RIGHT
  10 → go RIGHT
  14 → no right child ✓ MAX = 14
```

---

## C++ Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

// Definition of TreeNode
struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;

    TreeNode(int data) {
        val = data;
        left = right = nullptr;
    }
};

class Solution {
public:
    // Function to find the minimum value in the BST
    int findMin(TreeNode* root) {
        if (root == nullptr) return -1;

        // Keep moving left until the leftmost node
        while (root->left != nullptr) {
            root = root->left;
        }
        return root->val;
    }

    // Function to find maximum value in BST
    int findMax(TreeNode* root) {
        if (root == nullptr) return -1;

        // Keep moving right until the rightmost node
        while (root->right != nullptr) {
            root = root->right;
        }
        return root->val;
    }
};

// Driver code
int main() {
    TreeNode* root = new TreeNode(8);
    root->left = new TreeNode(3);
    root->right = new TreeNode(10);
    root->left->left = new TreeNode(1);
    root->left->right = new TreeNode(6);
    root->right->right = new TreeNode(14);
    root->left->right->left = new TreeNode(4);
    root->left->right->right = new TreeNode(7);
    root->right->right->left = new TreeNode(13);

    Solution obj;

    cout << "Minimum value in BST: " << obj.findMin(root) << endl;
    cout << "Maximum value in BST: " << obj.findMax(root) << endl;

    return 0;
}
```

### Complexity Analysis:

**Time Complexity:** O(H), where H is the height of the BST.
- For a balanced BST: O(log N)
- For a skewed BST: O(N)
```
  Balanced BST             Skewed BST
       8                       1
      / \                       \
     3   10                      3
    /      \                      \
   1        14                     8
                                    \
  Steps ~ log(N)                    10
                                  Steps ~ N
```

**Space Complexity:** O(1), only a single pointer is used to traverse the tree, no recursion or extra data structures involved.
