## Design a Browser History | LinkedList Implementation

### Intuition:
To simulate a browser's history with `back` and `visit` operations, we use a doubly linked list. Each node represents a page, with `next` and `back` pointers allowing us to move forward and backward efficiently. The current page is tracked by a pointer, and visiting a new page creates a new node, updating the links accordingly.

### Algorithm:
1. Initialize the browser with a homepage node.
2. For `visit(url)`, create a new node for the URL, link it to the current page, and update the current page pointer.
3. For `back(steps)`, move the current page pointer backward up to the given number of steps or until no more back pages exist.


### C++ Code:
``` cpp
#include <iostream>
#include <string>
using namespace std;

// Node class for doubly linked list
class Node {
public:
    string data;
    Node *next, *back;
    Node() : data(""), next(nullptr), back(nullptr) {}
    Node(string x) : data(x), next(nullptr), back(nullptr) {}
    Node(string x, Node *next, Node *random) : data(x), next(next), back(random) {}
};

// Browser class to simulate browser history
class Browser {
    Node* currentPage;
public:
    // Initialize browser with homepage
    Browser(string homepage) {
        currentPage = new Node(homepage);
    }

    // Visit a new URL
    void visit(string url) {
        Node* newNode = new Node(url);
        currentPage->next = newNode;
        newNode->back = currentPage;
        currentPage = newNode;
    }

    // Move back by 'steps' pages
    string back(int steps) {
        while (steps--) {
            if (currentPage->back)
                currentPage = currentPage->back;
            else
                break;
        }
        return currentPage->data;
    }

    // Move forward by 'steps' pages
    string forward(int steps) {
        while (steps--) {
            if (currentPage->next)
                currentPage = currentPage->next;
            else
                break;
        }
        return currentPage->data;
    }
};

// Driver code
int main() {
    Browser browser("home.com");
    browser.visit("page1.com");
    browser.visit("page2.com");
    cout << "Current: " << browser.back(0) << endl;      // page2.com
    cout << "Back 1: " << browser.back(1) << endl;       // page1.com
    cout << "Back 1: " << browser.back(1) << endl;       // home.com
    cout << "Forward 2: " << browser.forward(2) << endl; // page2.com
    cout << "Forward 1: " << browser.forward(1) << endl; // page2.com (no further page)
    return 0;
}
```
### Complexity Analysis:

- **Time Complexity:**
  - `visit(url)`: O(1), creates and links a new node.
  - `back(steps)`: O(steps), moves up to `steps` nodes backward.
  - `forward(steps)`: O(steps), moves up to `steps` nodes forward.

- **Space Complexity:** O(N), where N is the total number of visited pages (nodes created).
