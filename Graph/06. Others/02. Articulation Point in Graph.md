## Articulation Point in a Graph | Tarjan's Algorithm

---

### Problem Statement

Given an **undirected connected graph**, find all **articulation points** (also called cut vertices).

An **articulation point** is a vertex whose removal increases the number of connected components in the graph.

---

### Examples

**Example 1:**

```
Graph:
0 — 1 — 2
|   |
3   4

Articulation Points: 1
```

**Example 2:**

```
Graph: 0 — 1 — 2 — 3

Articulation Points: 1, 2
```

---

### Conditions for Articulation Points

- `disc[u]`: Discovery time of node `u`
- `low[u]`: Earliest discovery time reachable from subtree rooted at `u`

**Rules:**

1. **Root Node**: If root has 2 or more children in DFS tree → articulation point.
2. **Non-root Node `u`**: If `low[v] >= disc[u]` for any child `v` → articulation point.

---

### Code (C++)

```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    void DFS(int node, int parent, vector<int> adj[], vector<bool>& visited,
             vector<int>& disc, vector<int>& low, vector<bool>& artpoint, int& timer) {
        
        visited[node] = 1;
        disc[node] = low[node] = timer++;

        for (int j = 0; j < adj[node].size(); j++) {
            int neib = adj[node][j];

            if (neib == parent) continue;

            else if (visited[neib]) {
                low[node] = min(low[node], disc[neib]);
            } 
            else {
                DFS(neib, node, adj, visited, disc, low, artpoint, timer);

                low[node] = min(low[node], low[neib]);

                if (parent != -1 && low[neib] >= disc[node]) {
                    artpoint[node] = true;
                }
            }
        }
    }

    vector<int> articulationPoints(int V, vector<int> adj[]) {
        vector<bool> artpoint(V, 0);
        vector<int> disc(V);
        vector<int> low(V);
        vector<bool> visited(V, 0);
        int timer = 0;

        DFS(0, -1, adj, visited, disc, low, artpoint, timer);

        vector<int> ans;
        for (int i = 0; i < V; i++) {
            if (artpoint[i]) {
                ans.push_back(i);
            }
        }
        return ans;
    }
};

// ==================== DRIVER CODE ====================
int main() {
    Solution sol;

    int V = 5;
    vector<int> adj[V];
    vector<vector<int>> edges = {{0,1},{1,2},{0,3},{1,4}};

    for (auto& e : edges) {
        adj[e[0]].push_back(e[1]);
        adj[e[1]].push_back(e[0]);
    }

    vector<int> res = sol.articulationPoints(V, adj);
    cout << "Articulation Points: ";
    for (int x : res) cout << x << " ";
    cout << endl;

    return 0;
}
```

- **Space Complexity:** O(V + E)
