## Euler Path and Euler Circuit in a Graph

---

### Problem Statement

Given an **undirected graph**, determine whether it has an **Euler Path** or an **Euler Circuit**.

- **Euler Path**: A path that visits every **edge** exactly once.
- **Euler Circuit**: A circuit (closed path) that visits every **edge** exactly once and returns to the starting vertex.

---

### Examples

**Example 1: Euler Circuit Exists**

```
Graph: 0 — 1 — 2 — 0
       |         |
       3 — 4 — 3

Degrees: All vertices have even degree.
Output: Euler Circuit Exists
```

**Example 2: Euler Path Exists (but not Circuit)**

```
Graph: 0 — 1 — 2 — 3

Degrees: 0 and 3 have odd degree (2 vertices with odd degree).
Output: Euler Path Exists
```

**Example 3: Neither Exists**

```
Graph: A star with 3 leaves (center degree 3, leaves degree 1)
Degrees: More than 2 vertices have odd degree.
Output: Neither Euler Path nor Circuit Exists
```

---

### Conditions for Existence (Undirected Graph)

| Condition                          | Euler Circuit | Euler Path     |
|------------------------------------|---------------|----------------|
| All vertices have **even degree**  | Yes           | Yes            |
| Exactly **2 vertices** have odd degree | No         | Yes            |
| More than **2 vertices** have odd degree | No      | No             |

**Note:** The graph must also be **connected** (or have exactly one non-trivial connected component).

---

### Approach: Hierholzer’s Algorithm

Hierholzer’s algorithm is an efficient way to find the **Eulerian Path** or **Circuit**:

1. Check the degree conditions.
2. Start DFS from a vertex with odd degree (for path) or any vertex (for circuit).
3. Keep removing edges as we traverse.
4. When stuck, add the current path to the result.
5. The final path (in reverse) gives the Eulerian path/circuit.

---

### Code (C++)

```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    void dfs(int node, vector<vector<int>>& adj, vector<int>& path) {
        while (!adj[node].empty()) {
            int next = adj[node].back();
            adj[node].pop_back();
            
            // Remove the reverse edge as well (undirected graph)
            for (auto it = adj[next].begin(); it != adj[next].end(); ++it) {
                if (*it == node) {
                    adj[next].erase(it);
                    break;
                }
            }
            dfs(next, adj, path);
        }
        path.push_back(node);
    }

    vector<int> findEulerPath(int n, vector<vector<int>>& edges) {
        vector<vector<int>> adj(n);
        vector<int> degree(n, 0);

        // Build adjacency list and calculate degrees
        for (auto& e : edges) {
            int u = e[0], v = e[1];
            adj[u].push_back(v);
            adj[v].push_back(u);
            degree[u]++;
            degree[v]++;
        }

        // Count odd degree vertices
        int oddCount = 0, start = 0;
        for (int i = 0; i < n; i++) {
            if (degree[i] % 2 == 1) {
                oddCount++;
                start = i;
            }
        }

        if (oddCount > 2) {
            return {}; // Neither path nor circuit
        }

        vector<int> path;
        dfs(start, adj, path);
        reverse(path.begin(), path.end());

        // Check if all edges are used
        for (int i = 0; i < n; i++) {
            if (!adj[i].empty()) {
                return {}; // Graph is not connected or invalid
            }
        }

        return path;
    }
};

// ==================== DRIVER CODE ====================
int main() {
    Solution sol;

    // Example 1: Euler Circuit
    vector<vector<int>> edges1 = {{0,1},{1,2},{2,0},{0,3},{3,4},{4,0}};
    vector<int> res1 = sol.findEulerPath(5, edges1);
    cout << "Example 1 (Circuit): ";
    if (res1.empty()) cout << "Not Possible";
    else {
        for (int node : res1) cout << node << " ";
    }
    cout << endl;

    // Example 2: Euler Path
    vector<vector<int>> edges2 = {{0,1},{1,2},{2,3}};
    vector<int> res2 = sol.findEulerPath(4, edges2);
    cout << "Example 2 (Path): ";
    if (res2.empty()) cout << "Not Possible";
    else {
        for (int node : res2) cout << node << " ";
    }
    cout << endl;

    // Example 3: Not Possible
    vector<vector<int>> edges3 = {{0,1},{0,2},{0,3}};
    vector<int> res3 = sol.findEulerPath(4, edges3);
    cout << "Example 3: ";
    if (res3.empty()) cout << "Not Possible";
    else {
        for (int node : res3) cout << node << " ";
    }
    cout << endl;

    return 0;
}
```

---

### Time & Space Complexity

- **Time Complexity:** O(V + E)
- **Space Complexity:** O(V + E)

---

### Notes

- The code above finds the **Eulerian Path** (if it exists). For a circuit, start from any vertex with even degree.
- The algorithm works for **undirected** graphs.
- For **directed** graphs, the conditions change (in-degree == out-degree for circuit).
- This is also known as **Hierholzer’s Algorithm**.
