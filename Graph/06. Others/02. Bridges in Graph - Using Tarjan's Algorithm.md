## Bridge in a Graph | Critical Connections in a Network

---

### Problem Statement

Given an **undirected connected graph**, find all **bridges** (critical connections).

A **bridge** is an edge whose removal increases the number of connected components in the graph.

---

### Examples

**Example 1:**

```
Input: n = 4, connections = [[0,1],[1,2],[2,0],[1,3]]
Output: [[1,3]]
```

**Example 2:**

```
Input: n = 2, connections = [[0,1]]
Output: [[0,1]]
```

---

### Brute Force Approach

**Idea:**  
For every edge, temporarily remove it and check whether the graph remains connected using DFS or BFS. If removing the edge disconnects the graph, then that edge is a **bridge**.

**Code (Brute Force)**

```cpp
#include <bits/stdc++.h>
using namespace std;

bool isConnected(int V, vector<int> adj[], int skipU, int skipV) {
    vector<bool> visited(V, false);
    queue<int> q;
    q.push(0);
    visited[0] = true;
    int count = 1;

    while (!q.empty()) {
        int node = q.front(); q.pop();
        for (int neigh : adj[node]) {
            if ((node == skipU && neigh == skipV) || (node == skipV && neigh == skipU))
                continue;
            if (!visited[neigh]) {
                visited[neigh] = true;
                count++;
                q.push(neigh);
            }
        }
    }
    return count == V;
}

vector<vector<int>> findBridgesBruteForce(int V, vector<vector<int>>& connections) {
    vector<int> adj[V];
    for (auto& e : connections) {
        adj[e[0]].push_back(e[1]);
        adj[e[1]].push_back(e[0]);
    }

    vector<vector<int>> bridges;
    for (auto& edge : connections) {
        int u = edge[0], v = edge[1];
        if (!isConnected(V, adj, u, v)) {
            bridges.push_back({u, v});
        }
    }
    return bridges;
}

// ==================== BRUTE FORCE DRIVER ====================
int main() {
    // Example 1
    vector<vector<int>> conn1 = {{0,1},{1,2},{2,0},{1,3}};
    vector<vector<int>> res1 = findBridgesBruteForce(4, conn1);
    cout << "Brute Force Bridges: ";
    for (auto& e : res1) cout << "[" << e[0] << "," << e[1] << "] ";
    cout << endl;

    return 0;
}
```

**Time Complexity:** O(E × (V + E))

### Optimized Approach (Tarjan’s Algorithm)

**Bridge Condition:**

An edge `u → v` is a **bridge** if:

> `low[v] > disc[u]`

---

### Code (Optimized - Tarjan’s)

```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    void DFS(int node, int parent, vector<int> adj[], vector<bool>& visited,
             vector<int>& disc, vector<int>& low, vector<vector<int>>& bridges, int& timer) {
        
        visited[node] = 1;
        disc[node] = low[node] = timer++;

        for (int j = 0; j < adj[node].size(); j++) {
            int neib = adj[node][j];

            if (neib == parent) continue;

            else if (visited[neib]) {
                low[node] = min(low[node], disc[neib]);
            } 
            else {
                DFS(neib, node, adj, visited, disc, low, bridges, timer);

                low[node] = min(low[node], low[neib]);

                if (low[neib] > disc[node]) {
                    bridges.push_back({node, neib});
                }
            }
        }
    }

    vector<vector<int>> criticalConnections(int V, vector<vector<int>>& connections) {
        vector<int> adj[V];
        
        for (auto& e : connections) {
            adj[e[0]].push_back(e[1]);
            adj[e[1]].push_back(e[0]);
        }

        vector<bool> visited(V, 0);
        vector<int> disc(V);
        vector<int> low(V);
        vector<vector<int>> bridges;
        int timer = 0;

        DFS(0, -1, adj, visited, disc, low, bridges, timer);

        return bridges;
    }
};

// ==================== OPTIMIZED DRIVER ====================
int main() {
    Solution sol;

    vector<vector<int>> conn1 = {{0,1},{1,2},{2,0},{1,3}};
    vector<vector<int>> res1 = sol.criticalConnections(4, conn1);
    cout << "Optimized Bridges: ";
    for (auto& e : res1) cout << "[" << e[0] << "," << e[1] << "] ";
    cout << endl;

    return 0;
}
```

---

### Time & Space Complexity (Optimized)

- **Time Complexity:** O(V + E)
- **Space Complexity:** O(V + E)

---

### Notes

- Brute Force is simple but slow.
- Tarjan’s Algorithm is the efficient way to find all bridges in linear time.
- Both implementations follow the same style (`vector<int> adj[]`).
