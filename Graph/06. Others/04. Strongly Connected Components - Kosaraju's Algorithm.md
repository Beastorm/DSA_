## Strongly Connected Components (SCC) - Kosaraju's and Tarjan's Algorithms

---

### Problem Statement

Given a **Directed Graph** with `V` vertices and `E` edges, find all **Strongly Connected Components (SCCs)** or the **number of SCCs** in the graph.

A **Strongly Connected Component** is a maximal subgraph where every vertex is reachable from every other vertex.

---

### Examples

**Example 1:**
```
Graph:
0 → 1 ← 2
↑   ↓
└── 3
    4

Output (Number of SCCs): 3
SCCs: {0,1,3}, {2}, {4}
```

**Example 2:**
```
Graph:
0 → 1 → 2
    ↓   ↓
    3 → 4

Output: 5 (No cycles)
```

### Kosaraju's Algorithm

#### Intuition
Kosaraju’s algorithm uses the concept of **finishing times** from DFS and the **transposed graph**.

**Key Idea:**
- In a directed graph, nodes in the same SCC have a cycle between them.
- If we perform DFS and record nodes in the order of their **finishing times**, and then process them in reverse on the **transposed graph**, each DFS traversal will exactly isolate one SCC.

#### Algorithm Steps

1. **DFS on original graph** → Perform DFS and store nodes in a stack based on their finishing times.
2. **Transpose the graph** → Create a new graph with all edges reversed.
3. **DFS on transposed graph** → Pop nodes from the stack one by one. Each DFS traversal from an unvisited node gives one SCC.

#### C++ Code (Kosaraju)

```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
private:
    void dfs(int node, vector<int>& vis, vector<int> adj[], stack<int>& st) {
        vis[node] = 1;
        for (int neighbor : adj[node]) {
            if (!vis[neighbor]) {
                dfs(neighbor, vis, adj, st);
            }
        }
        st.push(node);
    }

    void dfsTranspose(int node, vector<int>& vis, vector<int> adjT[]) {
        vis[node] = 1;
        for (int neighbor : adjT[node]) {
            if (!vis[neighbor]) {
                dfsTranspose(neighbor, vis, adjT);
            }
        }
    }

public:
    int kosaraju(int V, vector<int> adj[]) {
        vector<int> vis(V, 0);
        stack<int> st;

        for (int i = 0; i < V; i++) {
            if (!vis[i]) {
                dfs(i, vis, adj, st);
            }
        }

        vector<int> adjT[V];
        for (int u = 0; u < V; u++) {
            for (int v : adj[u]) {
                adjT[v].push_back(u);
            }
        }

        fill(vis.begin(), vis.end(), 0);
        int sccCount = 0;

        while (!st.empty()) {
            int node = st.top();
            st.pop();

            if (!vis[node]) {
                sccCount++;
                dfsTranspose(node, vis, adjT);
            }
        }

        return sccCount;
    }
};

// ==================== DRIVER CODE ====================
int main() {
    int V = 5;
    vector<int> adj[V];

    adj[0].push_back(1);
    adj[1].push_back(2);
    adj[2].push_back(0);
    adj[1].push_back(3);
    adj[3].push_back(4);

    // Kosaraju
    Solution obj;
    int sccCount = obj.kosaraju(V, adj);
    cout << "Number of SCCs (Kosaraju): " << sccCount << endl;

    // Tarjan
    vector<vector<int>> result = tarjans(V, adj);
    cout << "\nStrongly Connected Components (Tarjan):" << endl;
    for (auto& comp : result) {
        for (int node : comp) cout << node << " ";
        cout << endl;
    }

    return 0;
}
```

### Tarjan's Algorithm

#### Intuition
Tarjan’s algorithm finds all SCCs in a **single DFS pass** using two values:

- `disc[node]`: Discovery time of the node (when it is first visited)
- `low[node]`: The smallest discovery time of any node that can be reached from the subtree rooted at this node (including itself)

**Key Idea:**
During DFS, if a node’s `disc` time equals its `low` time, it means this node cannot reach any earlier node. This node is the **root** of an SCC. All nodes currently in the stack from this node to the top form one SCC.

#### Algorithm Steps

1. Perform **DFS** while maintaining `disc[]`, `low[]`, and a stack.
2. For every node, update `low` using children and back edges.
3. When `disc[node] == low[node]`, pop all nodes from the stack until `node`. This forms one SCC.

#### C++ Code (Tarjan)

```cpp
#include <iostream>
#include <vector>
#include <stack>
#include <algorithm>

using namespace std;

// Helper function for Depth First Search (DFS) traversal
void DFS(int node, vector<int> adj[], vector<bool>& visited, vector<int>& disc,
         vector<int>& low, stack<int>& st, vector<bool>& inStack,
         vector<vector<int>>& ans, int& timer) {

    // Mark the current node as visited and initialize its discovery time and low value
    visited[node] = true;
    disc[node] = low[node] = timer++; // increment timer after assignment
    
    // Push the node onto the recursion stack and mark it as present in the stack
    st.push(node);
    inStack[node] = true;

    // Iterate through all adjacent vertices (neighbors) of the current node
    for (int j = 0; j < adj[node].size(); j++) {
        int neighbor = adj[node][j];

        // Case 1: If the neighbor has not been visited yet, visit it recursively
        if (!visited[neighbor]) {
            DFS(neighbor, adj, visited, disc, low, st, inStack, ans, timer);
            
            // Upon backtracking, update the low value of the current node.
            // The low value becomes the minimum of its current low and the neighbor's low.
            low[node] = min(low[node], low[neighbor]);
        } 
        // Case 2: If the neighbor is already visited AND is currently in the stack,
        // it means we found a back edge (a cycle).
        else if (inStack[neighbor]) {
            // Update the low value using the discovery time of the neighbor
            low[node] = min(low[node], disc[neighbor]);
        }
    }

    // If the current node is the root/head of a Strongly Connected Component (SCC)
    if (disc[node] == low[node]) {
        vector<int> temp;
        
        // Pop nodes from the stack until we reach the current node
        while (st.top() != node) {
            temp.push_back(st.top());
            inStack[st.top()] = false; // Mark node as no longer in stack
            st.pop();
        }
        // Finally, pop the root node itself
        temp.push_back(st.top());
        inStack[st.top()] = false;
        st.pop();

        // Sort the individual component elements for normalized output
        sort(temp.begin(), temp.end());
        ans.push_back(temp); // Store the found component
    }
}

// Main function to find Strongly Connected Components using Tarjan's algorithm
vector<vector<int>> tarjans(int v, vector<int> adj[]) {
    vector<vector<int>> ans;
    
    // disc[i] stores the time at which node i is discovered.
    // low[i] stores the lowest discovery time reachable from node i.
    vector<int> disc(v, -1), low(v, -1);
    
    // Tracks overall visited nodes and currently active nodes in the stack
    vector<bool> visited(v, false), inStack(v, false);
    stack<int> st;
    int timer = 0; // Global discovery time counter

    // Loop through all vertices to handle disconnected graphs
    for (int i = 0; i < v; i++) {
        if (!visited[i]) {
            DFS(i, adj, visited, disc, low, st, inStack, ans, timer);
        }
    }

    // Sort the outer list of components to maintain a predictable output order
    sort(ans.begin(), ans.end());
    return ans;
}

// ==================== DRIVER CODE (Tarjan) ====================
int main() { // Renamed to main for standard execution
    int V = 5;
    vector<int> adj[V];

    // Building a sample directed graph
    adj[0].push_back(1);
    adj[1].push_back(2);
    adj[2].push_back(0);
    adj[1].push_back(3);
    adj[3].push_back(4);

    // Run Tarjan's algorithm
    vector<vector<int>> result = tarjans(V, adj);

    // Print out the components found
    cout << "Strongly Connected Components (Tarjan):" << endl;
    for (auto& comp : result) {
        for (int node : comp) cout << node << " ";
        cout << endl;
    }
    return 0;
}
```

### Comparison: Kosaraju vs Tarjan

| Feature                    | Kosaraju                  | Tarjan                     |
|---------------------------|---------------------------|----------------------------|
| Number of DFS passes      | 2                         | 1                          |
| Needs graph transpose     | Yes                       | No                         |
| Uses stack                | Yes                       | Yes                        |
| Uses disc and low         | No                        | Yes                        |
| Time Complexity           | O(V + E)                  | O(V + E)                   |
| Ease of understanding     | Easier                    | Slightly harder            |
| Practical efficiency      | Good                      | Slightly better            |

---

### Time & Space Complexity

| Algorithm     | Time Complexity | Space Complexity |
|---------------|-----------------|------------------|
| Kosaraju      | O(V + E)        | O(V + E)         |
| Tarjan        | O(V + E)        | O(V + E)         |

---

### Notes

- **Kosaraju** is easier to understand and implement.
- **Tarjan** is more optimized (single pass) but has more complex logic.
- Both are standard algorithms for finding SCCs in directed graphs.
