## Articulation Point in a Graph | Tarjan's Algorithm

---

### Problem Statement

Given an **undirected connected graph**, find all **articulation points** (also called cut vertices).

An **articulation point** is a vertex whose removal increases the number of connected components in the graph.

---

### Examples

**Example 1:**

```
Graph:
0 — 1 — 2
|   |
3   4

Articulation Points: 1
Explanation: Removing vertex 1 disconnects the graph.
```

**Example 2:**

```
Graph: A complete graph with 4 vertices (all connected)

Articulation Points: None
Explanation: Removing any vertex still keeps the graph connected.
```

**Example 3:**

```
Graph: 0 — 1 — 2 — 3

Articulation Points: 1, 2
```

---

### Conditions for Articulation Points (Tarjan’s Algorithm)

During DFS traversal, we maintain two arrays:

- `disc[u]`: Discovery time of node `u`
- `low[u]`: Earliest discovery time reachable from the subtree rooted at `u`

**Rules:**

1. **Root Node**: If the root has **2 or more children** in the DFS tree → it is an articulation point.
2. **Non-root Node `u`**: If there exists a child `v` such that:
   > `low[v] ≥ disc[u]`
   then `u` is an articulation point.

---

### Approach (Tarjan’s Algorithm)

1. Perform DFS and track discovery time and low time.
2. Maintain a `parent` array to avoid considering the parent as a back edge.
3. Use a `ap[]` (articulation point) array to mark nodes.
4. For each node:
   - Update `low` values from children and back edges.
   - Apply the two conditions above.
5. Collect all marked articulation points.

---

### Code (C++)

```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
private:
    int timer = 0;
    vector<int> disc, low;
    vector<bool> ap;
    vector<vector<int>> adj;

    void dfs(int node, int parent) {
        disc[node] = low[node] = timer++;
        int children = 0;

        for (int neigh : adj[node]) {
            if (neigh == parent) continue;

            if (disc[neigh] == -1) {
                children++;
                dfs(neigh, node);

                low[node] = min(low[node], low[neigh]);

                // Condition for non-root node
                if (parent != -1 && low[neigh] >= disc[node]) {
                    ap[node] = true;
                }
            } 
            else {
                // Back edge
                low[node] = min(low[node], disc[neigh]);
            }
        }

        // Condition for root node
        if (parent == -1 && children > 1) {
            ap[node] = true;
        }
    }

public:
    vector<int> articulationPoints(int n, vector<vector<int>>& edges) {
        adj.resize(n);
        disc.assign(n, -1);
        low.assign(n, -1);
        ap.assign(n, false);

        // Build adjacency list
        for (auto& e : edges) {
            int u = e[0], v = e[1];
            adj[u].push_back(v);
            adj[v].push_back(u);
        }

        // Run DFS (handle disconnected graphs)
        for (int i = 0; i < n; i++) {
            if (disc[i] == -1) {
                dfs(i, -1);
            }
        }

        vector<int> result;
        for (int i = 0; i < n; i++) {
            if (ap[i]) result.push_back(i);
        }

        return result;
    }
};

// ==================== DRIVER CODE ====================
int main() {
    Solution sol;

    // Example 1
    vector<vector<int>> edges1 = {{0,1},{1,2},{0,3},{1,4}};
    vector<int> res1 = sol.articulationPoints(5, edges1);
    cout << "Example 1 Articulation Points: ";
    for (int node : res1) cout << node << " ";
    cout << endl;

    // Example 2: Complete graph
    vector<vector<int>> edges2 = {{0,1},{0,2},{0,3},{1,2},{1,3},{2,3}};
    vector<int> res2 = sol.articulationPoints(4, edges2);
    cout << "Example 2 Articulation Points: ";
    if (res2.empty()) cout << "None";
    else for (int node : res2) cout << node << " ";
    cout << endl;

    // Example 3: Line graph
    vector<vector<int>> edges3 = {{0,1},{1,2},{2,3}};
    vector<int> res3 = sol.articulationPoints(4, edges3);
    cout << "Example 3 Articulation Points: ";
    for (int node : res3) cout << node << " ";
    cout << endl;

    return 0;
}
```

### Time & Space Complexity

- **Time Complexity:** O(V + E)
- **Space Complexity:** O(V + E)

---

### Notes

- The algorithm uses **Tarjan’s DFS-based method** (same as for finding bridges).
- Root node handling is different from non-root nodes.
- Works for **undirected graphs**.
- For disconnected graphs, we run DFS from every unvisited node.
- This concept is heavily used in network reliability and critical node detection problems.
