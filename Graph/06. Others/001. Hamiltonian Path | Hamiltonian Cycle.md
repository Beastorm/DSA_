## Hamiltonian Path and Hamiltonian Cycle

---

### Problem Statement

Given an **undirected graph** with `n` vertices and `m` edges, determine whether a **Hamiltonian Path** exists in the graph.

- **Hamiltonian Path**: A path that visits every **vertex** exactly once.

---

### Examples

**Example 1: Hamiltonian Path Exists**
```
Graph: 0 — 1 — 2 — 3

Output: True
```

**Example 2: No Hamiltonian Path**
```
Graph: Star graph (one center connected to 3 leaves)

Output: False
```

---

### Approach (Backtracking)

We use **Backtracking** to solve the Hamiltonian Path problem.

**Intuition:**
- Try to visit every unvisited neighbor recursively.
- If we reach a state where all vertices are visited, a Hamiltonian Path is found.
- If not, we backtrack and try other paths.

---

### Code (C++)

```cpp
#include <iostream>
#include <vector>

using namespace std;

// DFS function to find the Hamiltonian Path using Backtracking
bool dfs(int node, vector<vector<int>>& adj, vector<bool>& visited, int& count, int n) {
    // Mark the current node as visited and increase the visited node count
    visited[node] = true;
    count++;

    // Base condition: If all nodes are visited exactly once, a Hamiltonian path is found
    if (count == n) {
        return true;
    }

    // Traverse all the adjacent neighbors of the current node
    for (int i = 0; i < adj[node].size(); i++) {
        int neighbor = adj[node][i];

        // If the neighbor is unvisited, apply DFS on it
        if (!visited[neighbor]) {
            if (dfs(neighbor, adj, visited, count, n)) {
                return true;
            }
        }
    }

    // Backtracking step: If no path is found from this node, unmark it and decrease the count
    visited[node] = false;
    count--;

    return false;
}

// Function to check if a Hamiltonian Path is present
bool checkHamiltonianPath(int n, int m, vector<vector<int>>& edges) {
    // Step 1: Create an Adjacency List from the given edges
    vector<vector<int>> adj(n);
    for (int i = 0; i < m; i++) {
        // Converting 1-based indexing to 0-based indexing
        int u = edges[i][0] - 1;
        int v = edges[i][1] - 1;

        // Undirected graph, so add edge in both directions
        adj[u].push_back(v);
        adj[v].push_back(u);
    }

    // Step 2: Since the Hamiltonian path can start from any node, test every node as a starting point
    for (int i = 0; i < n; i++) {
        vector<bool> visited(n, false);
        int count = 0;

        // If a path is found starting from node 'i', return true
        if (dfs(i, adj, visited, count, n)) {
            return true;
        }
    }

    // If all possible starting points are exhausted and no path is found
    return false;
}

// ==================== DRIVER CODE ====================
int main() {
    int n = 4, m = 3;
    vector<vector<int>> edges = {{1,2}, {2,3}, {3,4}};

    if (checkHamiltonianPath(n, m, edges))
        cout << "Hamiltonian Path Exists" << endl;
    else
        cout << "No Hamiltonian Path" << endl;

    return 0;
}
```

---

### Time & Space Complexity

**Time Complexity: O(N!)**

**Simple Derivation:**
- In the worst case (complete graph), every node is connected to every other node.
- From the first node, we have **(N-1)** choices.
- From the second node, we have **(N-2)** choices.
- From the third node, we have **(N-3)** choices.
- And so on...
- Total possibilities = **(N-1) × (N-2) × (N-3) × ... × 1 = (N-1)!**

Since we try starting from every node, overall complexity becomes **O(N!)**.

**Space Complexity: O(N)** (for `visited` array + recursion stack)

---

### Notes
- It tries every node as a starting point.
- Uses backtracking with proper `visited` and `count` tracking.
- Hamiltonian Path problem is **NP-Complete**.
