## Hamiltonian Path and Hamiltonian Cycle

---

### Problem Statement

Given an **undirected graph** with `n` vertices and `m` edges, determine whether a **Hamiltonian Path** exists in the graph.

- **Hamiltonian Path**: A path that visits every **vertex** exactly once.

---

### Examples

**Example 1: Hamiltonian Path Exists (No Cycle)**

```
Input:
n = 5, m = 4
edges = [[1,2], [2,3], [3,4], [2,5]]

Graph:
1 — 2 — 3 — 4
    |
    5

Hamiltonian Path: 1 → 2 → 5 → 3 → 4
Output: True
```

**Example 2: Hamiltonian Cycle Exists**

```
Input:
n = 5, m = 5
edges = [[1,2], [2,3], [3,4], [4,5], [5,1]]

Graph: Cycle Graph C5
1 — 2 — 3 — 4 — 5 — 1

Hamiltonian Path: 1 → 2 → 3 → 4 → 5
Hamiltonian Cycle: 1 → 2 → 3 → 4 → 5 → 1

Output: True (Both exist)
```

**Example 3: No Hamiltonian Path**

```
Input:
n = 5, m = 4
edges = [[1,2], [1,3], [1,4], [1,5]]

Graph: Star Graph
      2
      |
  3 — 1 — 4
      |
      5

Output: False
(After visiting one leaf, you must return to center → cannot visit all vertices exactly once)
```

**Example 4: Disconnected Graph**

```
Input:
n = 4, m = 2
edges = [[1,2], [3,4]]

Graph:
1 — 2     3 — 4

Output: False
(Hamiltonian Path cannot exist because the graph is disconnected)
```

---

### Approach (Backtracking)

We use **Backtracking** to solve the Hamiltonian Path problem.

**Intuition:**
- Try to visit every unvisited neighbor recursively.
- If we reach a state where all vertices are visited, a Hamiltonian Path is found.
- If not, we backtrack and try other paths.

---

### Code (C++)

#### 1. Hamiltonian Path

```cpp
#include <iostream>
#include <vector>

using namespace std;

// DFS function to find the Hamiltonian Path using Backtracking
bool dfs(int node, vector<vector<int>>& adj, vector<bool>& visited, int& count, int n) {
    visited[node] = true;
    count++;

    if (count == n) {
        return true;
    }

    for (int i = 0; i < adj[node].size(); i++) {
        int neighbor = adj[node][i];
        if (!visited[neighbor]) {
            if (dfs(neighbor, adj, visited, count, n)) {
                return true;
            }
        }
    }

    visited[node] = false;
    count--;
    return false;
}

// Function to check if a Hamiltonian Path exists
bool checkHamiltonianPath(int n, int m, vector<vector<int>>& edges) {
    vector<vector<int>> adj(n);
    for (int i = 0; i < m; i++) {
        int u = edges[i][0] - 1;
        int v = edges[i][1] - 1;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }

    for (int i = 0; i < n; i++) {
        vector<bool> visited(n, false);
        int count = 0;
        if (dfs(i, adj, visited, count, n)) {
            return true;
        }
    }
    return false;
}
```

#### 2. Hamiltonian Cycle

```cpp
// DFS function for Hamiltonian Cycle
bool dfsCycle(int node, int start, vector<vector<int>>& adj, vector<bool>& visited, int& count, int n) {
    visited[node] = true;
    count++;

    if (count == n) {
        // Check if last node is connected back to start
        for (int neighbor : adj[node]) {
            if (neighbor == start) {
                return true;
            }
        }
        return false;
    }

    for (int i = 0; i < adj[node].size(); i++) {
        int neighbor = adj[node][i];
        if (!visited[neighbor]) {
            if (dfsCycle(neighbor, start, adj, visited, count, n)) {
                return true;
            }
        }
    }

    visited[node] = false;
    count--;
    return false;
}

// Function to check if a Hamiltonian Cycle exists
bool checkHamiltonianCycle(int n, int m, vector<vector<int>>& edges) {
    vector<vector<int>> adj(n);
    for (int i = 0; i < m; i++) {
        int u = edges[i][0] - 1;
        int v = edges[i][1] - 1;
        adj[u].push_back(v);
        adj[v].push_back(u);
    }

    for (int i = 0; i < n; i++) {
        vector<bool> visited(n, false);
        int count = 0;
        if (dfsCycle(i, i, adj, visited, count, n)) {
            return true;
        }
    }
    return false;
}
```

#### Driver Code

```cpp
// ==================== DRIVER CODE ====================
int main() {
    int n = 4, m = 4;
    vector<vector<int>> edges = {{1,2}, {2,3}, {3,4}, {4,1}};

    if (checkHamiltonianPath(n, m, edges))
        cout << "Hamiltonian Path Exists" << endl;
    else
        cout << "No Hamiltonian Path" << endl;

    if (checkHamiltonianCycle(n, m, edges))
        cout << "Hamiltonian Cycle Exists" << endl;
    else
        cout << "No Hamiltonian Cycle" << endl;

    return 0;
}
```

### Time & Space Complexity

**Time Complexity: O(N!)**

**Simple Derivation:**
- In the worst case (complete graph), every node is connected to every other node.
- From the first node, we have **(N-1)** choices.
- From the second node, we have **(N-2)** choices.
- From the third node, we have **(N-3)** choices.
- And so on...
- Total possibilities = **(N-1) × (N-2) × (N-3) × ... × 1 = (N-1)!**

Since we try starting from every node, overall complexity becomes **O(N!)**.

**Space Complexity: O(N)** (for `visited` array + recursion stack)
