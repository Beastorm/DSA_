# Hamiltonian Path | Hamiltonian Cycle

## Definitions

### Hamiltonian Path
A **path** that visits **every vertex exactly once**. Can start and end at different vertices.

### Hamiltonian Cycle
A **cycle** that visits **every vertex exactly once** and returns to the starting vertex.

| Aspect | Hamiltonian Path | Hamiltonian Cycle |
|--------|-----------------|-------------------|
| **Visits each vertex** | Exactly once | Exactly once |
| **Closed** | No | Yes |
| **Returns to start** | Not required | Required |
| **Complexity** | NP-Complete | NP-Complete |
| **Edges needed** | V-1 | V |

---

## Key Differences: Euler vs Hamiltonian

| Property | Euler Path/Circuit | Hamiltonian Path/Cycle |
|----------|------------------|----------------------|
| **What to visit** | Every EDGE | Every VERTEX |
| **Polynomial time** | YES (O(V+E)) | NO (NP-Complete) |
| **Condition** | Degree-based | No simple condition |
| **Detectability** | Easy | Hard (no known polynomial algorithm) |

---

## Complexity & NP-Completeness

### Why is it Hard?

**Hamiltonian Cycle Problem is NP-Complete:**
- **Decision problem**: "Does a Hamiltonian cycle exist?" → YES/NO
- **Verification**: Easy (O(V) to check if given path is valid)
- **Finding**: No known polynomial-time algorithm
- **Brute force**: O(V!) permutations to check

### For Small V:
- V ≤ 20: Bitmask DP works well O(2^V × V²)
- V ≤ 10: Backtracking feasible
- V > 20: Only approximation or heuristics

---

## When Hamiltonian Path/Cycle Might Exist

### Sufficient Conditions (Not Necessary)

**Dirac's Theorem:**
- If every vertex has degree ≥ V/2 → Hamiltonian cycle exists

**Ore's Theorem:**
- If deg(u) + deg(v) ≥ V for every non-adjacent pair (u, v) → Hamiltonian cycle exists

**Examples:**
```
Complete Graph K_n → Always has Hamiltonian cycle
Complete Bipartite K_{n,n} → Always has Hamiltonian cycle
```

### No Simple Necessary Condition
```
Unlike Euler paths (degree-based check):
- There's NO simple necessary condition for Hamiltonian paths
- Must use search/DP algorithms
```

## Algorithms

### Algorithm 1: Backtracking (DFS)

**Idea**: Try to extend path vertex by vertex, backtrack if stuck.

```
HAMILTONIAN_PATH(v, path, visited):
    if length(path) == V:
        return path  // Found!
    
    for each neighbor u of v:
        if u not visited:
            visited[u] = true
            path.append(u)
            
            result = HAMILTONIAN_PATH(u, path, visited)
            if result != null:
                return result
            
            path.pop()
            visited[u] = false
    
    return null  // No path from here
```

**Complexity**: O(V!) worst case, but pruning helps in practice

### Algorithm 2: Bitmask DP (Held-Karp)

**Idea**: DP state = (current vertex, set of visited vertices)

```
dp[mask][v] = true if we can reach vertex v 
              having visited exactly the vertices in mask

HAMILTONIAN_DP(n, adj):
    dp = array of size [2^n][n]
    
    // Base case: single vertex paths
    for v in 0 to n-1:
        dp[1 << v][v] = true
    
    // Fill DP table
    for mask in 1 to 2^n - 1:
        for v in 0 to n-1:
            if dp[mask][v]:
                for u in adj[v]:
                    if (mask & (1 << u)) == 0:  // u not visited
                        newMask = mask | (1 << u)
                        dp[newMask][u] = true
    
    // Check final state
    fullMask = 2^n - 1
    for v in 0 to n-1:
        if dp[fullMask][v]:
            return true  // Hamiltonian path ends at v
    
    return false
```

**Complexity**: O(2^V × V²)

## Implementation - Backtracking

### Basic Backtracking Solution

```cpp
#include <bits/stdc++.h>
using namespace std;

class HamiltonianGraph {
private:
    vector<vector<int>> adj;
    vector<int> path;
    vector<bool> visited;
    int n;
    
public:
    HamiltonianGraph(int vertices) : n(vertices), 
                                     adj(vertices), 
                                     visited(vertices, false) {}
    
    void addEdge(int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    
    bool hamPathUtil(int v, int count) {
        // Mark current vertex as visited
        visited[v] = true;
        path.push_back(v);
        
        // If all vertices are included
        if (count == n) {
            return true;
        }
        
        // Try next vertex
        for (int u : adj[v]) {
            if (!visited[u]) {
                if (hamPathUtil(u, count + 1)) {
                    return true;
                }
            }
        }
        
        // Backtrack
        path.pop_back();
        visited[v] = false;
        return false;
    }
    
    vector<int> findHamiltonianPath() {
        // Try from each vertex as starting point
        for (int i = 0; i < n; i++) {
            fill(visited.begin(), visited.end(), false);
            path.clear();
            
            if (hamPathUtil(i, 1)) {
                return path;
            }
        }
        
        return {}; // Empty if no path found
    }
    
    bool hamCycleUtil(int v, int count) {
        visited[v] = true;
        path.push_back(v);
        
        // If all vertices visited, check if we can return to start
        if (count == n) {
            // Check if last vertex connects to first
            for (int u : adj[v]) {
                if (u == path[0]) {
                    return true;
                }
            }
            path.pop_back();
            visited[v] = false;
            return false;
        }
        
        // Try next vertex
        for (int u : adj[v]) {
            if (!visited[u]) {
                if (hamCycleUtil(u, count + 1)) {
                    return true;
                }
            }
        }
        
        path.pop_back();
        visited[v] = false;
        return false;
    }
    
    vector<int> findHamiltonianCycle() {
        // Hamiltonian cycle can start from any vertex (say 0)
        fill(visited.begin(), visited.end(), false);
        path.clear();
        
        if (hamCycleUtil(0, 1)) {
            path.push_back(path[0]); // Return to start
            return path;
        }
        
        return {}; // Empty if no cycle found
    }
};

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    HamiltonianGraph g(5);
    g.addEdge(0, 1);
    g.addEdge(0, 3);
    g.addEdge(1, 2);
    g.addEdge(1, 3);
    g.addEdge(2, 4);
    g.addEdge(3, 2);
    g.addEdge(3, 4);
    
    cout << "Graph structure:\n";
    cout << "  0 --- 1 --- 2\n";
    cout << "  |     |     |\n";
    cout << "  3 ----+---- 4\n\n";
    
    vector<int> path = g.findHamiltonianPath();
    if (!path.empty()) {
        cout << "Hamiltonian Path: ";
        for (int v : path) {
            cout << v << " ";
        }
        cout << "\n";
    } else {
        cout << "No Hamiltonian path found\n";
    }
    
    vector<int> cycle = g.findHamiltonianCycle();
    if (!cycle.empty()) {
        cout << "Hamiltonian Cycle: ";
        for (int v : cycle) {
            cout << v << " ";
        }
        cout << "\n";
    } else {
        cout << "No Hamiltonian cycle found\n";
    }
    
    return 0;
}
```

**Sample Output:**
```

  0 --- 1 --- 2
  |     |     |
  3 ----+---- 4

Hamiltonian Path: 0 1 2 4 3
Hamiltonian Cycle: 0 1 2 4 3 0
```

## Implementation - Bitmask DP

### Held-Karp Algorithm

```cpp
#include <bits/stdc++.h>
using namespace std;

class HamiltonianDP {
private:
    int n;
    vector<vector<int>> adj;
    vector<vector<int>> dp;      // dp[mask][v]
    vector<vector<int>> parent;  // For path reconstruction
    
public:
    HamiltonianDP(int vertices) : n(vertices), 
                                  adj(vertices),
                                  dp(1 << vertices, vector<int>(vertices, -1)),
                                  parent(1 << vertices, vector<int>(vertices, -1)) {}
    
    void addEdge(int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u);
    }
    
    bool hasHamiltonianPath() {
        // Base case: single vertex
        for (int v = 0; v < n; v++) {
            dp[1 << v][v] = 1;
        }
        
        // Fill DP table
        for (int mask = 1; mask < (1 << n); mask++) {
            for (int v = 0; v < n; v++) {
                if (dp[mask][v] == -1) continue;
                
                // Try extending to unvisited neighbor
                for (int u : adj[v]) {
                    if ((mask & (1 << u)) == 0) {  // u not visited
                        int newMask = mask | (1 << u);
                        if (dp[newMask][u] == -1) {
                            dp[newMask][u] = 1;
                            parent[newMask][u] = v;
                        }
                    }
                }
            }
        }
        
        // Check if all vertices visited (path exists)
        int fullMask = (1 << n) - 1;
        for (int v = 0; v < n; v++) {
            if (dp[fullMask][v] != -1) {
                return true;
            }
        }
        
        return false;
    }
    
    bool hasHamiltonianCycle() {
        if (!hasHamiltonianPath()) {
            return false;
        }
        
        // Check if last vertex connects back to start (vertex 0)
        int fullMask = (1 << n) - 1;
        for (int v = 0; v < n; v++) {
            if (dp[fullMask][v] != -1) {
                // Check if v connects to 0
                for (int u : adj[v]) {
                    if (u == 0) {
                        return true;
                    }
                }
            }
        }
        
        return false;
    }
    
    vector<int> reconstructPath() {
        int fullMask = (1 << n) - 1;
        int lastVertex = -1;
        
        // Find ending vertex
        for (int v = 0; v < n; v++) {
            if (dp[fullMask][v] != -1) {
                lastVertex = v;
                break;
            }
        }
        
        if (lastVertex == -1) return {};
        
        // Reconstruct path
        vector<int> path;
        int mask = fullMask;
        int v = lastVertex;
        
        while (v != -1) {
            path.push_back(v);
            int prevMask = mask ^ (1 << v);
            v = parent[mask][v];
            mask = prevMask;
        }
        
        reverse(path.begin(), path.end());
        return path;
    }
};

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    HamiltonianDP g(5);
    g.addEdge(0, 1);
    g.addEdge(0, 3);
    g.addEdge(1, 2);
    g.addEdge(1, 3);
    g.addEdge(2, 4);
    g.addEdge(3, 2);
    g.addEdge(3, 4);
    
    if (g.hasHamiltonianPath()) {
        cout << "Hamiltonian Path exists!\n";
        vector<int> path = g.reconstructPath();
        cout << "Path: ";
        for (int v : path) {
            cout << v << " ";
        }
        cout << "\n";
    } else {
        cout << "No Hamiltonian path\n";
    }
    
    if (g.hasHamiltonianCycle()) {
        cout << "Hamiltonian Cycle exists!\n";
    } else {
        cout << "No Hamiltonian cycle\n";
    }
    
    return 0;
}
```

## Step-by-Step Trace: Backtracking

### Example Graph

```
Graph:
  0 --- 1
  |     |
  3 --- 2

Edges: (0,1), (0,3), (1,2), (3,2)
```

### Finding Hamiltonian Path (starting from 0)

```
Start: v=0, count=1, path=[0], visited=[T,F,F,F]

Try neighbor 1:
  v=1, count=2, path=[0,1], visited=[T,T,F,F]
  
  Try neighbor 0: already visited
  Try neighbor 2:
    v=2, count=3, path=[0,1,2], visited=[T,T,T,F]
    
    Try neighbor 1: already visited
    Try neighbor 3:
      v=3, count=4, path=[0,1,2,3], visited=[T,T,T,T]
      
      count == n (4 == 4)
      FOUND! Return true
      Result: [0,1,2,3]

Hamiltonian Path: 0 → 1 → 2 → 3
```

### Finding Hamiltonian Cycle (from same path)

```
From path [0,1,2,3]:
Last vertex is 3
Check if 3 connects to 0 (start)
neighbors of 3: [0, 2]
3 connects to 0? YES!
CYCLE FOUND!

Hamiltonian Cycle: 0 → 1 → 2 → 3 → 0
```

## Bitmask DP Trace

### Example: Complete Graph K₄

```
Vertices: 0, 1, 2, 3 (all connected)

DP State: dp[mask][v] = can we visit vertices in 'mask' 
                        and end at vertex 'v'?

mask representation (binary):
  0001 = {0}
  0011 = {0,1}
  1111 = {0,1,2,3}

Base case (single vertices):
  dp[0001][0] = 1
  dp[0010][1] = 1
  dp[0100][2] = 1
  dp[1000][3] = 1

Transitions:
  From state dp[0011][1] (visited {0,1}, at 1):
    Can go to 2: dp[0111][2] = 1
    Can go to 3: dp[1011][3] = 1

Final check:
  mask = 1111 (all visited)
  If any dp[1111][v] = 1, path exists!
```

## Complexity Analysis

| Approach | Time | Space | Best For |
|----------|------|-------|----------|
| **Backtracking** | O(V!) worst case | O(V) | Small graphs (V ≤ 10) |
| **Bitmask DP** | O(2^V × V²) | O(2^V × V) | Medium graphs (V ≤ 20) |
| **Brute Force** | O(V! × E) | O(V) | Small V |

## Common Graph Types

### Hamiltonian Cycle Always Exists

```
1. Complete Graph K_n (n ≥ 3)
   Every vertex connected to every other
   
2. Cycle Graph C_n (n ≥ 3)
   Vertices in circle: 0-1-2-...-n-1-0
   
3. Complete Bipartite K_{n,n} (n ≥ 2)
   Two sets of n vertices, fully connected
```

### Hamiltonian Path Often Exists

```
1. Tree (n vertices)
   Usually has Hamiltonian path
   (might not have cycle)
   
2. Connected graph with high density
   More edges → more likely to have path
```

### No Hamiltonian Path/Cycle

```
1. Disconnected graph
   No Hamiltonian anything possible
   
2. Star graph (1 center, n-1 leaves)
   No Hamiltonian cycle
   Path only if visiting leaves in special order
   
3. Bipartite graph with unequal parts
   Can't have cycle (odd cycle impossible)
```

## Edge Cases

### Case 1: Single Vertex
```
n=1, no edges
Hamiltonian path: YES [0]
Hamiltonian cycle: NO (need at least 3 vertices for cycle)
```

### Case 2: Two Vertices
```
0 --- 1
Path: YES [0,1]
Cycle: NO (need self-loop or return edge)
```

### Case 3: Triangle (K₃)
```
  0
 / \
1---2

Path: YES [0,1,2]
Cycle: YES [0,1,2,0]
```

### Case 4: Bipartite Graph
```
0---1
|   |
2---3

Cycle: YES (even cycle)
Path: YES

But:
0---1
|   |
2---3
    |
    4

Cycle: NO (5 vertices, odd)
Path: YES
```

### Case 5: Star Graph
```
    0
   /|\
  1 2 3

Cycle: NO (can't return from leaves to center to other leaf to center)
Path: YES [1,0,2] (but only 3 vertices)
```
