# Euler Path and Euler Circuit in a Graph

## Definitions

### Euler Path
A **path** that visits **every edge exactly once**. Can start and end at different vertices.

### Euler Circuit
A **circuit** (closed path) that visits **every edge exactly once** and returns to the starting vertex.

| Aspect | Euler Path | Euler Circuit |
|--------|-----------|---------------|
| **Visits each edge** | Exactly once | Exactly once |
| **Closed** | No | Yes |
| **Returns to start** | Not required | Required |
| **Degree condition** | 0 or 2 odd-degree vertices | All even-degree vertices |

---

## Conditions for Existence

### Theorem: Conditions for Euler Path/Circuit

**Euler Circuit exists if and only if:**
- Graph is **connected** (considering only vertices with edges)
- **All vertices have even degree**

**Euler Path exists if and only if:**
- Graph is **connected** (considering only vertices with edges)
- **Exactly 0 or 2 vertices have odd degree**
  - 0 odd vertices → Euler circuit exists (special case of path)
  - 2 odd vertices → Euler path exists (from one odd to other odd)

### Proof Intuition

```
Degree Requirement Explanation:

For Euler Circuit:
  - Every time you enter a vertex, you must exit (except start/end)
  - Each edge used = 1 enter + 1 exit = 2 degree
  - So all vertices need even degree

For Euler Path (non-circuit):
  - Start vertex: 1 exit, then balanced enter/exit
  - End vertex: 1 enter, then balanced exits
  - Both have odd degree (1 extra for path endpoints)
  - Middle vertices still need even degree
```

## Algorithms

### Algorithm 1: Fleury's Algorithm

**Idea**: Greedily remove edges, avoiding bridges when possible.

```
FLEURY(Graph G, start vertex u):
    current = u
    path = []
    
    while has_edges(G):
        if current has outgoing edges:
            // Prefer non-bridge edge if possible
            v = next_vertex(current)
            remove_edge(current, v)
            path.append((current, v))
            current = v
        else:
            break
    
    return path
```

**Complexity**: O(E²) due to bridge-checking

### Algorithm 2: Hierholzer's Algorithm (Preferred)

**Idea**: Build circuit recursively, handle unused edges as separate components.

```
HIERHOLZER(Graph G, start vertex u):
    stack = [u]
    path = []
    
    while stack is not empty:
        v = stack.top()
        if v has outgoing edges:
            w = next_neighbor(v)
            remove_edge(v, w)
            stack.push(w)
        else:
            path.push_front(stack.pop())
    
    return path
```

**Complexity**: O(V + E)

## Implementation - Hierholzer's Algorithm

### Check if Euler Path/Circuit Exists

```cpp
bool canFindEulerPath(vector<vector<int>>& adj, int& startVertex) {
    int oddCount = 0;
    int n = adj.size();
    
    for (int i = 0; i < n; i++) {
        if (adj[i].size() % 2 == 1) {
            oddCount++;
            startVertex = i; // Start from odd-degree vertex if exists
        }
    }
    
    // 0 or 2 odd-degree vertices required
    if (oddCount == 0) {
        startVertex = 0; // Any vertex with edges
        return true;
    }
    if (oddCount == 2) {
        return true;
    }
    
    return false;
}
```

### Adjacency List (Undirected Graph)

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<int> eulerPath(vector<vector<int>>& adj, int start) {
    int n = adj.size();
    vector<int> degree(n);
    
    // Calculate degrees
    for (int i = 0; i < n; i++) {
        degree[i] = adj[i].size();
    }
    
    stack<int> st;
    vector<int> path;
    st.push(start);
    
    // Create edge count map for multigraph handling
    map<pair<int, int>, int> edges;
    for (int i = 0; i < n; i++) {
        for (int j : adj[i]) {
            if (i <= j) {
                edges[{i, j}]++;
            }
        }
    }
    
    while (!st.empty()) {
        int u = st.top();
        bool found = false;
        
        // Try to find next edge
        for (int v : adj[u]) {
            if (edges[{min(u, v), max(u, v)}] > 0) {
                edges[{min(u, v), max(u, v)}]--;
                st.push(v);
                found = true;
                break;
            }
        }
        
        if (!found) {
            path.push_back(st.top());
            st.pop();
        }
    }
    
    reverse(path.begin(), path.end());
    return path;
}

int main() {
    int n = 4, m = 4;
    vector<vector<int>> adj(n);
    
    // Build graph: A-B, B-C, C-D, D-A (0-1, 1-2, 2-3, 3-0)
    adj[0] = {1, 3};
    adj[1] = {0, 2};
    adj[2] = {1, 3};
    adj[3] = {2, 0};
    
    int start = 0;
    vector<int> path = eulerPath(adj, start);
    
    cout << "Euler Path: ";
    for (int v : path) {
        cout << v << " ";
    }
    cout << "\n";
    
    return 0;
}
```

### Adjacency Matrix (Directed Graph)

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<int> eulerPathDirected(vector<vector<int>>& adj, int start) {
    int n = adj.size();
    vector<vector<int>> graph = adj; // Copy to modify
    
    stack<int> st;
    vector<int> path;
    st.push(start);
    
    while (!st.empty()) {
        int u = st.top();
        bool found = false;
        
        for (int v = 0; v < n; v++) {
            if (graph[u][v] > 0) {
                graph[u][v]--;
                st.push(v);
                found = true;
                break;
            }
        }
        
        if (!found) {
            path.push_back(st.top());
            st.pop();
        }
    }
    
    reverse(path.begin(), path.end());
    return path;
}

int main() {
    int n = 4;
    vector<vector<int>> adj(n, vector<int>(n, 0));
    
    // Directed edges
    adj[0][1] = 1; // 0 → 1
    adj[1][2] = 1; // 1 → 2
    adj[2][3] = 1; // 2 → 3
    adj[3][0] = 1; // 3 → 0
    
    int start = 0;
    vector<int> path = eulerPathDirected(adj, start);
    
    cout << "Euler Path (Directed): ";
    for (int v : path) {
        cout << v << " ";
    }
    cout << "\n";
    
    return 0;
}
```

```
Iteration 5:
  u = 0, no more edges
  path = [0]
  stack = [0, 1, 2, 3]

Iteration 6:
  u = 3, no more edges
  path = [0, 3]
  stack = [0, 1, 2]

Iteration 7:
  u = 2, no more edges
```

## Complete Solution Template

```cpp
#include <bits/stdc++.h>
using namespace std;

class EulerGraph {
private:
    vector<vector<int>> adj;
    int n;
    
public:
    EulerGraph(int vertices) : n(vertices), adj(vertices) {}
    
    void addEdge(int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u); // For undirected
    }
    
    bool hasEulerPathOrCircuit(int& start) {
        vector<int> degree(n);
        
        for (int i = 0; i < n; i++) {
            degree[i] = adj[i].size();
        }
        
        int oddCount = 0;
        for (int i = 0; i < n; i++) {
            if (degree[i] % 2 == 1) {
                oddCount++;
                start = i;
            }
        }
        
        if (oddCount == 0 || oddCount == 2) {
            if (oddCount == 0) start = 0;
            return true;
        }
        
        return false;
    }
    
    vector<int> findEulerPath(int start) {
        vector<multiset<int>> graph(n);
        
        for (int i = 0; i < n; i++) {
            for (int j : adj[i]) {
                graph[i].insert(j);
            }
        }
        
        stack<int> st;
        vector<int> path;
        st.push(start);
        
        while (!st.empty()) {
            int u = st.top();
            
            if (!graph[u].empty()) {
                int v = *graph[u].begin();
                graph[u].erase(graph[u].find(v));
                graph[v].erase(graph[v].find(u));
                st.push(v);
            } else {
                path.push_back(u);
                st.pop();
            }
        }
        
        reverse(path.begin(), path.end());
        return path;
    }
};

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    EulerGraph g(4);
    g.addEdge(0, 1);
    g.addEdge(1, 2);
    g.addEdge(2, 3);
    g.addEdge(3, 0);
    
    int start = 0;
    if (g.hasEulerPathOrCircuit(start)) {
        vector<int> path = g.findEulerPath(start);
        
        cout << "Euler Path found: ";
        for (int v : path) {
            cout << v << " ";
        }
        cout << "\n";
    } else {
        cout << "No Euler path/circuit exists\n";
    }
    
    return 0;
}
```

## Key Properties

### Property 1: Connected Components
```
Only vertices with degree > 0 matter
Isolated vertices (degree 0) don't affect Euler path
```

### Property 2: Multigraph
```
Can have multiple edges between same vertices
Algorithms handle by counting occurrences
```

### Property 3: Self-Loops
```
Self-loop: edge from vertex to itself
Counts as 2 degree for that vertex
```

### Property 4: Directed vs Undirected
```
Undirected: degree = number of incident edges
Directed: 
  - In-degree + Out-degree = total degree
  - For Euler circuit: in-degree = out-degree for all vertices
  - For Euler path: at most 1 vertex has out-degree = in-degree + 1
                    at most 1 vertex has in-degree = out-degree + 1
```

---

## Directed Graph Conditions

**Euler Circuit (Directed):**
- All vertices with edges have **in-degree = out-degree**

**Euler Path (Directed):**
- All vertices except at most 2 have **in-degree = out-degree**
- At most 1 vertex has **out-degree = in-degree + 1** (start)
- At most 1 vertex has **in-degree = out-degree + 1** (end)

## Edge Cases

### Case 1: Single Vertex, No Edges
```
n=1, edges=0
Euler circuit exists: YES (empty path)
```

### Case 2: Single Edge
```
0 --- 1
Degrees: 0=1, 1=1 (two odd)
Euler path exists: YES (0-1 or 1-0)
Euler circuit: NO
```

### Case 3: Disconnected Graph
```
Component 1:    Component 2:
  0 --- 1         2 --- 3

Euler path exists: NO (4 components with edges)
```

### Case 4: Self-Loop
```
0: self-loop (degree 2)
1: edge to 0 (degree 1)
2: edge to 0 (degree 1)

Degrees: 0=4, 1=1, 2=1 (two odd vertices)
Euler path exists: YES
```

### Case 5: Multigraph
```
0 --- 1 (2 edges)
|_____|

Degrees: 0=3, 1=3 (two odd)
Euler path exists: YES
```
