# Print Shortest Path - Dijkstra's Algorithm

## Problem Statement

Given a weighted undirected graph with **V** vertices and **E** edges, find the **shortest distance** from source **S** to all vertices and **print the path**.

**Constraints:**
- All edge weights ≥ 0
- 1 ≤ V ≤ 10^5, 1 ≤ E ≤ 10^5

---

## Intuition

Always process the closest unvisited vertex first. Since weights are non-negative, the shortest distance to a vertex is finalized when we visit it for the first time.

---

## Solution

### Approach
1. Use min-heap (priority queue) to get closest vertex
2. Relax edges: update distances to neighbors
3. Track parent to reconstruct path
4. Continue until all vertices processed

### Code

```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    vector<int> shortestPath(int n, int m, vector<vector<int>>& edges) {
        // Create an adjacency list of pairs: node -> {adjacent_node, weight}
        vector<pair<int, int>> adj[n + 1];
        for (auto it : edges) {
            adj[it].push_back({it[1], it[2]});
            adj[it[1]].push_back({it, it[2]});
        }

        // Min-heap priority queue storing pair of {dist, node}
        priority_queue<pair<int, int>, vector<pair<int, int>>, greater<pair<int, int>>> pq;
        vector<int> dist(n + 1, 1e9); // Initialize distances to infinity
        vector<int> parent(n + 1);    // Array to reconstruct path
        
        // Initialize parent of each node as itself
        for (int i = 1; i <= n; i++) {
            parent[i] = i;
        }

        dist[1] = 0;      // Distance to source node 1 is 0
        pq.push({0, 1});

        while (!pq.empty()) {
            auto it = pq.top();
            int dis = it.first;
            int node = it.second;
            pq.pop();

            // Skip processing if a shorter path to this node has already been resolved
            if (dis > dist[node]) continue;

            for (auto it : adj[node]) {
                int adjNode = it.first;
                int wt = it.second;

                // Relaxation step
                if (dis + wt < dist[adjNode]) {
                    dist[adjNode] = dis + wt;
                    pq.push({dist[adjNode], adjNode});
                    parent[adjNode] = node; // Record where we came from
                }
            }
        }

        // If the destination node n is unreachable, return {-1}
        if (dist[n] == 1e9) return {-1};

        // Reconstruct the path backwards from destination (n) to source (1)
        vector<int> ans;
        int node = n;
        while (parent[node] != node) {
            ans.push_back(node);
            node = parent[node];
        }
        ans.push_back(1);

        // Prepend the total shortest path weight to the answer
        ans.push_back(dist[n]);
        
        // Reverse to get the format: [weight, source, ..., destination]
        reverse(ans.begin(), ans.end());
        return ans;
    }
};

// --- DRIVER CODE ---
int main() {
    // Number of vertices (n) and edges (m)
    // Using Striver's 1-indexed graph from the tutorial
    int n = 5; 
    int m = 6; 

    // Edges of the undirected graph: {node_u, node_v, edge_weight}
    vector<vector<int>> edges = {
        {1, 2, 2},
        {2, 5, 5},
        {1, 4, 1},
        {4, 3, 3},
        {3, 5, 1},
        {2, 3, 4}
    };

    Solution solver;
    vector<int> result = solver.shortestPath(n, m, edges);

    // Output processing
    if (result.size() == 1 && result == -1) {
        cout << "No path exists from node 1 to node " << n << endl;
    } else {
        int totalWeight = result;
        cout << "Total Shortest Path Weight: " << totalWeight << endl;
        cout << "Shortest Path Sequence: ";
        for (size_t i = 1; i < result.size(); ++i) {
            cout << result[i] << (i == result.size() - 1 ? "" : " -> ");
        }
        cout << endl;
    }

    return 0;
}
```

---

## Complexity
- **Time:** O((V + E) log V)
- **Space:** O(V + E)

---

## Key Points

✅ Use **min-heap** for efficiency  
✅ Skip vertex if already processed with better distance  
✅ Track **parent** for path reconstruction  
✅ Works only with **non-negative weights**  

---

## When to Use Other Algorithms

| Condition | Algorithm |
|-----------|-----------|
| Non-negative weights | Dijkstra ✓ |
| Negative weights, no cycle | Bellman-Ford |
| All pairs shortest path | Floyd-Warshall |
| Unweighted graph | BFS |
