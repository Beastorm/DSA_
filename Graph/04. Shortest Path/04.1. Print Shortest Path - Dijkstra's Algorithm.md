# Print Shortest Path - Dijkstra's Algorithm

## Problem Statement

Given a weighted undirected graph with **V** vertices and **E** edges, find the **shortest distance** from source **S** to all vertices and **print the path**.

**Constraints:**
- All edge weights ≥ 0
- 1 ≤ V ≤ 10^5, 1 ≤ E ≤ 10^5

---

## Intuition

Always process the closest unvisited vertex first. Since weights are non-negative, the shortest distance to a vertex is finalized when we visit it for the first time.

---

## Solution

### Approach
1. Use min-heap (priority queue) to get closest vertex
2. Relax edges: update distances to neighbors
3. Track parent to reconstruct path
4. Continue until all vertices processed

### Code

```cpp
#include <bits/stdc++.h>
using namespace std;

const long long INF = 1e18;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, m;
    cin >> n >> m;
    
    vector<vector<pair<int, long long>>> adj(n);
    
    for (int i = 0; i < m; i++) {
        int u, v;
        long long w;
        cin >> u >> v >> w;
        adj[u].push_back({v, w});
        adj[v].push_back({u, w});  // Undirected
    }
    
    int source;
    cin >> source;
    
    // Dijkstra's Algorithm
    vector<long long> dist(n, INF);
    vector<int> parent(n, -1);
    priority_queue<pair<long long, int>, 
                   vector<pair<long long, int>>, 
                   greater<pair<long long, int>>> pq;
    
    dist[source] = 0;
    pq.push({0, source});
    
    while (!pq.empty()) {
        auto [d, u] = pq.top();
        pq.pop();
        
        if (d > dist[u]) continue;
        
        for (auto [v, w] : adj[u]) {
            if (dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                parent[v] = u;
                pq.push({dist[v], v});
            }
        }
    }
    
    // Print results
    cout << "Shortest paths from vertex " << source << ":\n";
    cout << "─────────────────────────────\n";
    
    for (int i = 0; i < n; i++) {
        if (dist[i] == INF) {
            cout << "To " << i << ": UNREACHABLE\n";
        } else {
            cout << "To " << i << " | Distance: " << dist[i] << " | Path: ";
            
            // Reconstruct path
            vector<int> path;
            int curr = i;
            while (curr != -1) {
                path.push_back(curr);
                curr = parent[curr];
            }
            reverse(path.begin(), path.end());
            
            for (int j = 0; j < path.size(); j++) {
                cout << path[j];
                if (j < path.size() - 1) cout << " → ";
            }
            cout << "\n";
        }
    }
    
    return 0;
}
```

---

## Example

### Input
```
5 6
0 1 4
0 2 2
1 2 1
1 3 5
2 3 8
3 4 2
0
```

### Output
```
Shortest paths from vertex 0:
─────────────────────────────
To 0 | Distance: 0 | Path: 0
To 1 | Distance: 3 | Path: 0 → 2 → 1
To 2 | Distance: 2 | Path: 0 → 2
To 3 | Distance: 8 | Path: 0 → 2 → 1 → 3
To 4 | Distance: 10 | Path: 0 → 2 → 1 → 3 → 4
```

---

## Complexity
- **Time:** O((V + E) log V)
- **Space:** O(V + E)

---

## Key Points

✅ Use **min-heap** for efficiency  
✅ Skip vertex if already processed with better distance  
✅ Track **parent** for path reconstruction  
✅ Works only with **non-negative weights**  

---

## When to Use Other Algorithms

| Condition | Algorithm |
|-----------|-----------|
| Non-negative weights | Dijkstra ✓ |
| Negative weights, no cycle | Bellman-Ford |
| All pairs shortest path | Floyd-Warshall |
| Unweighted graph | BFS |
