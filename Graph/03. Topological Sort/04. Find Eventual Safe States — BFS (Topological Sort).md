## Find Eventual Safe States — BFS (Topological Sort)

### Problem Statement:

Given a directed graph with `V` vertices, find all **safe nodes** — nodes from which every possible path leads to a **terminal node** (a node with no outgoing edges). Return the safe nodes in **sorted ascending order**.

---

## Examples

**Example 1:**
- **Input:** `N=7, E=7`
- **Output:** `{2, 4, 5, 6}`
```
  Graph:
  0 → {1,2}
  1 → {2,3}
  2 → {5}
  3 → {0}      ← back edge → CYCLE: 0→1→3→0
  4 → {5}
  5 → {}       ← terminal
  6 → {}       ← terminal

  Cycle nodes: {0, 1, 3}  → NOT safe
  Safe nodes: {2, 4, 5, 6} ✓
```

**Example 2:**
- **Input:** `N=4, E=4`
- **Output:** `{3}`
```
  0 → {1,2}, 1 → {2}, 2 → {0}, 3 → {}
  Cycle: 0→2→0 → unsafe: {0,1,2}
  Safe: {3} (terminal) ✓
```

---

## Key Insight
```
  Safe node = NOT in a cycle AND does NOT lead to a cycle
  Unsafe node = in a cycle OR leads to a cycle

  How Topological Sort helps:
  Topo sort processes nodes with 0 in-degree first.
  Cyclic nodes NEVER reach in-degree 0 → never processed.
  Processed nodes = SAFE nodes!

  BUT: Normal topo sort uses IN-DEGREE.
  We want to find terminal nodes (0 OUT-DEGREE).

  Solution: REVERSE ALL EDGES!
  → Terminal nodes (0 out-degree) become 0 in-degree nodes in reversed graph
  → Apply Kahn's BFS on reversed graph
  → Processed nodes = safe nodes ✓
```

---

## Algorithm
```
  Step 1: Reverse all edges
    Original: u→v becomes v→u in reversed graph
    Also compute indegree in reversed graph:
    indegree[u]++ for each original edge u→v

  Step 2: Push all nodes with indegree=0 (terminal nodes)
    These are nodes with no outgoing edges in ORIGINAL graph

  Step 3: BFS on reversed graph
    Pop node → add to safeNodes
    For each neighbor in reversed graph:
        indegree[neighbor]--
        If == 0 → push to queue

  Step 4: Sort safeNodes and return
```
```
  Why reverse?

  Original:           Reversed:
  0→1, 1→3, 3→0       0←1, 1←3, 3←0
  (cycle, can't topo  (also cycle, topo
  sort)                sort skips them)

  Terminal 5 (out=0): Reversed → in-degree=0 → starts BFS!
  All nodes leading to 5 safely get processed ✓
```

## Code
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    vector<int> eventualSafeNodes(int V, vector<int> adj[]) {

        // Step 1: Build reversed graph + compute indegree
        vector<int> adjRev[V];
        int indegree[V] = {0};

        for (int i = 0; i < V; i++) {
            for (int neighbor : adj[i]) {
                adjRev[neighbor].push_back(i); // Reverse: neighbor→i
                indegree[i]++;                 // i has outgoing edge → in reversed graph
            }
        }

        // Step 2: Push all terminal nodes (indegree=0 in reversed = out-degree=0 in original)
        queue<int> q;
        for (int i = 0; i < V; i++) {
            if (indegree[i] == 0) q.push(i);
        }

        // Step 3: BFS on reversed graph
        vector<int> safeNodes;
        while (!q.empty()) {
            int node = q.front();
            q.pop();
            safeNodes.push_back(node); // This node is SAFE

            for (int parent : adjRev[node]) {
                indegree[parent]--;
                if (indegree[parent] == 0) {
                    q.push(parent);
                }
            }
        }

        // Step 4: Sort and return
        sort(safeNodes.begin(), safeNodes.end());
        return safeNodes;
    }
};

int main() {
    int V = 7;
    vector<int> adj[V];
    adj[0] = {1, 2};
    adj[1] = {2, 3};
    adj[2] = {5};
    adj[3] = {0};
    adj[4] = {5};
    adj[5] = {};
    adj[6] = {};

    Solution obj;
    vector<int> safe = obj.eventualSafeNodes(V, adj);

    cout << "Safe nodes: ";
    for (int x : safe) cout << x << " ";
    cout << endl; // 2 4 5 6

    return 0;
}
```

## Step-by-Step Trace
```
  V=7, adj: 0→{1,2}, 1→{2,3}, 2→{5}, 3→{0}, 4→{5}, 5→{}, 6→{}

  ── Step 1: Build Reversed Graph + indegree ──

  Original edges → reversed:
  0→1: adjRev[1].push(0), indegree[0]++
  0→2: adjRev[2].push(0), indegree[0]++
  1→2: adjRev[2].push(1), indegree[1]++
  1→3: adjRev[3].push(1), indegree[1]++
  2→5: adjRev[5].push(2), indegree[2]++
  3→0: adjRev[0].push(3), indegree[3]++
  4→5: adjRev[5].push(4), indegree[4]++

  indegree = [2, 2, 1, 1, 1, 0, 0]
              0  1  2  3  4  5  6

  adjRev:
  0: [3]       ← who points to 0? node 3
  1: [0]       ← who points to 1? node 0
  2: [0,1]     ← who points to 2? nodes 0,1
  3: [1]       ← who points to 3? node 1
  4: []        ← no one points to 4
  5: [2,4]     ← who points to 5? nodes 2,4
  6: []        ← no one points to 6

  ── Step 2: Push indegree=0 nodes ──
  indegree[5]=0 → push 5
  indegree[6]=0 → push 6
  queue=[5,6]

  ── Step 3: BFS ──

  Pop 5: safeNodes=[5]
    adjRev[5]=[2,4]: neighbors in reversed graph
    indegree[2]=1→0 → push 2
    indegree[4]=1→0 → push 4
    queue=[6,2,4]

  Pop 6: safeNodes=[5,6]
    adjRev[6]=[] → nothing to process
    queue=[2,4]

  Pop 2: safeNodes=[5,6,2]
    adjRev[2]=[0,1]:
    indegree[0]=2→1 (not 0)
    indegree[1]=2→1 (not 0)
    queue=[4]

  Pop 4: safeNodes=[5,6,2,4]
    adjRev[4]=[] → nothing
    queue=[]

  BFS done. Nodes 0,1,3 NEVER processed (stuck in cycle).

  ── Step 4: Sort ──
  safeNodes = [5,6,2,4] → sorted → [2,4,5,6] ✓
```

  t=0: {5,6} → safe

  t=2: (0,1 not safe yet — still in cycle 0→1→3→0)

## DFS vs BFS for Safe Nodes
```
  DFS (previous approach):       BFS/Topo (this approach):
  ───────────────────────        ────────────────────────────────
  vis[] + pathVis[]              Reverse graph + Kahn's BFS
  check[] = safe/unsafe          Processed nodes = safe
  O(V+E)                         O(V+E) + O(N logN) sort

  Both work, same complexity.
  BFS: iterative, no recursion stack overflow risk ✓
  DFS: slightly simpler code for some ✓
```

---

## Edge Cases
```
  All terminal (no edges):      All in one cycle:
  V=3, no edges                 0→1→2→0

  All indegree=0 → all safe     All indegree≥1 in reversed
  return [0,1,2] ✓              Nothing processed → return [] ✓

  Mixed:                        Single terminal node:
  Some cycle, some safe         V=1, no edges
                                indegree[0]=0 → safe
  Safe = non-cycle nodes ✓      return [0] ✓
```

## Complexity Analysis

**Time Complexity:** O(V + E) + O(N log N)
- Building reversed graph: O(V + E)
- BFS (Kahn's): O(V + E)
- Sorting safe nodes: O(N log N) where N = number of safe nodes
- Total: **O(V + E + N log N)**

**Space Complexity:** O(V + E) ≈ O(3N) ~ O(N)
- `indegree[]`: O(V)
- BFS queue: O(V)
- Reversed adjacency list: O(V + E)
- Total: **O(V + E)**
```
  Key Trick:
  Terminal nodes = 0 out-degree in original
                 = 0 in-degree in REVERSED graph
  → Apply Kahn's BFS on reversed graph
  → Nodes processed = safe nodes ✓
```
