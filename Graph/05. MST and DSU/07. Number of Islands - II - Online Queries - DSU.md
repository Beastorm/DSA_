## Number of Provinces - Disjoint Set

### Problem Statement:

Given an **undirected graph** with `V` vertices represented as an **adjacency matrix**, find the **number of provinces** (i.e., number of connected components).

- Two vertices `u` and `v` belong to the same province if there is a path between them.
- The graph is given as a 2D matrix where `adj[i][j] == 1` means there is an edge between `i` and `j`.

**Goal:** Return the count of connected components.

---

### Why Use Disjoint Set?

We have already solved this using DFS/BFS traversals.  
This video demonstrates solving it using **Disjoint Set Union (DSU)** to understand the data structure deeply for harder problems.

---

## Approach

1. Create a Disjoint Set of size `V`.
2. Iterate over the adjacency matrix:
   - For every `adj[i][j] == 1`, perform `union(i, j)`.
3. Count the number of nodes where `findUltimateParent(i) == i` (these are the "bosses" or roots of each component).

Each unique ultimate parent represents one connected component (province).

---

### Key Intuition:

- Every connected component will have exactly **one ultimate parent** (the boss).
- Nodes whose parent is themselves after all unions are the roots.
- Count of such roots = number of provinces.

---

### Code (C++)

```cpp
class DisjointSet {
public:
    vector<int> parent, size;
    DisjointSet(int n) {
        parent.resize(n);
        size.resize(n, 1);
        for(int i = 0; i < n; i++) parent[i] = i;
    }

    int findUltimateParent(int node) {
        if(parent[node] == node) return node;
        return parent[node] = findUltimateParent(parent[node]); // path compression
    }

    void unionBySize(int u, int v) {
        int pu = findUltimateParent(u);
        int pv = findUltimateParent(v);
        if(pu == pv) return;
        if(size[pu] < size[pv]) {
            parent[pu] = pv;
            size[pv] += size[pu];
        } else {
            parent[pv] = pu;
            size[pu] += size[pv];
        }
    }
};

class Solution {
public:
    int findCircleNum(vector<vector<int>>& isConnected) {
        int V = isConnected.size();
        DisjointSet ds(V);

        for(int i = 0; i < V; i++) {
            for(int j = 0; j < V; j++) {
                if(isConnected[i][j] == 1) {
                    ds.unionBySize(i, j);
                }
            }
        }

        int count = 0;
        for(int i = 0; i < V; i++) {
            if(ds.findUltimateParent(i) == i) count++;
        }
        return count;
    }
};
```

**Alternative (using parent array directly):**
You can also check `ds.parent[i] == i` instead of calling `findUltimateParent` (both work due to how the structure is maintained).

---

### Time & Space Complexity:

- **Time:** O(V²) (for iterating matrix) + O(V) for counting  
  (Union & Find are nearly O(1) with path compression + union by size/rank)
- **Space:** O(V)
