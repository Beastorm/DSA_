## Connected Components

**Problem Statement:**   
Given an undirected graph with `V` vertices (numbered 0 to V-1) and `E` edges, find the **number of connected components** in the graph.

A **connected component** is a subgraph where there exists a path between any two vertices, and no vertex shares an edge with a vertex outside the subgraph.

---

>### Examples

**Example 1:**
- **Input:** `V=4`, `edges=[[0,1],[1,2]]`
- **Output:** `2`
```
  Graph:
  0 ─── 1 ─── 2     3

  Component 1: {0, 1, 2}
  Component 2: {3}  (isolated vertex)

  Total = 2 ✓
```

**Example 2:**
- **Input:** `V=7`, `edges=[[0,1],[1,2],[2,3],[4,5]]`
- **Output:** `3`
```
  Graph:
  0 ─── 1 ─── 2 ─── 3     4 ─── 5     6

  Component 1: {0, 1, 2, 3}
  Component 2: {4, 5}
  Component 3: {6}  (isolated vertex)

  Total = 3 ✓
```

---

### Key Insight:
```
  Start BFS/DFS from every UNVISITED node.
  Each new BFS/DFS call = a new connected component!

  Visited array tracks which nodes are already explored.
  Unvisited node → new component found → increment count.

  Algorithm:
  for each node v from 0 to V-1:
      if v is NOT visited:
          count++
          BFS/DFS from v (marks all reachable nodes as visited)

  return count
```

---
## Appraoch
### Algorithm:
```
  1. Build an adjacency list from edges
  2. Create visited[] array, all false initially
  3. count = 0
  4. For each node v (0 to V-1):
       If NOT visited:
         count++
         BFS (or DFS) from v
         → marks all nodes in this component as visited
  5. Return count
```
```
  V=7, edges=[[0,1],[1,2],[2,3],[4,5]]

  adj[0]=[1], adj[1]=[0,2], adj[2]=[1,3]
  adj[3]=[2], adj[4]=[5],   adj[5]=[4]
  adj[6]=[]

  visited = [F,F,F,F,F,F,F]

  v=0: NOT visited → count=1, BFS(0)
       Visit: 0,1,2,3
  visited = [T,T,T,T,F,F,F]

  v=1: visited → skip
  v=2: visited → skip
  v=3: visited → skip

  v=4: NOT visited → count=2, BFS(4)
       Visit: 4,5
  visited = [T,T,T,T,T,T,F]

  v=5: visited → skip

  v=6: NOT visited → count=3, BFS(6)
       Visit: 6 (no neighbors)
  visited = [T,T,T,T,T,T,T]

  Return count = 3 ✓
```

---

### Code (BFS Approach):
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
private:
    // BFS to visit all nodes in one component
    void bfs(int start, vector<vector<int>>& adj, vector<bool>& visited) {
        queue<int> q;
        q.push(start);
        visited[start] = true;

        while (!q.empty()) {
            int node = q.front();
            q.pop();

            for (int neighbor : adj[node]) {
                if (!visited[neighbor]) {
                    visited[neighbor] = true;
                    q.push(neighbor);
                }
            }
        }
    }

public:
    int countComponents(int V, vector<vector<int>>& edges) {
        // Build adjacency list
        vector<vector<int>> adj(V);
        for (auto& edge : edges) {
            int u = edge[0], v = edge[1];
            adj[u].push_back(v);
            adj[v].push_back(u); // Undirected
        }

        vector<bool> visited(V, false);
        int count = 0;

        // Try starting BFS from every unvisited node
        for (int v = 0; v < V; v++) {
            if (!visited[v]) {
                count++;          // New component found!
                bfs(v, adj, visited);
            }
        }

        return count;
    }
};

int main() {
    Solution sol;

    // Example 1
    int V1 = 4;
    vector<vector<int>> edges1 = {{0,1},{1,2}};
    cout << "Example 1: " << sol.countComponents(V1, edges1) << endl; // 2

    // Example 2
    int V2 = 7;
    vector<vector<int>> edges2 = {{0,1},{1,2},{2,3},{4,5}};
    cout << "Example 2: " << sol.countComponents(V2, edges2) << endl; // 3

    return 0;
}
```

### Code (DFS Approach):
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
private:
    // DFS to visit all nodes in one component
    void dfs(int node, vector<vector<int>>& adj, vector<bool>& visited) {
        visited[node] = true;

        for (int neighbor : adj[node]) {
            if (!visited[neighbor]) {
                dfs(neighbor, adj, visited);
            }
        }
    }

public:
    int countComponents(int V, vector<vector<int>>& edges) {
        vector<vector<int>> adj(V);
        for (auto& edge : edges) {
            adj[edge[0]].push_back(edge[1]);
            adj[edge[1]].push_back(edge[0]);
        }

        vector<bool> visited(V, false);
        int count = 0;

        for (int v = 0; v < V; v++) {
            if (!visited[v]) {
                count++;
                dfs(v, adj, visited);
            }
        }

        return count;
    }
};
```

### Edge Cases:
```
  No edges (all isolated):     All connected (one component):
  V=4, edges=[]                V=4, edges=[[0,1],[1,2],[2,3]]

  count=4 (each node alone)    count=1
  0   1   2   3                0─1─2─3

  Single node:                 Already 1 component:
  V=1, edges=[]                V=3, edges=[[0,1],[1,2],[0,2]]
  count=1                      count=1 (triangle)
```

### Complexity Analysis:

**Time Complexity:** O(V + E)
- Building adjacency list: O(E)
- BFS/DFS visits each node once: O(V)
- Each edge processed twice (undirected): O(E)
- Total: **O(V + E)**

**Space Complexity:** O(V + E)
- Adjacency list: O(V + E)
- Visited array: O(V)
- BFS queue / DFS stack: O(V)
- Total: **O(V + E)**
```
  V = vertices, E = edges

  We visit every node and edge exactly once.
  Most efficient possible for this problem. ✓
```
