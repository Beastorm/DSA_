## Topological Sort Algorithm | DFS & BFS (Kahn's Algorithm)

**Problem Statement:**   
Given a **Directed Acyclic Graph (DAG)** with `V` vertices (0 to V-1), find a **Topological Sort** — a linear ordering of vertices such that for every directed edge `u → v`, vertex `u` appears **before** vertex `v`.

---

## Examples

**Example 1:**
- **Input:** `V=6`, edges: `5→0, 5→2, 4→0, 4→1, 2→3, 3→1`
- **Output:** `[5, 4, 2, 3, 1, 0]`
```
  5 ──→ 0
  |         ↑
  └──→ 2 ──→ 3 ──→ 1
       ↑         ↑
  4 ───┘─────────┘

  Valid orderings:
  [5,4,2,3,1,0] ✓
  [4,5,2,3,1,0] ✓ (multiple valid answers)
```

**Example 2:**
- **Input:** `V=4`, edges: `1→0, 2→0, 3→0`
- **Output:** `[3, 2, 1, 0]`
```
  1 ──┐
  2 ──┼──→ 0
  3 ──┘

  1,2,3 must all appear before 0 ✓
```

---

## Key Insight
```
  Topological Sort = Linear ordering of a DAG
  (Only possible for DAGs — no cycles!)

  Rule: If u → v exists, then u comes before v.

  Two approaches:
  ┌──────────────────────────────────────────────────┐
  │ Method 1: DFS + Stack                            │
  │   Push node to stack AFTER exploring all         │
  │   its dependencies. Reverse order = topo sort.  │
  │                                                  │
  │ Method 2: BFS (Kahn's Algorithm)                 │
  │   Process nodes with 0 in-degree first.          │
  │   Remove edges, update in-degrees, repeat.       │
  └──────────────────────────────────────────────────┘
```

---

## Method 1: DFS + Stack

### Intuition
```
  DFS explores all dependencies of a node first.
  After ALL dependencies explored → push to stack.
  Stack reverse = nodes with no dependencies appear FIRST.

  Example: 5→2→3→1
  DFS(5): explore 2 → explore 3 → explore 1
          push 1, push 3, push 2, push 5
  Stack (bottom→top): 5, 2, 3, 1
  Pop order: 5, 2, 3, 1 ← valid topo sort ✓
```

### Algorithm
```
  topoSort(V, adj):
    vis[] = all 0
    stack st

    For each vertex i (0 to V-1):
        If NOT vis[i]: DFS(i, vis, st)

    Pop all from stack → result

  DFS(node, vis, st):
    vis[node] = 1
    For each neighbor of node:
        If NOT vis[neighbor]: DFS(neighbor, vis, st)
    st.push(node)  ← push AFTER all neighbors explored
```

### Code
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    void dfs(int node, vector<int> adj[],
             vector<int>& vis, stack<int>& st) {

        vis[node] = 1;

        // Explore all neighbors first (dependencies)
        for (int neighbor : adj[node]) {
            if (!vis[neighbor]) {
                dfs(neighbor, adj, vis, st);
            }
        }

        // Push AFTER all neighbors are processed
        st.push(node);
    }

    vector<int> topoSort(int V, vector<int> adj[]) {
        vector<int> vis(V, 0);
        stack<int> st;

        // Handle disconnected components
        for (int i = 0; i < V; i++) {
            if (!vis[i]) dfs(i, adj, vis, st);
        }

        // Pop stack to get topological order
        vector<int> ans;
        while (!st.empty()) {
            ans.push_back(st.top());
            st.pop();
        }

        return ans;
    }
};

int main() {
    int V = 6;
    vector<int> adj[V];
    adj[5] = {0, 2};
    adj[4] = {0, 1};
    adj[2] = {3};
    adj[3] = {1};

    Solution obj;
    vector<int> res = obj.topoSort(V, adj);

    cout << "Topological Sort (DFS): ";
    for (int x : res) cout << x << " ";
    cout << endl;

    return 0;
}
```

### Step-by-Step DFS Trace
```
  Graph: 5→{0,2}, 4→{0,1}, 2→{3}, 3→{1}
  vis = [0,0,0,0,0,0]

  i=0: not visited → DFS(0)
    vis[0]=1, no neighbors → push 0
    stack=[0]

  i=1: not visited → DFS(1)
    vis[1]=1, no neighbors → push 1
    stack=[0,1]

  i=2: not visited → DFS(2)
    vis[2]=1, neighbor 3: DFS(3)
      vis[3]=1, neighbor 1: visited → skip
      push 3, stack=[0,1,3]
    push 2, stack=[0,1,3,2]

  i=3: visited → skip
  i=4: not visited → DFS(4)
    vis[4]=1
    neighbor 0: visited → skip
    neighbor 1: visited → skip
    push 4, stack=[0,1,3,2,4]

  i=5: not visited → DFS(5)
    vis[5]=1
    neighbor 0: visited → skip
    neighbor 2: visited → skip
    push 5, stack=[0,1,3,2,4,5]

  Pop stack: 5,4,2,3,1,0 ✓
```

## Method 2: BFS — Kahn's Algorithm

### Intuition
```
  In-degree of a node = number of edges COMING INTO it.

  Nodes with in-degree 0 → no dependencies → process first!

  After processing a node:
  → Remove its outgoing edges
  → Reduce in-degrees of neighbors
  → If any neighbor's in-degree becomes 0 → add to queue

  Keep processing until queue empty.

  Example: 5→0, 5→2, 4→0, 4→1, 2→3, 3→1

  In-degrees: 0→2, 1→2, 2→1, 3→1, 4→0, 5→0
  Start: queue=[4,5] (in-degree=0)

  Process 4: reduce in-degree of 0,1 → indeg=[1,1,1,1,0,0]
  Process 5: reduce in-degree of 0,2 → indeg=[0,1,0,1,0,0]
             0 and 2 become 0 → push to queue
  ...
```

### Algorithm
```
  topologicalSort(V, adj):
    indegree[] = all 0

    For each vertex i:
        For each neighbor of i:
            indegree[neighbor]++

    queue q
    Push all vertices with indegree[i]==0 to q

    topo = []
    While q not empty:
        node = q.front(); pop
        topo.add(node)
        For each neighbor of node:
            indegree[neighbor]--
            If indegree[neighbor] == 0: q.push(neighbor)

    Return topo
```

### Code
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    vector<int> topologicalSort(int V, vector<int> adj[]) {

        // Step 1: Calculate in-degree of all vertices
        vector<int> indegree(V, 0);
        for (int i = 0; i < V; i++) {
            for (int neighbor : adj[i]) {
                indegree[neighbor]++;
            }
        }

        // Step 2: Push all 0 in-degree nodes to queue
        queue<int> q;
        for (int i = 0; i < V; i++) {
            if (indegree[i] == 0) q.push(i);
        }

        // Step 3: BFS
        vector<int> topo;
        while (!q.empty()) {
            int node = q.front();
            q.pop();
            topo.push_back(node); // Add to result

            // Reduce in-degree of all neighbors
            for (int neighbor : adj[node]) {
                indegree[neighbor]--;
                if (indegree[neighbor] == 0) {
                    q.push(neighbor); // Ready to process
                }
            }
        }

        return topo;
    }
};

int main() {
    int V = 6;
    vector<int> adj[V];
    adj[5] = {0, 2};
    adj[4] = {0, 1};
    adj[2] = {3};
    adj[3] = {1};

    Solution obj;
    vector<int> ans = obj.topologicalSort(V, adj);

    cout << "Topological Sort (BFS/Kahn's): ";
    for (int x : ans) cout << x << " ";
    cout << endl;

    return 0;
}
```

### Step-by-Step BFS (Kahn's) Trace
```
  Graph: 5→{0,2}, 4→{0,1}, 2→{3}, 3→{1}

  ── Step 1: Compute in-degrees ──
  5→0: indeg[0]++    → indeg[0]=1
  5→2: indeg[2]++    → indeg[2]=1
  4→0: indeg[0]++    → indeg[0]=2
  4→1: indeg[1]++    → indeg[1]=1
  2→3: indeg[3]++    → indeg[3]=1
  3→1: indeg[1]++    → indeg[1]=2

  indegree = [2, 2, 1, 1, 0, 0]
              0  1  2  3  4  5

  ── Step 2: Push 0 in-degree nodes ──
  queue = [4, 5]

  ── Step 3: BFS ──

  Pop 4: topo=[4]
    neighbor 0: indeg[0]=2→1
    neighbor 1: indeg[1]=2→1
    queue=[5]

  Pop 5: topo=[4,5]
    neighbor 0: indeg[0]=1→0 → push 0
    neighbor 2: indeg[2]=1→0 → push 2
    queue=[0,2]

  Pop 0: topo=[4,5,0]
    no neighbors
    queue=[2]

  Pop 2: topo=[4,5,0,2]
    neighbor 3: indeg[3]=1→0 → push 3
    queue=[3]

  Pop 3: topo=[4,5,0,2,3]
    neighbor 1: indeg[1]=1→0 → push 1
    queue=[1]

  Pop 1: topo=[4,5,0,2,3,1]
    no neighbors
    queue=[]

  Result: [4, 5, 0, 2, 3, 1] ✓
```

## DFS vs BFS (Kahn's) Comparison
```
  Property          DFS + Stack            BFS (Kahn's)
  ─────────────────────────────────────────────────────────
  Data Structure    Stack (recursion)      Queue + in-degree
  Order             Finish-time reverse    In-degree driven
  Cycle Detection   Need separate check    If topo.size()<V → cycle!
  Implementation    Recursive              Iterative
  Intuition         "Dependencies first"   "No prereq → process first"

  Bonus: Kahn's detects cycle automatically!
  If topo.size() < V → cycle exists (some nodes never reached 0 in-degree)
```

## Cycle Detection via Kahn's
```cpp
// If result doesn't contain all V vertices → cycle exists!
if (topo.size() != V) {
    cout << "Cycle detected! Not a DAG." << endl;
}
```
```
  Cyclic graph: 0→1→2→0
  indegree = [1, 1, 1]
  No node has in-degree 0 → queue never gets anything
  topo.size() = 0 ≠ 3 → CYCLE DETECTED ✓
```

## Complexity Analysis

**DFS + Stack:**

**Time:** O(V + E) — each vertex visited once, each edge explored once.
**Space:** O(V + E) — adjacency list O(V+E), visited O(V), stack O(V), recursion O(V).

**BFS (Kahn's):**

**Time:** O(V + E) — in-degree calc O(E), BFS visits each vertex/edge once.
**Space:** O(V + E) — adjacency list O(V+E), in-degree array O(V), queue O(V).
```
  Both approaches: O(V + E) time and space.

  Key difference:
  DFS: natural recursion, harder to detect cycle inline
  BFS: iterative, cycle detection is FREE (check result size)!
```
