## Bipartite Graph | BFS Implementation

**Problem Statement:**   
Given an adjacency list of a graph with `V` vertices (0-based index), check whether the graph is **bipartite** or not using **BFS**.

A graph is bipartite if we can color all vertices using **2 colors** such that **no two adjacent vertices have the same color**.

---

>### Examples:

**Example 1:**
- **Input:** Even cycle (square graph)
- **Output:** `1` (Bipartite)
```
  0 ─── 1
  |     |
  3 ─── 2

  Color 0: {0, 2}   Color 1: {1, 3}
  No adjacent same-color pair → Bipartite ✓
```

**Example 2:**
- **Input:** Odd cycle (triangle)
- **Output:** `0` (Not Bipartite)
```
  0 ─── 1
   \   /
    \ /
     2

  0=color0, 1=color1, 2=?
  2 adj to both 0(c0) and 1(c1) → CONFLICT ✗
```

---

## Key Insight
```
  BFS colors nodes level by level:
  Level 0: color 0
  Level 1: color 1
  Level 2: color 0
  ...

  If we ever find a neighbor with SAME color as the current
  → ODD CYCLE EXISTS → NOT BIPARTITE!

  BFS vs DFS for this problem:
  Both work identically in logic.
  BFS = iterative (queue), DFS = recursive (stack)

  color[] array: -1=uncolored, 0=color0, 1=color1
  !col or (1-col) gives the opposite color.
```

---
## Approach
### Algorithm:
```
  isBipartite(V, adj):
    color[] = all -1

    For each vertex i (0 to V-1):         ← handle disconnected
        If color[i] == -1:
            If bfsCheck(i, adj, color) == false:
                return false

    return true

  bfsCheck(src, adj, color):
    color[src] = 0                   ← start with color 0
    queue.push(src)

    While the queue is not empty:
        node = queue.front(); pop

        For each neighbor of node:
            If color[neighbor] == -1:
                color[neighbor] = 1 - color[node]  ← opposite
                queue.push(neighbor)

            Else if color[neighbor] == color[node]:
                return false                        ← CONFLICT!

    return true                      ← all colored, no conflict
```

---

### Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
private:
    // BFS to check bipartiteness from the source node
    bool bfsCheck(int src, vector<int> adj[], int color[]) {
        // Color source node with 0
        color[src] = 0;
        queue<int> q;
        q.push(src);

        while (!q.empty()) {
            int node = q.front();
            q.pop();

            for (int neighbor : adj[node]) {
                if (color[neighbor] == -1) {
                    // Uncolored → assign opposite color
                    color[neighbor] = 1 - color[node];
                    q.push(neighbor);
                }
                else if (color[neighbor] == color[node]) {
                    // Same color → conflict → NOT bipartite
                    return false;
                }
                // Different color → fine, continue
            }
        }

        return true; // Component is bipartite
    }

public:
    bool isBipartite(int V, vector<int> adj[]) {
        int color[V];
        for (int i = 0; i < V; i++) color[i] = -1; // Uncolored

        // Check each component
        for (int i = 0; i < V; i++) {
            if (color[i] == -1) {
                if (!bfsCheck(i, adj, color))
                    return false;
            }
        }

        return true;
    }
};

void addEdge(vector<int> adj[], int u, int v) {
    adj[u].push_back(v);
    adj[v].push_back(u);
}

int main() {
    Solution sol;

    // Example 1: Bipartite (square 0-1-2-3-0)
    int V1 = 4;
    vector<int> adj1[V1];
    addEdge(adj1, 0, 1); addEdge(adj1, 1, 2);
    addEdge(adj1, 2, 3); addEdge(adj1, 3, 0);
    cout << "Example 1: " << (sol.isBipartite(V1, adj1) ? "1 (Bipartite)" : "0 (Not Bipartite)") << endl;

    // Example 2: Not bipartite (triangle 0-1-2-0)
    int V2 = 3;
    vector<int> adj2[V2];
    addEdge(adj2, 0, 1); addEdge(adj2, 1, 2); addEdge(adj2, 2, 0);
    cout << "Example 2: " << (sol.isBipartite(V2, adj2) ? "1 (Bipartite)" : "0 (Not Bipartite)") << endl;

    // Example 3: Disconnected graph
    int V3 = 6;
    vector<int> adj3[V3];
    addEdge(adj3, 0, 1); addEdge(adj3, 1, 2);  // component 1 (path)
    addEdge(adj3, 3, 4); addEdge(adj3, 4, 5); addEdge(adj3, 5, 3); // triangle
    cout << "Example 3: " << (sol.isBipartite(V3, adj3) ? "1 (Bipartite)" : "0 (Not Bipartite)") << endl;

    return 0;
}
```

## Step-by-Step Trace

### Bipartite Case (Square: 0─1─2─3─0):
```
  color = [-1, -1, -1, -1]

  i=0: color[0]==-1 → bfsCheck(0)
    color[0]=0, queue=[0]

    ── Pop 0 (color=0) ──
    neighbor 1: color=-1 → color[1]=1-0=1, push 1
    neighbor 3: color=-1 → color[3]=1, push 3
    queue=[1, 3]
    color=[0, 1, -1, 1]

    ── Pop 1 (color=1) ──
    neighbor 0: color=0 ≠ 1 → OK
    neighbor 2: color=-1 → color[2]=1-1=0, push 2
    queue=[3, 2]
    color=[0, 1, 0, 1]

    ── Pop 3 (color=1) ──
    neighbor 2: color=0 ≠ 1 → OK
    neighbor 0: color=0 ≠ 1 → OK
    queue=[2]

    ── Pop 2 (color=0) ──
    neighbor 1: color=1 ≠ 0 → OK
    neighbor 3: color=1 ≠ 0 → OK
    queue=[]

    return true ✓

  All vertices colored. return true → BIPARTITE ✓

  Final coloring:
  0(c0) ─── 1(c1)
  |               |
  3(c1) ─── 2(c0)

  Set A (color 0): {0, 2}
  Set B (color 1): {1, 3} ✓
```

### Not Bipartite Case (Triangle: 0─1─2─0):
```
  color = [-1, -1, -1]

  i=0: bfsCheck(0)
    color[0]=0, queue=[0]

    ── Pop 0 (color=0) ──
    neighbor 1: color=-1 → color[1]=1, push 1
    neighbor 2: color=-1 → color[2]=1, push 2
    queue=[1, 2]
    color=[0, 1, 1]

    ── Pop 1 (color=1) ──
    neighbor 0: color=0 ≠ 1 → OK
    neighbor 2: color=1 == 1 → CONFLICT! return false ✗

  return false → NOT BIPARTITE ✓
```

## Queue State Table
```
  Graph: 0─1─2─3─0 (square)

  Step  Action          Queue         Color Array
  ──────────────────────────────────────────────────
  Init  color[0]=0      [0]           [0,-1,-1,-1]
  Pop 0 color 1→1,3→1  [1,3]         [0,1,-1,1]
  Pop 1 color 2→0      [3,2]         [0,1,0,1]
  Pop 3 check 0,2 ≠ 1  [2]           [0,1,0,1]
  Pop 2 check 1,3 ≠ 0  []            [0,1,0,1]
  Done  queue empty     []            Bipartite ✓
```

## Disconnected Graph Handling
```
  Graph:
  Component 1: 0─1─2 (path, bipartite)
  Component 2: 3─4─5─3 (triangle, NOT bipartite)

  color = [-1,-1,-1,-1,-1,-1]

  i=0: bfsCheck(0) → colors {0,1,2} → true (bipartite)
  i=3: color[3]==-1 → bfsCheck(3):
       Finds triangle → returns false

  isBipartite returns false ✓

  Must check ALL components!
  Even if first component is bipartite,
  another component might not be ✓
```

---

## Color Assignment Logic
```
  color[neighbor] = 1 - color[node]

  If node has color 0 → neighbor gets 1-0 = 1
  If node has color 1 → neighbor gets 1-1 = 0

  Always gives the OPPOSITE color ✓

  Alternative: !color[node] (using boolean 0/1)
  Or: color[node] ^ 1 (XOR toggle)

  All equivalent! ✓
```

---

## DFS vs BFS Comparison
```
  BFS (this article):              DFS (previous):
  ─────────────────────            ──────────────────────
  Uses queue                       Uses recursion
  Iterative                        Recursive (can stack overflow)
  Colors level by level            Colors depth first
  Assigns color when PUSHING       Assigns color at ENTRY

  Code difference:
  BFS:                             DFS:
  color[neighbor] = 1-color[node]  color[node] = col
  q.push(neighbor)                 dfs(neighbor, !col)

  Same result, same complexity ✓
  BFS preferred for large graphs (no stack overflow risk)
```

## Edge Cases
```
  No edges:                  Complete bipartite (K3,3):
  V=4, no edges              3 nodes each side, all cross edges
  Each node alone            → Always bipartite ✓
  → Bipartite (trivially) ✓

  Single node:               Even cycle only:
  V=1, no edges              0─1─2─3─4─5─0
  color[0]=0, no neighbors   Length 6 (even) → Bipartite ✓
  return true ✓

  Odd cycle only:
  0─1─2─0 (triangle)
  Length 3 (odd) → NOT Bipartite ✓
```

## Complexity Analysis

**Time Complexity:** O(V + E)
- Each vertex enqueued and dequeued once: O(V)
- Each edge examined twice (undirected): O(E)
- Total: **O(V + E)**

**Space Complexity:** O(V + E)
- Adjacency list: O(V + E)
- Color array: O(V)
- BFS queue: O(V) worst case
- Total: **O(V + E)**
```
  Summary:
  Bipartite check using BFS:
  1. Color graph with 2 colors using BFS
  2. If conflict found (adjacent = same color) → false
  3. Handle all components with outer loop
  4. O(V+E) time, O(V) extra space
```
