## Find Eventual Safe States — DFS

**Problem Statement:**  
A node is called a **safe node** if every possible path starting from that node leads to a **terminal node** (a node with no outgoing edges) or to another safe node.

Given a **directed graph** with `V` nodes and adjacency list `adj`, return all **eventual safe nodes** in sorted order.

---

>### Examples:

**Example 1:**
- **Input:**
```
  0 → 1 → 2 → 5
  1 → 3 → 0    ← cycle!
  4 → 5
```
- **Output:** `[2, 4, 5, 6]`
```
  Nodes in cycle: {0, 1, 3} → NOT safe
  Node 2 → 5 (terminal) → SAFE
  Node 4 → 5 → SAFE
  Node 5 → terminal → SAFE
  Node 6 → terminal → SAFE
```

**Example 2:**
- **Input:** `0 → 1 → 2 → 0` (cycle)
- **Output:** `[]`
```
  All nodes in cycle → none are safe
```

---

### Key Insight:
```
  Safe node = NOT part of a cycle AND
              NOT leading to a cycle

  Unsafe node = part of a cycle OR leads to a cycle

  Strategy: Use DFS with TWO tracking arrays:
  vis[]     = globally visited (ever visited)
  pathVis[] = visited on CURRENT DFS path

  If we reach a node that is:
    - Already visited on CURRENT PATH → CYCLE DETECTED!
      (node is unsafe)

  After DFS backtrack:
    - If node is safe → check[node] = 1
    - If node is unsafe → check[node] = 0 (in cycle)
```

---

### Three Arrays Explained:
```
  vis[node]     = 1 if node was ever visited in any DFS call
  pathVis[node] = 1 if node is on the CURRENT DFS path
                  (reset to 0 when backtracking)
  check[node]   = 1 if node is eventually safe
                = 0 if node is part of/leads to cycle

  Cycle detection:
  If we reach node X and pathVis[X] == 1:
  → X is on current path → back edge → CYCLE!

  After successful DFS (no cycle found from node):
  check[node] = 1 → node is SAFE ✓
```

---
## Approach
### Algorithm:
```
  eventualSafeNodes(V, adj):
    vis[]     = all 0
    pathVis[] = all 0
    check[]   = all 0

    For i = 0 to V-1:
        If NOT vis[i]:
            dfsCheck(i, adj, vis, pathVis, check)

    safeNodes = []
    For i = 0 to V-1:
        If check[i] == 1: safeNodes.add(i)

    return safeNodes

  dfsCheck(node, adj, vis, pathVis, check):
    vis[node]     = 1
    pathVis[node] = 1
    check[node]   = 0    ← assume unsafe initially

    For each neighbor of node:
        If NOT vis[neighbor]:
            If dfsCheck(neighbor) == true:
                check[node] = 0
                return true  ← cycle found, propagate up

        Else if pathVis[neighbor]:
            check[node] = 0
            return true      ← cycle detected!

    ← No cycle found from this node
    check[node]   = 1    ← mark as SAFE
    pathVis[node] = 0    ← BACKTRACK (remove from path)
    return false         ← no cycle
```

---

### Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
private:
    bool dfsCheck(int node, vector<int> adj[],
                  int vis[], int pathVis[], int check[]) {
        vis[node]     = 1;
        pathVis[node] = 1;
        check[node]   = 0; // Assume unsafe (in cycle)

        // Traverse all adjacent nodes
        for (int neighbor : adj[node]) {
            // If neighbor not visited yet
            if (!vis[neighbor]) {
                if (dfsCheck(neighbor, adj, vis, pathVis, check) == true) {
                    check[node] = 0; // Cycle found downstream
                    return true;     // Propagate cycle detection
                }
            }
            // If neighbor visited and on current path → CYCLE!
            else if (pathVis[neighbor]) {
                check[node] = 0;
                return true;
            }
        }

        // No cycle found from this node
        check[node]   = 1; // Mark as SAFE
        pathVis[node] = 0; // Backtrack: remove from current path
        return false;      // No cycle
    }

public:
    vector<int> eventualSafeNodes(int V, vector<int> adj[]) {
        int vis[V]     = {0};
        int pathVis[V] = {0};
        int check[V]   = {0};

        // Run DFS from every unvisited node
        for (int i = 0; i < V; i++) {
            if (!vis[i]) {
                dfsCheck(i, adj, vis, pathVis, check);
            }
        }

        // Collect all safe nodes
        vector<int> safeNodes;
        for (int i = 0; i < V; i++) {
            if (check[i] == 1) safeNodes.push_back(i);
        }

        return safeNodes; // Already sorted (0 to V-1 order)
    }
};

int main() {
    int V = 7;
    vector<int> adj[V];

    // 0→1, 0→2, 1→2, 1→3, 2→5, 3→0, 4→5, 5→6 (example)
    adj[0] = {1, 2};
    adj[1] = {2, 3};
    adj[2] = {5};
    adj[3] = {0};
    adj[4] = {5};
    adj[5] = {6};
    adj[6] = {};

    Solution sol;
    vector<int> safe = sol.eventualSafeNodes(V, adj);

    cout << "Safe nodes: ";
    for (int x : safe) cout << x << " ";
    cout << endl;

    return 0;
}
```

### Why Two Arrays (vis and pathVis)?
```
  vis[]     prevents re-exploring already-processed nodes.
            Once a node is fully processed, don't redo it.

  pathVis[] detects BACK EDGES (cycle markers).
            Only nodes on CURRENT PATH have pathVis=1.

  Critical distinction:
  Node 5 in our example:

  After DFS(2→5→6): check[5]=1 (safe), pathVis[5]=0

  When DFS(4) sees neighbor 5:
  vis[5]=1 → already visited
  pathVis[5]=0 → NOT on current path → NOT a cycle!
  → Node 4 is SAFE ✓

  If we only had vis[]:
  When DFS(4) sees 5: vis[5]=1 → wrongly report cycle? No.
  We need pathVis to distinguish:
    "visited before" vs "visited in current path" ✓
```

---

### check[] Array Logic:
```
  check[node] = 0 → unsafe (in cycle or leads to cycle)
  check[node] = 1 → safe (all paths lead to terminal)

  When is check[node] set to 1?
  → When DFS from node completes WITHOUT finding any cycle
  → pathVis[node] = 0 (backtrack) at the same time

  check[node] = 1 means:
  Every neighbor either:
    1. Is a terminal node (no outgoing edges)
    2. Has check = 1 itself (verified safe)
  → All paths from node eventually terminate ✓
```

---

### Edge Cases:
```
  No edges (all terminal):      All nodes in one cycle:
  0, 1, 2, 3 (no edges)         0→1→2→0

  All are terminal → all safe   All unsafe → []
  return [0,1,2,3] ✓            return [] ✓

  Node pointing to safe node:   Mixed:
  4 → 5 (5 is safe)             Some cycle, some leading to cycle,
  4 is also SAFE ✓              some safe

  Nodes leading to unsafe:
  4 → 0 (0 is in cycle)
  4 is UNSAFE (check[4]=0) ✓
```

---

### Complexity Analysis:

**Time Complexity:** O(V + E)
- Each node visited once in DFS: O(V)
- Each edge processed once: O(E)
- Final scan for safe nodes: O(V)
- Total: **O(V + E)**

**Space Complexity:** O(V)
- `vis[]`, `pathVis[]`, `check[]` arrays: O(V) each
- DFS recursion stack: O(V) worst case
- Total: **O(V)**
```
  Key Arrays Summary:
  vis[]     → "ever visited?" (prevents reprocessing)
  pathVis[] → "on current DFS path?" (detects cycles)
  check[]   → "is this node eventually safe?"

  Return all nodes where check[i] == 1 ✓
```
