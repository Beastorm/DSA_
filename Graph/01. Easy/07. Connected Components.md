## Connected Components

**Problem Statement:**   
Given an undirected graph with `V` vertices (numbered 0 to V-1) and `E` edges, find the **number of connected components**.

A connected component is a subgraph where every vertex can reach every other vertex, and no vertex shares an edge with a vertex outside the subgraph.

---

>### Examples:

**Example 1:**
- **Input:** `V=4`, `edges=[[0,1],[1,2]]`
- **Output:** `2`
```
  Graph:
  0 ─── 1 ─── 2     3

  Component 1: {0, 1, 2}
  Component 2: {3}  ← isolated vertex

  Total = 2 ✓
```

**Example 2:**
- **Input:** `V=7`, `edges=[[0,1],[1,2],[2,3],[4,5]]`
- **Output:** `3`
```
  Graph:
  0 ─── 1 ─── 2 ─── 3     4 ─── 5     6

  Component 1: {0, 1, 2, 3}
  Component 2: {4, 5}
  Component 3: {6}  ← isolated vertex

  Total = 3 ✓
```

---

### Key Insight:
```
  Each time we start BFS/DFS from an UNVISITED node
  → we've found a NEW connected component!

  Count = number of times we start a new BFS/DFS

  Like counting islands: jump to a new island only
  when current island is fully explored.

  Algorithm idea:
  ┌─────────────────────────────────────────────┐
  │ for each vertex v:                          │
  │     if v NOT visited:                       │
  │         count++  ← new component!          │
  │         BFS/DFS(v) ← explore entire comp.  │
  └─────────────────────────────────────────────┘
```

---
## Appraoch
### Algorithm:
```
  1. Build an adjacency list from edges
  2. Initialize visited[] = all 0
  3. components = 0

  4. For each vertex i (0 to V-1):
       If NOT visited[i]:
           components++          ← new component
           BFS from i:
               Push i to the queue, mark visited
               While the queue is not empty:
                   Pop node
                   For each neighbor:
                       If not visited → mark, push

  5. Return components
```
```
  V=7, edges=[[0,1],[1,2],[2,3],[4,5]]

  adjList:
  0:[1], 1:[0,2], 2:[1,3], 3:[2]
  4:[5], 5:[4]
  6:[]

  visited=[0,0,0,0,0,0,0], components=0

  i=0: not visited → components=1, BFS(0)
    queue=[0]→pop 0, push 1
    queue=[1]→pop 1, push 2
    queue=[2]→pop 2, push 3
    queue=[3]→pop 3, no new neighbors
    visited=[1,1,1,1,0,0,0]

  i=1,2,3: visited → skip

  i=4: not visited → components=2, BFS(4)
    queue=[4]→pop 4, push 5
    queue=[5]→pop 5, no new neighbors
    visited=[1,1,1,1,1,1,0]

  i=5: visited → skip

  i=6: not visited → components=3, BFS(6)
    queue=[6]→pop 6, no neighbors
    visited=[1,1,1,1,1,1,1]

  Return 3 ✓
```

### Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
    int countComponents(int V, vector<vector<int>>& edges) {

        // Step 1: Build adjacency list
        vector<vector<int>> adj(V);
        for (auto& e : edges) {
            adj[e[0]].push_back(e[1]);
            adj[e[1]].push_back(e[0]); // Undirected
        }

        // Step 2: Initialize visited array
        vector<int> visited(V, 0);
        int components = 0;

        // Step 3: Try starting BFS from every unvisited node
        for (int i = 0; i < V; i++) {
            if (!visited[i]) {
                components++; // New component discovered!

                // BFS to explore entire component
                queue<int> q;
                q.push(i);
                visited[i] = 1;

                while (!q.empty()) {
                    int node = q.front();
                    q.pop();

                    for (int nbr : adj[node]) {
                        if (!visited[nbr]) {
                            visited[nbr] = 1;
                            q.push(nbr);
                        }
                    }
                }
            }
        }

        return components;
    }
};

int main() {
    Solution sol;

    // Example 1
    int V1 = 4;
    vector<vector<int>> edges1 = {{0,1},{1,2}};
    cout << "Example 1: " << sol.countComponents(V1, edges1) << endl; // 2

    // Example 2
    int V2 = 7;
    vector<vector<int>> edges2 = {{0,1},{1,2},{2,3},{4,5}};
    cout << "Example 2: " << sol.countComponents(V2, edges2) << endl; // 3

    // Example 3
    int V3 = 5;
    vector<vector<int>> edges3 = {{0,1},{1,2},{3,4}};
    cout << "Example 3: " << sol.countComponents(V3, edges3) << endl; // 2

    return 0;
}
```

## DFS Alternative
```cpp
class Solution {
private:
    void dfs(int node, vector<vector<int>>& adj, vector<int>& visited) {
        visited[node] = 1;
        for (int nbr : adj[node]) {
            if (!visited[nbr]) {
                dfs(nbr, adj, visited);
            }
        }
    }

public:
    int countComponents(int V, vector<vector<int>>& edges) {
        vector<vector<int>> adj(V);
        for (auto& e : edges) {
            adj[e[0]].push_back(e[1]);
            adj[e[1]].push_back(e[0]);
        }

        vector<int> visited(V, 0);
        int components = 0;

        for (int i = 0; i < V; i++) {
            if (!visited[i]) {
                components++;
                dfs(i, adj, visited); // Marks entire component
            }
        }

        return components;
    }
};
```

```

  V=4, edges=[]                V=3, edges=[[0,1]]

  adj: all empty               0─1    2(isolated)

  count=4 ✓

  All connected:               Already 1 component:
  V=3, edges=[[0,1],[1,2],[0,2]] V=4, edges=[[0,1],[1,2],[2,3]]

  0─1─2 (triangle)             0─1─2─3
  count=1 ✓                    count=1 ✓

  V=1, edges=[]:               Large disconnected:
  Single isolated vertex       V=6, edges=[[0,1],[2,3],[4,5]]
  count=1 ✓                    count=3 ✓
```

### Complexity Analysis:

**Time Complexity:** O(V + E)
- Building adjacency list: O(E)
- Each vertex visited exactly once: O(V)
- Each edge processed at most twice (undirected): O(E)
- Total: **O(V + E)**

**Space Complexity:** O(V + E)
- Adjacency list: O(V + E)
- Visited array: O(V)
- BFS queue: O(V) worst case
- Total: **O(V + E)**
```
  Why O(V+E) and not O(V²)?
  Adjacency LIST (not matrix):
  → Only store actual edges, not all V² pairs
  → For sparse graphs (E << V²): much more efficient!

  Example: V=10^6, E=10^6
  Matrix: 10^12 cells (impossible!)
  List:   2×10^6 entries ✓
```
