## Detect Cycle in an Undirected Graph (BFS)

**Problem Statement:**   
Given an undirected graph with `V` vertices and `E` edges, check whether it contains any cycle or not. Return `1` if a cycle exists, `0` otherwise.

---

>### Examples:

**Example 1:**
- **Input:** `V=8, E=7`
- **Output:** `0` (No cycle)
```
  1 ─── 2 ─── 3
        |
        4 ─── 7
        |
        5     6     8

  No node is visited twice via different paths.
  No cycle ✓
```

**Example 2:**
- **Input:** `V=8, E=6`
- **Output:** `1` (Cycle exists)
```
  1 ─── 2
        |
  4 ─── 5 ─── 6
  └───────────┘  ← cycle: 4→5→6→4

  4→5→6→4 forms a cycle! ✓
```

---

### Key Insight:
```
  In an undirected graph:
  If we visit a node that is ALREADY VISITED
  and it is NOT the PARENT of the current node
  → CYCLE DETECTED!

  Why track parent?
  In an undirected graph, edge A─B means:
  A's neighbor = B, B's neighbor = A

  When at B (came from A):
  A appears as a neighbor of B → but it's just the SAME edge!
  → Don't confuse this as a cycle.

  So: visited AND not parent → TRUE cycle ✓

  Without parent tracking:
  1 ─── 2
  BFS from 1: push 2 (parent=1)
  From 2: see neighbor 1 → 1 is visited!
  → Wrongly report cycle ✗

  With parent tracking:
  From 2: see neighbor 1 → 1 is visited BUT 1 = parent → SKIP
  → Correctly report no cycle ✓
```

---

### Algorithm:
```
  detect(src, adj, visited):
    visited[src] = 1
    queue = [(src, parent=-1)]

    While the queue is not empty:
      (node, parent) = queue.pop()

      For each neighbor of node:
        If NOT visited:
            visited[neighbor] = 1
            queue.push((neighbor, node))

        Else if neighbor ≠ parent:
            CYCLE FOUND → return true

    return false (no cycle)

  isCycle(V, adj):
    visited[] = all 0

    For each vertex i:
        If NOT visited[i]:
            If detect(i, adj, visited) → return true

    return false
```
```
  Why loop over all vertices?
  → Graph might be DISCONNECTED (multiple components)
  → Each component checked independently
  → If ANY component has a cycle → return true
```

---

### Code:
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
private:
    // BFS to detect cycle starting from src
    bool detect(int src, vector<int> adj[], int vis[]) {
        vis[src] = 1;

        // Queue stores {node, parent}
        queue<pair<int,int>> q;
        q.push({src, -1}); // src has no parent initially

        while (!q.empty()) {
            int node   = q.front().first;
            int parent = q.front().second;
            q.pop();

            // Explore all neighbors
            for (int adjacentNode : adj[node]) {
                if (!vis[adjacentNode]) {
                    // Unvisited → mark and push
                    vis[adjacentNode] = 1;
                    q.push({adjacentNode, node});
                }
                else if (parent != adjacentNode) {
                    // Visited AND not the parent → CYCLE!
                    return true;
                }
                // If visited AND is parent → just the back edge, skip
            }
        }

        return false; // No cycle in this component
    }

public:
    bool isCycle(int V, vector<int> adj[]) {
        int vis[V] = {0}; // All unvisited

        // Check each component
        for (int i = 0; i < V; i++) {
            if (!vis[i]) {
                if (detect(i, adj, vis)) return true;
            }
        }

        return false;
    }
};

int main() {
    // Example: No cycle
    // Graph: 1-2-3, 2-4
    vector<int> adj1[4] = {{}, {2}, {1, 3}, {2}};
    Solution obj;
    cout << (obj.isCycle(4, adj1) ? "1" : "0") << endl; // 0

    // Example: Cycle exists
    // Graph: 0-1-2-0 (triangle)
    vector<int> adj2[3];
    adj2[0] = {1, 2};
    adj2[1] = {0, 2};
    adj2[2] = {0, 1};
    cout << (obj.isCycle(3, adj2) ? "1" : "0") << endl; // 1

    return 0;
}
```

## Step-by-Step Trace

### No Cycle Case:
```
  Graph: 0 ─── 1 ─── 2 ─── 3
  adj[0]=[1], adj[1]=[0,2], adj[2]=[1,3], adj[3]=[2]

  vis=[0,0,0,0]

  detect(0, adj, vis):
    vis[0]=1, queue=[(0,-1)]

    Pop (0,-1): parent=-1
      neighbor 1: not visited → vis[1]=1, push(1,0)
      queue=[(1,0)]

    Pop (1,0): parent=0
      neighbor 0: visited, but 0==parent → SKIP (same edge)
      neighbor 2: not visited → vis[2]=1, push(2,1)
      queue=[(2,1)]

    Pop (2,1): parent=1
      neighbor 1: visited, 1==parent → SKIP
      neighbor 3: not visited → vis[3]=1, push(3,2)
      queue=[(3,2)]

    Pop (3,2): parent=2
      neighbor 2: visited, 2==parent → SKIP
      queue=[]

  Queue empty → return false (no cycle)
  isCycle → false → output: 0 ✓
```

### Cycle Case:
```
  Graph: 0 ─── 1 ─── 2
         └───────────┘  (triangle)

  adj[0]=[1,2], adj[1]=[0,2], adj[2]=[0,1]

  detect(0, adj, vis):
    vis[0]=1, queue=[(0,-1)]

    Pop (0,-1): parent=-1
      neighbor 1: not visited → vis[1]=1, push(1,0)
      neighbor 2: not visited → vis[2]=1, push(2,0)
      queue=[(1,0),(2,0)]

    Pop (1,0): parent=0
      neighbor 0: visited, 0==parent → SKIP
      neighbor 2: visited, 2≠parent(0) → CYCLE DETECTED!
      return true ✓

  isCycle → true → output: 1 ✓
```

---

### Visualization:
```
  Cycle detection logic at each neighbor:

  neighbor is...    visited?    == parent?    Action
  ─────────────────────────────────────────────────
  Fresh node        NO          —             Mark, push to queue
  Parent node       YES         YES           Skip (same edge, not cycle)
  Other visited     YES         NO            CYCLE FOUND! return true

  Example - Triangle 0-1-2-0:

  Queue: [(0,-1)]
            ↓
  Pop 0: push (1,0), (2,0)
  Queue: [(1,0), (2,0)]
            ↓
  Pop 1 (parent=0):
    neighbor 0 → visited, 0==parent → SKIP
    neighbor 2 → visited, 2≠0 → CYCLE!
```

---

### Disconnected Graph Handling:
```
  Graph: 0─1─2    3─4─5─3 (cycle in component 2)

  Component 1: {0,1,2}  → no cycle
  Component 2: {3,4,5}  → has cycle 3→4→5→3

  isCycle loop:
  i=0: not visited → detect(0) → false
  i=3: not visited → detect(3) → TRUE → return true ✓

  We MUST check all components,
  not just from node 0! ✓
```

---

### Edge Cases:
```
  Single node:            Two nodes, one edge:
  V=1, no edges           V=2, edge 0-1

  vis[0]=1                No cycle (just back edge)
  No neighbors            return false ✓
  return false ✓

  Self-loop:              Multiple components, one cyclic:
  V=1, edge 0-0           Component 1: 0─1─2 (no cycle)
                          Component 2: 3─4─3 (cycle)
  adj[0]=[0]
  Pop (0,-1):             → return true ✓
    neighbor 0: visited,
    0 ≠ -1(parent) → CYCLE!
  return true ✓
```

---

### Complexity Analysis:

**Time Complexity:** O(V + E)
- Each vertex visited at most once: O(V)
- Each edge processed at most twice (undirected): O(E)
- Total: **O(V + E)**

**Space Complexity:** O(V + E)
- Adjacency list: O(V + E)
- Visited array: O(V)
- BFS queue: O(V) worst case
- Total: **O(V + E)**
```
  For dense graphs (E ≈ V²): O(V²)
  For sparse graphs (E ≈ V): O(V)

  BFS avoids stack overflow issues
  that recursive DFS might face for large graphs ✓
```
