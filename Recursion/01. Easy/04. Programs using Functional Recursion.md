## Reverse an Array Using Functional Recursion
**Problem Statement:** Reverse an array using a functional recursion approach.

### Approach using 2 pointers:
Use two pointers (left and right) and swap elements recursively.

### C++ Code:

```cpp
#include <iostream>
using namespace std;

void reverseArray(int arr[], int left, int right) {
    // Base case: when pointers meet or cross
    if (left >= right) {
        return;
    }
    
    // Swap elements at left and right
    swap(arr[left], arr[right]);
    
    // Recursive call: move pointers inward
    reverseArray(arr, left + 1, right - 1);
}

void printArray(int arr[], int n) {
    for (int i = 0; i < n; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int n = 5;
    
    cout << "Original array: ";
    printArray(arr, n);
    
    reverseArray(arr, 0, n - 1);
    
    cout << "Reversed array: ";
    printArray(arr, n);
    
    return 0;
}
```

### Approach using one pointer:
Using a Single Pointer

### C++ Code:
``` cpp
void reverseArraySingle(int arr[], int i, int n) {
    // Base case: reached middle
    if (i >= n / 2) {
        return;
    }
    
    // Swap element at i with element at (n-i-1)
    swap(arr[i], arr[n - i - 1]);
    
    // Recursive call
    reverseArraySingle(arr, i + 1, n);
}

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int n = 5;
    
    cout << "Original array: ";
    printArray(arr, n);
    
    reverseArraySingle(arr, 0, n);
    
    cout << "Reversed array: ";
    printArray(arr, n);
    
    return 0;
}
```
### Complexity Analysis

- **Time Complexity:** O(n)

- n/2 recursive calls
- Each call does O(1) work (swap)

- **Space Complexity:** O(n/2) = O(n)

- Recursion stack depth: n/2
- No extra data structures

### Key Points:

- Uses functional recursion approach
- Modifies array in-place
- Two-pointer technique: left and right converge
- Single pointer technique: i goes from 0 to n/2
- Base case: when pointers meet or cross

### Check if a String is Palindrome - Functional Recursion

**Problem Statement:**
Check whether a given string is a palindrome or not using functional recursion.
A palindrome reads the same forward and backward.

Examples: "madam", "racecar", "level".

---

### Approach: Two Pointers
Use two pointers (left and right) and compare characters recursively.
If all corresponding characters match, the string is a palindrome.

### C++ Code:

```cpp
#include <iostream>
#include <string>
using namespace std;

bool isPalindrome(string& s, int left, int right) {
    // Base case: pointers meet or cross
    if (left >= right) {
        return true;
    }
    
    // If characters don't match, not a palindrome
    if (s[left] != s[right]) {
        return false;
    }
    
    // Recursive call: move pointers inward
    return isPalindrome(s, left + 1, right - 1);
}

int main() {
    string s1 = "madam";
    string s2 = "hello";
    string s3 = "racecar";
    string s4 = "a";
    string s5 = "";
    
    cout << "\"" << s1 << "\" is palindrome: " << (isPalindrome(s1, 0, s1.length() - 1) ? "Yes" : "No") << endl;
    cout << "\"" << s2 << "\" is palindrome: " << (isPalindrome(s2, 0, s2.length() - 1) ? "Yes" : "No") << endl;
    cout << "\"" << s3 << "\" is palindrome: " << (isPalindrome(s3, 0, s3.length() - 1) ? "Yes" : "No") << endl;
    cout << "\"" << s4 << "\" is palindrome: " << (isPalindrome(s4, 0, s4.length() - 1) ? "Yes" : "No") << endl;
    cout << "\"" << s5 << "\" is palindrome: " << (isPalindrome(s5, 0, s5.length() - 1) ? "Yes" : "No") << endl;
    
    return 0;
}
```

### Approach: One Pointer
Single Pointer Approach

### C++ Code:
``` cpp
#include <iostream>
#include <string>
using namespace std;

bool isPalindromeSingle(string& s, int i, int n) {
    // Base case: reached middle
    if (i >= n / 2) {
        return true;
    }
    
    // If characters don't match, not a palindrome
    if (s[i] != s[n - i - 1]) {
        return false;
    }
    
    // Recursive call
    return isPalindromeSingle(s, i + 1, n);
}

int main() {
    string s1 = "madam";
    string s2 = "hello";
    
    int n1 = s1.length();
    int n2 = s2.length();
    
    cout << "\"" << s1 << "\" is palindrome: " << (isPalindromeSingle(s1, 0, n1) ? "Yes" : "No") << endl;
    cout << "\"" << s2 << "\" is palindrome: " << (isPalindromeSingle(s2, 0, n2) ? "Yes" : "No") << endl;
    
    return 0;
}
```

### Complexity Analysis:

- **Time Complexity:** O(n)
    
    - n/2 recursive calls
    - Each call does O(1) work (comparison)

- **Space Complexity:** O(n)

    - Recursion stack depth: n/2
    - No extra data structures

### Key Points:

- Uses functional recursion with a return value
- Two-pointer technique: compare left and right characters
- Single pointer technique: compare i and n-i-1
- Base case: when pointers meet or cross, return true
- Early termination: if characters don't match, return false immediately
- Empty string and single character are palindromes
