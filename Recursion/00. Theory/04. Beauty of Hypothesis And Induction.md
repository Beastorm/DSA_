## Lecture 4: Beauty of Hypothesis and Induction

## Steps to Design a Recursive Solution

### 1. First, Design Your Hypothesis

- **Hypothesis:**  
  Assume your recursive function works correctly for a smaller or simpler input.
- **Why:**  
  This helps you focus on how to use the result of the smaller problem to solve the bigger one.
- **Example (Factorial):**  
  Hypothesis: Assume `factorial(n-1)` correctly returns the factorial of `n-1`.

### 2. Second, Design Your Induction

- **Induction:**  
  Use the hypothesis to build the solution for the current (bigger) problem.
- **How:**  
  Figure out how to combine the result of the smaller problem with the current input to get the answer for the current problem.
- **Example (Factorial):**  
  Induction: `factorial(n) = n * factorial(n-1)`

### 3. Then, Design Your Base Case

- **Base Case:**  
  Define the simplest case where the answer is obvious, and the recursion should stop.
- **Why:**  
  This prevents infinite recursion and provides a stopping point.
- **Example (Factorial):**  
  Base: If `n == 0`, return 1.

---

### Example 1: Factorial

- **Hypothesis:** `factorial(n-1)` gives the correct answer for `n-1`.
- **Induction:** Multiply `n` with `factorial(n-1)` to get `factorial(n)`.
- **Base Case:** If `n == 0`, return 1.

```cpp
int factorial(int n) {
    if (n == 0) return 1;           // Base case
    return n * factorial(n - 1);    // Induction using hypothesis
}
```

### Example 2: Print 1 to N

- **Hypothesis:**  
  Assume you can print numbers from 1 to n-1.

- **Induction:**  
  Print numbers from 1 to n-1 (using recursion), then print n.

- **Base Case:**  
  If n == 0, stop (do nothing).

```cpp
void print1toN(int n) {
    if (n == 0) return;         // Base case
    print1toN(n-1);             // Hypothesis/Induction
    cout << n << " ";           // Print after recursion
}
```
### Example 3: Sum of Array

- **Hypothesis:**  
  Assume you can sum the array from index 1 to n-1.

- **Induction:**  
  Add the first element to the sum of the rest of the array.

- **Base Case:**  
  If the array is empty (n == 0), return 0.

```cpp
int sumArray(int arr[], int n) {
    if (n == 0) return 0;                 // Base case
    return arr[0] + sumArray(arr + 1, n - 1); // Induction using hypothesis
}
```
