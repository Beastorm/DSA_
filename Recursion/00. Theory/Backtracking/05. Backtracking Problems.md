Below are classic backtracking problems and how the general backtracking template applies to each:

---

### 1. Permutations of a String

- **State:** Current string and used/unused characters.
- **Choices:** Pick any unused character for the next position.
- **Base Case:** All positions filled (permutation complete).
- **Backtrack:** Unmark character as used.

---

### 2. Largest Number in K Swaps

- **State:** Current string/number and remaining swaps.
- **Choices:** Swap any two digits (if swaps left).
- **Base Case:** No swaps left.
- **Backtrack:** Undo the swap.

---

### 3. N-Digit Numbers with Digits in Increasing Order

- **State:** Current number being built, last digit used, digits left.
- **Choices:** Next digit must be greater than the last used.
- **Base Case:** N digits chosen.
- **Backtrack:** Remove last digit.

---

### 4. Rat in a Maze

- **State:** Current cell (i, j) and path so far.
- **Choices:** Move in any valid direction (up, down, left, right).
- **Base Case:** Reached destination cell.
- **Backtrack:** Unmark cell as visited.

---

### 5. Word Break

- **State:** Current index in string, current sentence.
- **Choices:** Try every dictionary word that matches the prefix.
- **Base Case:** Reached end of string.
- **Backtrack:** Remove the last word from the sentence.

---

### 6. Palindrome Partitioning

- **State:** Current index in string, current list of partitions.
- **Choices:** Partition at every possible palindrome substring.
- **Base Case:** Reached end of string.
- **Backtrack:** Remove last partition.

---

### 7. Combinational Sum

- **State:** Current combination, current sum, index in array.
- **Choices:** Include or exclude each number (can reuse if allowed).
- **Base Case:** Current sum equals target.
- **Backtrack:** Remove the last number from the combination.

---

### 8. Sudoku

- **State:** Current board configuration.
- **Choices:** Place any valid number in the next empty cell.
- **Base Case:** Board is completely filled.
- **Backtrack:** Remove the number from the cell.

---

### 9. N Queens

- **State:** Current board, row being filled.
- **Choices:** Place a queen in any valid column in the current row.
- **Base Case:** All queens placed (all rows filled).
- **Backtrack:** Remove queen from cell.

---

### 10. Phone Letter Digits

- **State:** Current index in digit string, current combination.
- **Choices:** For the current digit, try all possible mapped letters.
- **Base Case:** All digits processed.
- **Backtrack:** Remove the last letter from the combination.

---

**In all these problems, the backtracking template is:**
- Try all valid choices at each step,
- Recurse with the updated state,
- Undo the choice (backtrack) before trying the next option.
